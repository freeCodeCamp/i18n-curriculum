# --description--

في الدرس السابق، تعلّمت عن أنواع البيانات الشائعة في Python بما في ذلك سلسلة نصية، أعداد صحيحة، وأنواع الفاصلة العائمة. أثناء بناء برامجك، ستحتاج إلى معرفة كيفية عرض نوع المتغير.


إليك مثال على متغير:

```py
developer = 'Devin'
```

لمعرفة نوع `developer`، يمكنك استخدام الدالة `type()` بهذه الطريقة:

```py
developer = 'Devin'

print(type(developer)) # <class 'str'>
```

الناتج `<class 'str'>` يعني أن `developer` هو من نوع سلسلة نصية.

إذا لم تقدّم أي معلمات إلى الدالة `type()`، فستتلقى رسالة الخطأ التالية:

```md
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: type() takes 1 or 3 arguments
```

هذه جميع أنواع البيانات التي تعلمتها حتى الآن مع أنواعها في الطرفية:

```python
my_integer_var = 10
print(type(my_integer_var))  # <class 'int'>

my_float_var = 4.50
print(type(my_float_var))  # <class 'float'>

my_string_var = 'hello'
print(type(my_string_var))  # <class 'str'>

my_boolean_var = True
print(type(my_boolean_var))  # <class 'bool'>

my_set_var = {7, 'hello', 8.5}
print(type(my_set_var))  # <class 'set'>

my_dictionary_var = {'name': 'Alice', 'age': 25}
print(type(my_dictionary_var))  # <class 'dict'>

my_tuple_var = (7, 'hello', 8.5)
print(type(my_tuple_var))  # <class 'tuple'>

my_range_var = range(5)
print(type(my_range_var))  # <class 'range'>

my_list = [22, 'Hello world', 3.14, True]
print(type(my_list)) # <class 'list'>

my_none_var = None
print(type(my_none_var))  # <class 'NoneType'>
```

ستحتاج أحيانًا في برنامجك إلى التحقق من أن متغيرًا معينًا هو من نوع محدد قبل تنفيذ العمليات عليه. هنا تأتي فائدة الدالة `isinstance()`.

إليك مثال على متغير مُعيَّن له سلسلة نصية:

```py
account_balance = '12'
```

إذا حاولت إجراء عمليات رياضية مثل القسمة باستخدام المتغير `account_balance`، فستتلقى رسالة خطأ.

```py
account_balance = '12'

account_balance / 2

# Traceback (most recent call last):
#   File "<stdin>", line 1, in <module>
# TypeError: unsupported operand type(s) for /: 'str' and 'int'
```

لمعرفة ما إذا كان `account_balance` عددًا صحيحًا، يمكنك التحقق باستخدام الدالة `isinstance()` بهذه الطريقة:

```py
account_balance = '12'

isinstance(account_balance, int) # False
```

تسمح لك الدالة المدمجة `isinstance()` بالتحقق مما إذا كان المتغير يطابق نوع بيانات معين. تأخذ كائنًا والنوع الذي تريد التحقق منه، ثم تُرجع قيمة منطقية. في هذه الحالة، بما أن `account_balance` سلسلة نصية، فستُرجع `False`.

تسمح الدالة `isinstance()` أيضًا بالتحقق من عدة أنواع في نفس الوقت.

إليك مثالًا يتحقق مما إذا كان `account_balance` من نوع `int` أو `float`:

```py
account_balance = 12
isinstance(account_balance, (int, float)) # True
```

في هذا المثال، `account_balance` عدد صحيح لذا تُرجع `isinstance()` القيمة `True`. إذا كان `account_balance` يساوي `12.0`، فستظل `isinstance()` تُرجع `True` لأنك تتحقق من الأعداد الصحيحة أو الفاصلة العائمة.

في ورش العمل والمختبرات المستقبلية، ستستخدم الدالتين `type()` و `isinstance()` لضمان احتواء متغيراتك على أنواع البيانات الصحيحة قبل تنفيذ العمليات عليها.
