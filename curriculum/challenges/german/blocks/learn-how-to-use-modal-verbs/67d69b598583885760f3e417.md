# --explanation--

`Might` wird verwendet, um über Möglichkeiten zu sprechen – etwas, das passieren könnte, aber nicht sicher ist. Zum Beispiel:

- `It might rain tomorrow.` – Es gibt eine Chance, aber es ist nicht garantiert.

- `We might need more time to finish the project.` – Es besteht eine Möglichkeit, aber sie ist nicht bestätigt.
