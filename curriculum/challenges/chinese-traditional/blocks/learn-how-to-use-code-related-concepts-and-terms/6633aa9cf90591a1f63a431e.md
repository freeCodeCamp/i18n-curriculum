# --description--

連接詞 `so` 用來介紹與前述內容相關的結果或效果。它經常顯示句子兩個零件之間的因果關係。

例如，`It was raining, so we canceled the picnic.` 這表示野餐因為下雨而取消。

# --questions--

## --text--

哪一句正確使用了 `so`？

## --answers--

`I was hungry so I am eating.`

### --feedback--

這句話幾乎正確使用了 `so`，但第二部分的時態應該與第一部分相符。正確的說法是：`I was hungry, so I ate.`

---

`She is late to work, so she misses the meeting.`

### --feedback--

使用 `so` 表示結果，但時態不匹配。應該是：`She is late to work, so she will miss the meeting.`

---

`It started raining, so we took shelter.`

---

`He likes to read books, so he buys a lamp.`

### --feedback--

這句話沒有正確使用 `so`，因為買燈並不是喜歡閱讀書籍的直接結果。更好的用法可能是：`He likes to read at night, so he bought a lamp.`
