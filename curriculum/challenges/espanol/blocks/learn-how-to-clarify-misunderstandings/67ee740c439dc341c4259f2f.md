`another thing`

### --feedback--

Esta frase de dos palabras se usa para introducir un punto o problema adicional que aún no se ha mencionado.

# --explanation--

`Mostly the menu` significa que el problema generalmente ocurre con el menú, pero también podrían estar afectadas otras partes.

`There's another thing` se usa para mencionar un problema adicional que aún no se ha mencionado. Por ejemplo:

`There's another thing I wanted to tell you.` – Esto significa que hay algo más importante que necesitas contarle a alguien.
