# --description--

如果整数的十进制扩展不包含任何表示11的幂的子串（除1之外），则称其为十一无码。

例如，2404和13431不包含11，而911和4121331不是。

设 $E(n)$ 为第 $n$ 个正的无十一整数。例如，$E(3) = 3$，$E(200) = 213$ 和 $E(500\,000) = 531\,563$。

查找 $E({10}^{18})$。
