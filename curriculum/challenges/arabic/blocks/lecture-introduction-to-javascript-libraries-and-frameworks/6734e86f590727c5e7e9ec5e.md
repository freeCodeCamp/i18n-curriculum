# --description--

React هي واحدة من أشهر مكتبات JavaScript لبناء واجهات المستخدم وتطبيقات الويب. تم تطوير React وصيانتها في الأصل بواسطة Facebook، واكتسبت شعبية كبيرة في تطوير الويب بسبب كفاءتها ومرونتها وميزاتها.

مفهوم أساسي في React هو إنشاء مكوِّنات واجهة مستخدم قابلة لإعادة الاستخدام. يمكن بسهولة إعادة استخدام هذه المكوِّنات، مثل الأزرار والبطاقات ومكوِّنات الصورة الرمزية، في جميع أنحاء التطبيق. يمكنك تسكين هذه المكوِّنات داخل بعضها لبناء واجهات أكثر تعقيدًا وديناميكية.

واحدة من المزايا الرئيسية لـ React هي أن هذه المكوِّنات المخصصة يمكنها التحديث والعرض بشكل مستقل مع تغير البيانات.

على عكس جافاسكريبت التقليدية التي تتطلب التلاعب المباشر بـ DOM (نموذج كائن المستند)، يستخدم React DOM افتراضيًا، مما يحسن الأداء والكفاءة. ستتعلم المزيد عن DOM الافتراضي وكيفية عمله في الدروس القادمة.

بالإضافة إلى المكوِّنات القابلة لإعادة الاستخدام، يوفر React طريقة قوية لإدارة حالة التطبيق. تشير الحالة إلى البيانات التي تحدد كيفية عرض المكوِّن وتصرفه.

مع React، يمكنك تتبع وتحديث حالة المكوِّنات بسهولة، مما يضمن أن واجهة المستخدم تعكس أحدث البيانات. ستتعلم المزيد عن العمل مع الحالة في الدروس المستقبلية.

بينما توجد العديد من المكتبات ضمن نظام JavaScript البيئي، سيركز freeCodeCamp بشكل رئيسي على تعليم React بسبب استخدامه الواسع والطلب عليه في الصناعة.

ستتعمق دروسنا القادمة في بناء مكوِّنات مخصصة، وإدارة الحالة، والمزيد.
