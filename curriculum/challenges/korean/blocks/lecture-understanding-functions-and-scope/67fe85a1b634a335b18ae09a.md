# --description--

파이썬에서 범위는 변수를 접근할 수 있는 지점을 결정합니다. 이는 변수의 생명 주기와 코드의 여러 부분에서 변수가 어떻게 해결되는지를 제어합니다.

범위를 올바르게 결정하기 위해 파이썬은 **LEGB** 규칙을 따릅니다. 이 규칙은 다음을 의미합니다:

- **지역 범위 (L)**: 함수나 클래스 내에 정의된 변수입니다.
    
- **포함 범위 (E)**: 포함되거나 중첩된 함수 내에 정의된 변수입니다.
    
- **전역 범위 (G):** 모듈이나 파일의 최상위에 정의된 변수입니다.
    
- **내장 범위 (B)**: 파이썬에서 미리 정의된 함수, 모듈, 키워드, 객체에 예약된 이름입니다.

파이썬은 LEGB 규칙을 사용해 프로그램 내 변수의 범위를 해결합니다. 각 규칙을 자세히 살펴보면서 이 과정을 더 잘 이해할 수 있도록 하겠습니다.

**지역 범위**는 함수나 클래스 내부에서 선언된 변수가 그 함수나 클래스 내에서만 접근 가능하다는 뜻입니다.

예를 들어:

```python
def my_func():
    my_var = 10
    print(my_var)
```

이 경우 `my_func` 함수는 자체 범위를 가지며 함수 외부에서 접근할 수 없습니다. `my_func`을 호출하면 `10`가 출력되지만, 함수 외부에서 `my_var`을 출력하려 하면 `NameError`가 발생합니다:

```python
def my_func():
    my_var = 10 # Locally scoped to my_func
    print(my_var)

my_func() # 10

print(my_var) # NameError: name 'my_var' is not defined
```

**포함 범위**는 한 함수가 다른 함수 안에 중첩되어 있을 때, 중첩된 함수가 포함된 함수의 변수를 접근할 수 있다는 뜻입니다.

예를 들어:

```python
def outer_func():
    msg = 'Hello there!'

    def inner_func():
        print(msg)

    inner_func()

outer_func() # Hello there!
```

이 예제에서 내부 함수 `inner_func`는 외부 함수 `msg`에 정의된 `outer_func` 변수를 자유롭게 접근할 수 있습니다. 하지만 외부 함수는 중첩된 함수 내에 정의된 변수를 접근할 수 없다는 점에 주의하세요:

```python
def outer_func():
    msg = 'Hello there!'
    print(res)

    def inner_func():
        res = 'How are you?'
        print(msg)

    inner_func()

outer_func() # NameError: name 'res' is not defined
```

이는 `res`이 `inner_func`에 지역 범위로 한정되었기 때문입니다. 또한 `outer_func`이 `res`가 호출되기 전에 `inner_func`을 출력하려 시도하는 점도 주목하세요.

한 가지 해결책은 `res`을 포함 범위인 `outer_func` 내에서 빈 문자열로 초기화하는 것입니다. 그리고 `inner_func` 내에서 `res`을 `nonlocal` 키워드로 비지역 변수로 지정하세요:

```python
def outer_func():
    msg = 'Hello there!'
    res = ""  # Declare res in the enclosing scope

    def inner_func():
        nonlocal res  # Allow modification of an enclosing variable
        res = 'How are you?'
        print(msg)  # Accessing msg from outer_func()

    inner_func()
    print(res)  # Now res is accessible and modified

outer_func()

# Output:
# Hello there!
# How are you?
```

**전역 범위**는 함수나 클래스 외부에서 선언되어 프로그램 어디서나 접근할 수 있는 변수를 의미합니다. 여기서 `my_var`은 정의되지 않은 함수 내부에서도 접근할 수 있습니다:

```python
my_var = 100

def show_var():
    print(my_var)

show_var() # 100
print(my_var) # 100
```

그리고 함수 내부에 정의된 지역 변수를 전역에서 접근 가능하게 만들고 싶다면 `global` 키워드를 사용할 수 있습니다:

```python
my_var_1 = 7

def show_vars():
    global my_var_2
    my_var_2 = 10
    print(my_var_1)
    print(my_var_2)

show_vars() # 7 10

# my_var_2 is now a global variable and can be accessed anywhere in the program
print(my_var_2) # 10
```

또한 전역 변수를 수정하려면 `global` 키워드를 사용할 수 있습니다:

```python
my_var = 10  # A global variable

def change_var():
    global my_var  # Allows modification of a global variable
    my_var = 20

change_var()

print(my_var)  # my_var is now modified globally to 20
```

마지막으로, **내장 범위**는 파이썬의 내장 함수, 모듈, 키워드를 모두 포함하며 프로그램 어디서나 사용할 수 있습니다:

```python
print(str(45)) # '45'
print(type(3.14)) # <class 'float'>
print(isinstance(3, str)) # False
```
