# --description--

考慮所有具有以下條件的三角形：

- 它們所有的頂點都在格點上。
- 以原點 O 為外心。
- 垂心位於點 H(5, 0)。

有九個此類三角形的 $\text{周長} ≤ 50$。

它們周長的總和，四捨五入到小數點後 4 位，是 291.0089。

尋找所有周長 $\text{perimeter} ≤ {10}^5$ 的此類三角形。將它們的周長總和（四捨五入到小數點後四位）作為你的答案。
