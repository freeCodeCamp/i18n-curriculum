# --interactive--

`Canvas` APIは、JavaScriptファイル内でグラフィックスを操作できる強力なツールです。すべてはHTMLの`canvas`要素から始まります。この要素は描画面として機能し、`Canvas` APIのインスタンスメソッドやプロパティを使って操作できます。

`Canvas` APIは、図形、テキスト、アニメーション、さらには複雑なゲームまで作成するために必要なすべてを提供します。`HTMLCanvasElement`、`CanvasRenderingContext2D`、`CanvasGradient`、`CanvasPattern`、`TextMetrics`のようなインターフェースがあり、JavaScriptファイルでグラフィックスを作成するためのメソッドやプロパティを提供しています。

`Canvas` APIがどのように動作するか見ていきましょう。これでJavaScriptでグラフィックスを作成し始められます。

まず、HTMLファイルに`canvas`要素を作成する必要があります。

```html
<canvas id="my-canvas"></canvas>
```

`canvas`要素は`HTMLCanvasElement`インターフェースで表されており、これが操作のためのメソッドやプロパティを提供します。さらに、`Canvas` APIの他のインターフェースのメソッドやプロパティも使えます。

HTML内で`canvas`に`width`や`height`を指定できます。

```html
<canvas id="my-canvas" width="400" height="400"></canvas>
```

または、`width`インターフェースの`height`や`HTMLCanvasElement`プロパティを使うこともできます。

```js
const canvas = document.getElementById("my-canvas");
canvas.width = 400;
canvas.height = 400;
```

現時点では画面に何も表示されません。`canvas`要素を作成した後は、`getContext()`インターフェースの`HTMLCanvasElement`メソッドでキャンバスの描画コンテキストにアクセスする必要があります。

最も一般的なコンテキストは`2d`で、2次元で描画できます。

```js
const canvas = document.getElementById("my-canvas");
const ctx = canvas.getContext('2d');
```

`ctx`変数をコンソールに出力すると、`CanvasRenderingContext2D`のメソッドやプロパティが表示され、図形、色、線などを作成するために使え、デフォルト値も確認できます。

```js
console.log(ctx);
```

2Dコンテキストを取得したら、キャンバスに描画を始められます。

`Canvas` APIは図形、線、テキストを描くためのいくつかのメソッドやプロパティを提供しています。その中の一つが`fillStyle`プロパティで、`fillRect()`メソッドと組み合わせて長方形や正方形を描けます。

:::interactive_editor

```html
<html>
  <head>
  </head>
  <body>
    <canvas id="my-canvas" width="400" height="400"></canvas>
    <script src="index.js"></script>
  </body>
</html>
```

```js
const canvas = document.getElementById("my-canvas");

const ctx = canvas.getContext("2d");

// Set the background color
ctx.fillStyle = "crimson";

// Draw a rectangle
ctx.fillRect(1, 1, 150, 100);
```

:::

`fillRect`はx軸、y軸、幅、高さを表す4つの数値を取ります。

画面に何か表示されましたね。テキストを描いたり、アニメーションを作成することもできます。こちらはテキストを表すキャンバスです。

```html
<canvas id="my-text-canvas" width="300" height="70"></canvas>
```

テキストを描くには、`fillText()`メソッドの最初の引数にテキストを渡し、続けてx軸とy軸の値を渡します。

:::interactive_editor

```html
<html>
  <head>
  </head>
  <body>
    <canvas id="my-text-canvas" width="300" height="70"></canvas>
    <script src="index.js"></script>
  </body>
</html>
```

```js
const textCanvas = document.getElementById("my-text-canvas");

const textCanvasCtx = textCanvas.getContext("2d");

// Set font family and size
textCanvasCtx.font = "30px Arial";

// Set text color
textCanvasCtx.fillStyle = "crimson";

// Draw the text
textCanvasCtx.fillText("Hello HTML Canvas!", 1, 50);
```

:::

ブラウザでの結果は赤いテキスト`Hello HTML Canvas!`になります。

`Canvas` APIではさらに多くのことができます。例えば、`requestAnimationFrame`と組み合わせてカスタムアニメーション、ビジュアライゼーション、ゲームなどを作成できます。
