# --description--

배열의 요소에 접근하려면 인덱스 번호를 사용할 수 있습니다. 인덱스 번호는 0부터 시작하므로 배열의 첫 번째 요소는 인덱스 0에 있고, 두 번째 요소는 인덱스 1에 있으며, 이런 식으로 이어집니다.

예를 들어, `fruits` 배열의 첫 번째 요소에 접근하려면 다음 코드를 사용할 수 있습니다:

```javascript
const fruits = ['apple', 'banana', 'orange'];
console.log(fruits[0]); // Output: apple
console.log(fruits[2]); // Output: orange
```

인덱스가 범위를 벗어나면 JavaScript는 `undefined`를 반환합니다. 즉, 위 예제에서 `fruits[3]`에 접근하려고 하면 `undefined`를 반환합니다.
