# --description--

두 개의 문자열을 결합해 하나의 메시지를 만들고 싶다고 가정해 보세요. 예를 들어 아래와 같습니다:

```js
const str1 = "Hello";
const str2 = "World!";
const message = str1.push(str2);
```

<img src="https://cdn.statically.io/gh/TheOdinProject/curriculum/4ed59981b4ce2c60b5b83bf7415d3127b61821f5/foundations/javascript_basics/understanding_errors/imgs/03.png" style="width:100%" alt="Chrome 개발자 콘솔에서 타입 오류 예시">

여기서 `TypeError`가 나타나며 `str1.push is not a function`라는 메시지가 표시됩니다. 이 오류 메시지는 학습자들이 혼란스러워하는 흔한 오류입니다. 왜냐하면 `.push()`가 확실히 함수라는 것을 알고 있을 수 있기 때문입니다(예를 들어, 이전에 배열에 항목을 추가할 때 사용했다면).

하지만 핵심은 `.push()`가 문자열 메서드가 아니라 배열 메서드라는 점입니다. 따라서 문자열 메서드로서 찾을 수 있는 “함수가 아님”입니다. `.push()`를 문자열 메서드인 `.concat()`로 바꾸면 코드가 의도한 대로 실행됩니다!

`TypeError`에 직면했을 때 기억할 좋은 팁은 메서드나 연산을 실행하려는 데이터 타입을 고려하는 것입니다. 아마도 생각한 타입이 아니거나 그 타입과 호환 가능한 연산이나 메서드가 아닐 가능성이 큽니다.

## --text--

다음 JavaScript 코드가 `TypeError`가 되는 이유는 무엇인가요?

```js
const str1 = "Hello";
const str2 = "World!";
const message = str1.push(str2);
```

## --answers--

`.push()` 메서드는 문자열이 아니라 객체에만 사용할 수 있습니다.

---

`.push()` 메서드는 배열 메서드이기 때문에 문자열에서는 사용할 수 없습니다.

---

`.push()` 메서드 이름이 잘못되어 .pusch()로 써야 합니다.

---

`TypeError`는 JavaScript 코드의 구문 오류 때문입니다.
