# --description--

Para un entero positivo $n$ y dígitos $d$, definimos $F(n, d)$ como el número de divisores de $n$ cuyos últimos dígitos son iguales a $d$.

Por ejemplo, $F(84, 4) = 3$. Entre los divisores de 84 (1, 2, 3, 4, 6, 7, 12, 14, 21, 28, 42, 84), tres de ellos (4, 14, 84) tienen el último dígito 4.

También podemos verificar que $F(12!, 12) = 11$ y $F(50!, 123) = 17\\,888$.

Encuentra $F({10}^6!, 65\\,432) \text{ módulo } ({10}^{16} + 61)$.
