# --description--

ليكن $a_k$ و $b_k$ و $c_k$ تمثل الحلول الثلاثة (أعداد حقيقية أو مركبة) للتعبير $\frac{1}{x} = {\left(\frac{k}{x} \right)}^2 (k + x^2) - kx$.

على سبيل المثال، عندما يكون $k = 5$، نرى أن $\\{a_5, b_5, c_5\\}$ تقريبًا يساوي $\\{5.727244, -0.363622 + 2.057397i, -0.363622 - 2.057397i\\}$.

ليكن $S(n) = \displaystyle\sum_{p = 1}^n \sum_{k = 1}^n {(a_k + b_k)}^p {(b_k + c_k)}^p {(c_k + a_k)}^p$ لكل الأعداد الصحيحة $p$ و $k$ بحيث $1 ≤ p, k ≤ n$.

من المثير للاهتمام أن $S(n)$ دائمًا عدد صحيح. على سبيل المثال، $S(4) = 51\\,160$.

احسب $S({10}^6) \text{ modulo } 1\\,000\\,000\\,007$.
