# --explanation--

Um `operating system` é o software principal que permite que outros programas sejam executados em um computador.Ele gerencia os recursos de hardware e software, garantindo que tudo funcione junto de forma harmoniosa.Por exemplo:

- `The app is compatible with any operating system.` - O aplicativo funciona em todos os tipos de sistemas como Windows ou MacOS.

- `I installed a new operating system on my laptop.` - Você configurou um sistema diferente para rodar seu computador.
