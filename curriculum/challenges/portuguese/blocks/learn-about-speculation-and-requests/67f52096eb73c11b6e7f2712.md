`physical access`

### --feedback--

Esta frase de duas palavras refere-se a alguém entrando fisicamente em uma área restrita em vez de invadir um sistema remotamente.

# --explanation--

`Physical access` significa obter entrada em um local, como um escritório ou sala de servidores.

Uma `physical access breach` acontece quando alguém entra em um local seguro sem permissão.Por exemplo:

- `To avoid a physical access breach, we should require ID checks at the entrance.` - Uma empresa pode prevenir a entrada não autorizada verificando as identificações na porta.

- `A physical access breach can happen if someone enters the server room without permission.` - Um problema de segurança física pode ocorrer se alguém entrar na sala do servidor sem autorização.
