# --interactive--

JavaScriptでは、文字列は文字のシーケンスとして扱われ、文字列内の各文字にはブラケット記法を使ってアクセスできます。これにより、文字列の中の特定の位置にある文字を取得できます。この位置のことをインデックスと呼びます。

インデックスは文字列内の文字の位置で、ゼロベースです。つまり、文字列の最初の文字はインデックス`0`、2番目の文字はインデックス`1`、というようになります。

例えば、文字列`hello`では、文字`h`がインデックス`0`にあり、`e`はインデックス`1`、`l`はインデックス`2`にあります。

ブラケット記法は大括弧（`[]`）とアクセスしたい文字のインデックスを使います。例を見てみましょう。

:::interactive_editor

```js
let greeting = "hello";
console.log(greeting[1]); // "e"
```

:::

この例では、インデックス`1`の文字、つまり`e`にアクセスできます。

文字列の最後の文字を取得したい場合は、文字列の長さから1を引いた値を使います。
文字列の`length`プロパティは文字数を教えてくれるので、最後の文字にアクセスするには長さから1を引きます。

:::interactive_editor

```js
let greeting = "hello";
console.log(greeting[greeting.length - 1]); // "o"
```

:::

この場合、`length`の`hello`は`5`で、最後の文字（`o`）はインデックス`4`、つまり`5 - 1`にあります。

複数の文字を取得したい場合は、次のようにブラケット記法を使えます。

:::interactive_editor

```js
let greeting = "hello";
let firstTwo = greeting[0] + greeting[1]; // "he"
console.log(firstTwo);
```

:::

この例では、ブラケット記法を使って1番目と2番目の文字を連結し、文字列`he`を作っています。

ブラケット記法は、名前のイニシャルを取り出したり、特定の文字をバリデーションでチェックしたりするときに便利です。
