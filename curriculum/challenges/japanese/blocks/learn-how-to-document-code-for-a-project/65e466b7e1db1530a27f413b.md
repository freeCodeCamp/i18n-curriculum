# --description--

コンピュータのファイルシステムでは、`root directory` はメインまたは最上位のディレクトリを指します。これはファイリングキャビネットの一番上の引き出しのようなもので、他のすべてのフォルダーやファイルを含んでいます。

`main folder` は、プロジェクトや一連のファイルが保存されている主要なフォルダーです。

`subfolder` は、最上位の引き出しの中にある小さな引き出しのようなもので、別のフォルダーの中にあります。

たとえば、`MyProject` というプロジェクトでは、`root directory` は `MyProject` フォルダーであり、その中の `Images` のようなフォルダーは `subfolder` になります。

## --text--

プロジェクトの文脈で `root directory` とは何ですか？

## --answers--

メインフォルダー

---

サブフォルダー

### --feedback--

サブフォルダーは別のフォルダーの中にあります。`root directory` はメインフォルダーであり、サブフォルダーではありません。

---

ファイルタイプ

### --feedback--

`root directory` はフォルダーであり、ファイルタイプではありません。プロジェクトのファイルやフォルダーが保存される主要な場所です。

---

プログラミング言語

### --feedback--

`root directory` はファイルシステムの概念であり、プログラミング言語ではありません。プロジェクトのメインフォルダーを指します。
