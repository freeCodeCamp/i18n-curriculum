# --description--

Normalerweise unterscheidet sich ein Adjektiv von einem Adverb durch die Endung `-ly`, die an ein Adjektiv angehängt wird, um ein Adverb zu bilden. Beispiele: `extreme` (Adjektiv) x `extremely` (Adverb) / `quick` (Adjektiv) x `quickly` (Adverb).

Wissen Sie auch, dass es möglich ist, ein Adverb wie `extremely` zu verwenden, um ein Adjektiv oder sogar ein anderes Adverb zu verstärken. Wenn Sie sagen, dass etwas `quickly` passiert, ist es eine schnelle Handlung. Wenn es `extremely quickly` ist, ist die Handlung super schnell.

Füllen Sie die Lücken entsprechend dem, was Sarah im Dialog sagt.
