# --description--

Um dos atributos é `allow`. É como uma lista de permissões que informa ao navegador quais recursos o `iframe` tem permissão para usar.

Aqui está um elemento `iframe` com o atributo `allow`:

```html
<iframe
  allow="accelerometer autoplay clipboard-write encrypted-media gyroscope picture-in-picture web-share"
></iframe>
```

Adicione o atributo `allow` com o valor `accelerometer`, `autoplay` e `clipboard-write`.

`accelerometer` permite que o `iframe` use sensores de movimento para que ele possa detectar coisas como inclinação e rotação do dispositivo. `autoplay` permite que o vídeo comece a ser reproduzido automaticamente e `clipboard-write` permite que o iframe escreva dados na área de transferência do usuário.
