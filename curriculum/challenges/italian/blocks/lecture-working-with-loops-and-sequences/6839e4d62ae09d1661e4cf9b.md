# --description--

Nelle ultime lezioni ti sei abituato a lavorare con loop come questo:

```py
even_numbers = []

for num in range(21):
    if num % 2 == 0:
        even_numbers.append(num)

print(even_numbers)
```

Questo esempio crea una nuova lista vuota chiamata `even_numbers` e itera su una sequenza di numeri tra `0` e `20`. All’interno del loop, c’è una condizione che verifica se il numero corrente ha un resto di `0` quando viene diviso per `2`. Questo serve a determinare se il numero è pari. Se la condizione è `True`, allora il `num` corrente viene aggiunto in coda alla lista `even_numbers`. Infine, stampiamo la lista `even_numbers` sulla console.

Anche se questo codice funziona, esiste un modo più conciso per scriverlo usando invece la list comprehension. La list comprehension ti permette di creare una nuova lista in una sola riga combinando direttamente un loop e una condizione all’interno delle parentesi quadre. Questo rende il codice più breve e spesso più facile da leggere.

Ecco l’esempio rifattorizzato di prima usando le parentesi quadre:

```py
even_numbers = [num for num in range(21) if num % 2 == 0]
print(even_numbers)
```

In questo esempio rifattorizzato, la lista `even_numbers` viene creata con una sola riga di codice. La list comprehension itera sui numeri da `0` a `20` e include solo quelli divisibili per `2`. Questo approccio è più compatto ed elimina la necessità di un loop e di un blocco condizionale separati.

Diamo un’occhiata a un altro esempio per capire meglio come funziona la list comprehension:

```py
numbers = [1, 2, 3, 4, 5]
result = [(num, 'Even') if num % 2 == 0 else (num, 'Odd') for num in numbers]
print(result)
```

In questo esempio, abbiamo una lista di numeri e vogliamo creare una nuova lista di tuple che indicano quali numeri sono pari o dispari. Nella prima parte della list comprehension, usiamo un’istruzione `if` per verificare se il numero è divisibile per `2`. Se sì, il risultato è una tupla con quel numero seguito dalla parola `Even`. Altrimenti, il risultato è una tupla con il numero seguito dalla parola `Odd`.

Ecco come appare il risultato stampato sulla console:

```py
[(1, 'Odd'), (2, 'Even'), (3, 'Odd'), (4, 'Even'), (5, 'Odd')]
```

Un altro modo per creare una lista a partire da un iterabile esistente è la funzione `filter()`. Ecco un esempio di creazione di una nuova lista contenente solo parole più lunghe di quattro caratteri:

```py
words = ['tree', 'sky', 'mountain', 'river', 'cloud', 'sun']

def is_long_word(word):
    return len(word) > 4

long_words = list(filter(is_long_word, words))
print(long_words) # ['mountain', 'river', 'cloud']
```

La funzione `filter()` viene usata per selezionare elementi da un iterabile che soddisfano una condizione specifica. La funzione `filter()` accetta come argomenti una funzione e un iterabile. In questo esempio, stiamo passando una funzione `is_long_word` alla funzione `filter()` per verificare se il conteggio dei caratteri della parola corrente è maggiore di `4`. Tutte le parole che hanno un numero di caratteri maggiore di `4` vengono aggiunte in una nuova lista e assegnate alla variabile `long_words`.

Oltre alla funzione `filter()`, ci sono altre funzioni utili quando si lavora con le liste. Un’altra funzione da conoscere è la funzione `map()`, che prende un iterabile e applica una funzione a ciascuno dei suoi elementi. Ecco un esempio di utilizzo della funzione `map()` per convertire una lista di temperature da Celsius a Fahrenheit:

```py
celsius = [0, 10, 20, 30, 40]

def to_fahrenheit(temp):
    return (temp * 9/5) + 32

fahrenheit = list(map(to_fahrenheit, celsius))
print(fahrenheit) # [32.0, 50.0, 68.0, 86.0, 104.0]
```

Proprio come la funzione `filter()`, `map()` accetta come argomenti una funzione e un iterabile. La funzione `to_fahrenheit` prende una temperatura e la converte da Celsius a Fahrenheit.

L’ultima funzione che vedremo è la funzione `sum()`. Questa funzione serve per ottenere la somma da un iterabile come una lista o una tupla. Ecco un esempio di utilizzo della funzione `sum()`:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers)
print(total) # Result: 50
```

Puoi anche passare un argomento `start` opzionale che imposta il valore iniziale per la somma. Ecco un esempio aggiornato che usa l’argomento `start` come argomento posizionale:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, 10) # positional argument
print(total) # 60
```

Puoi anche scegliere di usare l’argomento `start` come argomento con nome in questo modo:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, start=10) # keyword argument
print(total) # 60
```

Entrambe le versioni produrranno lo stesso risultato, ma l’argomento con nome è un po’ più esplicito.

La list comprehension così come altre funzioni come `map()`, `filter()` e `sum()` possono sembrare un po’ confuse all’inizio. Ma con abbastanza pratica e tempo, inizierai a sentirti più a tuo agio nell’usarle nei tuoi programmi Python.
