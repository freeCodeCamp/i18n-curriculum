# --description--

Ora che conosci i modi più comuni per iterare su valori e array, ci sono altri modi per iterare su array. Gli array hanno un metodo integrato chiamato `map()`, che serve a creare un nuovo array applicando una funzione a ogni elemento dell'array originale. Il metodo `map()` non modifica l'array originale. La sintassi del metodo `map()` è la seguente:

```javascript
const array = [1, 2, 3, 4, 5];

const newArray = array.map((arrayValue) => {
  return arrayValue * 2;
});

console.log(newArray); // Output: [2, 4, 6, 8, 10]
```

Il metodo `map()` crea un nuovo array applicando la funzione `(arrayValue) => { return arrayValue * 2; }` a ogni elemento dell'array originale. Questo è particolarmente utile quando vuoi trasformare gli elementi di un array senza modificare l'array originale.
