# --description--

Windows에서 탐색기를 사용해 파일을 생성하려면 왼쪽 상단에 있는 "새로 만들기" 옵션을 클릭하세요. 그러면 생성할 수 있는 다양한 파일 유형 목록이 표시됩니다.

또한 해당 애플리케이션에서 파일을 생성할 수도 있습니다. 텍스트 파일을 만들어 과정을 보여드리겠습니다.

"새로 만들기"를 클릭하고 파일 유형을 선택하면 현재 폴더에 기본 이름이 지정된 새 텍스트 파일이 나타납니다. 원한다면 이름을 변경할 수 있습니다.

폴더를 생성하는 과정도 똑같습니다. 왼쪽 상단의 "새로 만들기" 버튼을 클릭한 후 "폴더" 옵션을 선택하거나 빈 공간에서 마우스 오른쪽 버튼을 클릭해 "새로 만들기"로 이동한 다음 "폴더"를 선택하세요. 그러면 현재 위치에 새 빈 폴더가 생성됩니다.

파일과 폴더 이름을 변경하려면 선택한 후 상단의 "이름 바꾸기" 옵션을 클릭하세요. 현재 파일 이름이 강조 표시되면 새 이름을 입력할 수 있습니다. 변경 사항을 확인하려면 Enter 키를 누르세요.

또는 파일이나 폴더를 마우스 오른쪽 버튼으로 클릭해 "이름 바꾸기"를 선택하거나 F2 키보드 단축키를 사용할 수도 있습니다. 새 이름을 입력한 후 Enter 키를 눌러 변경 사항을 확인하세요.

파일이나 폴더를 다른 폴더로 이동하려면 대상 폴더를 탭으로 열어둔 상태에서 드래그 앤 드롭하세요.

또는 파일이나 폴더를 선택한 후 상단의 "잘라내기" 옵션을 선택할 수 있습니다.

그다음 대상 폴더로 이동해 상단의 "붙여넣기" 버튼을 클릭하세요. 이전에 파일을 복사하거나 잘라냈다면 이 버튼이 활성화됩니다.

빈 공간에서 마우스 오른쪽 버튼을 클릭한 후 붙여넣기 아이콘을 클릭해도 같은 작업을 할 수 있습니다. 이 작업은 파일이나 폴더를 원래 위치에서 제거합니다.

파일이나 폴더를 삭제하려면 선택한 후 상단의 휴지통 아이콘을 클릭하세요. 마우스 오른쪽 버튼을 클릭하면 "삭제" 옵션도 찾을 수 있습니다.

좋습니다. 이제 macOS에서 파인더를 사용해 이 작업을 하는 방법을 알아보겠습니다.

파인더는 macOS의 기본 파일 관리자입니다.

macOS에서 파일을 생성하려면 생성하려는 문서 유형을 만들 수 있는 애플리케이션을 열어야 합니다. 현재 파인더에서 직접 파일을 생성하는 내장 방법은 없습니다.

예를 들어, 간단한 일반 텍스트 파일을 생성하려면 TextEdit를 열고 파일을 저장하세요. 이 과정은 모든 파일 유형에 대해 똑같이 작동합니다.

TextEdit를 여는 방법은 여러 가지가 있습니다. 그중 하나는 Launchpad로 가는 것입니다. 그런 다음 생성하려는 파일 유형을 만들 수 있는 앱을 검색하세요. 이 경우 "TextEdit"입니다. 아이콘을 클릭해 앱을 여세요.

앱 안에 들어가면 상단 왼쪽에 있는 Apple 메뉴 아이콘 옆의 파일 메뉴에서 파일을 저장하는 옵션을 볼 수 있습니다.

앱 메뉴 옵션 중 하나에서 파일 저장 명령을 볼 수 있습니다. 보통 파일 메뉴 아래에 있습니다.

그 옆에는 파일 저장을 위한 키보드 단축키가 표시됩니다. macOS에서는 보통 `Command + S`입니다.

파일을 처음 저장할 때는 파일 이름을 입력하고 위치를 선택할 수 있는 대화 상자가 나타납니다.

보통 파일 확장자는 애플리케이션에서 자동으로 추가하지만 저장하기 전에 변경할 수 있습니다.

파일을 저장한 후에는 Dock에서 파인더 아이콘을 클릭해 파인더를 열 수 있습니다. 저장한 폴더에서 새 파일을 볼 수 있습니다.

새 폴더를 생성하려면 새 폴더를 만들 위치로 이동해 빈 공간에서 마우스 오른쪽 버튼을 클릭하세요. 옵션 목록이 나타납니다. 첫 번째 옵션은 "새 폴더"입니다.

이 옵션을 클릭하면 새 폴더 이름을 지정할 수 있습니다. 이름을 입력하고 Enter 키를 눌러 확인하세요.

파인더에서 파일과 폴더를 이동하는 방법은 여러 가지가 있습니다. 파인더의 "즐겨찾기" 섹션에 있는 폴더로 이동하려면 드래그 앤 드롭하면 됩니다.

다른 폴더로 이동하려면 두 가지 방법이 있습니다.

대상 폴더를 탭으로 열고 파일이나 폴더를 탭으로 드래그 앤 드롭하거나,

키보드 단축키를 사용할 수 있습니다. 먼저 `Command + C`로 파일을 복사한 후 새 폴더로 이동해 `Command + Option + V`로 파일이나 폴더를 붙여넣으세요. 이 작업은 원래 폴더에서 파일을 제거합니다.

파일이나 폴더를 삭제하려면 마우스 오른쪽 버튼을 클릭해 "휴지통으로 이동"을 선택하세요.

또는 Dock의 휴지통 아이콘으로 드래그 앤 드롭해도 됩니다. 같은 작업입니다.

파일이나 폴더가 휴지통에 있을 때 마우스 오른쪽 버튼을 클릭해 즉시 삭제하거나 휴지통을 비워 삭제된 모든 파일과 폴더를 영구적으로 제거할 수 있습니다.

Windows와 macOS에서 파일과 폴더를 생성, 이동, 삭제하는 방법을 아는 것은 매우 중요합니다. 이 기술을 익히면 파일을 체계적으로 정리하고 빠르게 찾을 수 있으며, 불필요한 파일을 삭제해 컴퓨터 저장 공간을 확보할 수 있습니다.
