## --text--

Qual frase usa `I'd rather` corretamente?

## --answers--

`I'd rather use Python for data analysis than JavaScript.`

---

`I'd rather to use Python for data analysis than JavaScript.`

### --feedback--

`Rather` é seguido pela forma base do verbo, não por `to`.

---

`I'd rather using Python for data analysis than JavaScript.`

### --feedback--

`Rather` é seguido pela forma base do verbo, não por `using`.

---

`I'd rather use Python to data analysis than JavaScript.`

### --feedback--

A frase `to data analysis` deve ser `for data analysis`.

# --explanation--

`I'd rather` é uma frase usada para expressar preferência entre duas ações ou opções.É seguido pela forma base do verbo.
