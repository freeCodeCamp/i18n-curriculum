# --description--

根据你所做的工作类型，你可能更多地处理文本片段而不是数字。**string** 是一段文本……并且是语言的基本创建块。

超文本标记语言为文本提供结构和意义，CSS 允许我们精确地为其设置样式，而 JavaScript 提供了许多操作字串的特色。这些包括创建自定义欢迎消息和提示符，在需要时显示正确的文本标签，将术语排序到所需的顺序，等等。

字串是 JavaScript 中的基本数据类型。它们用于表现文本，并用单引号、双引号或反引号包裹：

```javascript
let greeting = "Hello World!";

let greeting2 = 'I am learning JavaScript!';

let greeting3 = `with The Odin Project!`;
```

使用单引号声明的字串和使用双引号声明的字串是相同的，选择哪种取决于个人偏好——尽管选择一种风格并在你的代码中保持一致的做法是好的实践。

字串允许在 JavaScript 中进行文本操作，包括但不限于创建自定义消息、排序术语以及使用 CSS 样式动态更新超文本标记语言内容。
