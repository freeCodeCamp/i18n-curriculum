# --description--

¡Bienvenido al desafío diario número 250!

Dada una cadena codificada, decodifícala usando una clave de cifrado y devuélvela.

Para encontrar la clave:

- Observa todos los desafíos diarios hasta hoy cuyo número de desafío sea múltiplo de 25 (incluyendo este).
- Toma la primera letra de cada uno de esos títulos de desafío y combínalas en una cadena. Si el título comienza con un carácter que no es letra, busca su primera letra.

Para decodificar el mensaje, recorre cada letra del mensaje codificado y:

  1. Mira la letra correspondiente en la clave (repite la clave si el mensaje es más largo que la clave).
  2. Convierte la letra de la clave a su número correspondiente: `"A"` = 1, `"B"` = 2, ..., `"Z"` = 26.
  3. Desplaza la letra codificada hacia atrás en el alfabeto esa cantidad de lugares.
  4. Si el desplazamiento pasa antes de `"A"`, continúa desde `"Z"`.

Por ejemplo, si el mensaje codificado comienza con `"Y"` y la primera letra de la clave es `"V"` (22), desplaza `"Y"` 22 lugares hacia atrás para obtener `"C"`. Repite este proceso para cada letra para decodificar el mensaje completo.

- Solo se desplazan las letras, los espacios se mantienen igual.
- Todas las letras dadas y devueltas están en mayúsculas.

```js
function decode(message) {
  const key = "VLHCGMDLNH";
  let keyIndex = 0;
  return message.split('').map(char => {
    if (char === ' ') return char;
    const shift = key[keyIndex % key.length].charCodeAt(0) - 64;
    keyIndex++;
    return String.fromCharCode(((char.charCodeAt(0) - 65 - shift + 26) % 26) + 65);
  }).join('');
}
```
