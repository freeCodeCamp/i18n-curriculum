# --interactive--

다이얼로그는 사용자에게 중요한 정보나 작업을 보여줄 수 있게 합니다. HTML의 내장 `dialog` 요소를 사용하면 웹 애플리케이션에서 이러한 다이얼로그(모달과 비모달 다이얼로그 모두)를 쉽게 만들 수 있습니다.

모달 다이얼로그는 사용자가 애플리케이션이나 웹페이지의 나머지 부분에 접근하기 전에 반드시 상호작용해야 하는 다이얼로그 유형입니다. 사용자가 다이얼로그를 닫거나 폼을 제출하는 등의 작업을 완료할 때까지 다른 콘텐츠와의 상호작용을 차단합니다.

반면, 비모달 다이얼로그는 다이얼로그가 열려 있어도 사용자가 페이지나 애플리케이션의 다른 부분과 계속 상호작용할 수 있게 합니다. 나머지 콘텐츠에 대한 접근을 막지 않습니다.

이번 수업에서는 `showModal()`와 `close()` 메서드를 사용해 모달을 여닫는 방법을 배웁니다.

사용자가 모달의 특정 작업이나 메시지에 포커스를 맞추게 하려면 `showModal()` 메서드를 사용해 모달 다이얼로그를 열 수 있습니다. 이 메서드는 페이지의 다른 항목에 배경을 추가하고 비활성화합니다. 폼, 확인, 사용자 작업이 필요한 중요한 정보를 표시하는 모달에 이상적입니다.

모달 다이얼로그의 HTML은 다음과 같습니다:

:::interactive_editor

```html
<dialog id="my-modal">
  <p>This is a modal dialog.</p>
</dialog>
```

:::

현재는 모달이 초기 렌더링 시 닫혀 있어서 페이지에 아무것도 보이지 않습니다. `showModal()` 메서드를 사용하면 모달을 자동으로 열 수 있습니다:

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
</dialog>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
dialog.showModal();
```

:::

브라우저 결과는 `This is a modal dialog.` 텍스트가 있는 모달을 보여줍니다.

사용자에게 제어권을 주는 것이 가장 좋습니다. 이를 위해 버튼에 클릭 이벤트 리스너를 추가하고 `showModal()` 메서드를 사용할 수 있습니다:

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
</dialog>
<button id="open-modal-btn">Open Modal Dialog</button>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
const openButton = document.getElementById("open-modal-btn");

openButton.addEventListener("click", () => {
  dialog.showModal();
});
```

:::

다이얼로그를 표시하면서도 다이얼로그 외부 콘텐츠와 상호작용을 허용하려면 `show()` 메서드를 사용할 수 있습니다:

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
</dialog>
<button id="open-modal-btn">Open Modal Dialog</button>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
const openButton = document.getElementById("open-modal-btn");

openButton.addEventListener("click", () => {
  dialog.show();
});
```

:::

모달을 닫으려면 `dialog` 요소 안에 버튼을 추가하고 `close()` 메서드를 사용하세요:

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
  <button id="close-modal-btn">Close Modal</button>
</dialog>
<button id="open-modal-btn">Open Modal Dialog</button>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
const openButton = document.getElementById("open-modal-btn");
const closeButton = document.getElementById("close-modal-btn");

openButton.addEventListener("click", () => {
  dialog.show();
});

closeButton.addEventListener("click", () => {
  dialog.close();
});
```

:::
