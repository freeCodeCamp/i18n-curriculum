# --description--

`Different from` 是你用来表示两件事物不相同的短语。例如：

- `My phone is different from yours.` - 这意味着这两部手机不一样。

- `Her job is different from mine.`——这意味着她的工作和你的工作不同。

注意 `different` 是形容词，且在此结构中通常后接 `from`。
