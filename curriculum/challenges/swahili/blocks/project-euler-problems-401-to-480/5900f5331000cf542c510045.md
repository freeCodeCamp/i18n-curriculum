# --description--

Katika usemi ufuatao $x$, $y$, na $n$ ni nambari kamili chanya.

$$\frac{1}{x} + \frac{1}{y} = \frac{1}{n}$$

Kwa kikomo $L$ tunaeleza $F(L)$ kama idadi ya suluhisho zinazokidhi $x < y ≤ L$.

Tunaweza kuthibitisha kuwa $F(15) = 4$ na $F(1000) = 1069$.

Tafuta $F({10}^{12})$.
