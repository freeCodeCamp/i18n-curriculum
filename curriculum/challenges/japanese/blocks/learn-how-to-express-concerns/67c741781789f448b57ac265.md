# --explanation--

`There's always` は、何度も繰り返し起こる、または決して終わらないように感じるネガティブなことについて話すときによく使われます。どんなに努力しても、そのネガティブな出来事が続いているように感じられます。例えば：

`There's always another problem to fix.` - これは問題が次々と現れ、状況が終わりのないものに感じられることを意味します。
