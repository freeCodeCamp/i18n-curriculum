# --interactive--

`superscript` 元素用於將一段文字顯示為上標。上標是列印在正常文字行上方的符號或字母。

這裡是一個使用上標元素來說明指數的範例（要查看預覽，請啟用互動式編輯器）：

:::interactive_editor

```html
<p>2<sup>2</sup> (2 squared) is 4.</p>
```

:::

在這個範例中，數字 2 被包裹在 `sup` 頁籤中，以表述段落內的上標。在預覽視窗中，你會看到第二個數字 2 比第一個數字 2 較小且稍微偏高。

上標元素的常見使用情境包括指數、上標字母和序數字。以下是一個使用上標元素表示上標字母的範例（要查看預覽，請啟用互動式編輯器）： 

:::interactive_editor

```html
<p>
  Monseigneur is often written as <strong>M<sup>gr</sup></strong>.
</p>
```

:::

上標字母是指以上標形式書寫的字母，通常用來顯示縮寫。在此範例中，字母 `g` 和 `r` 被包裹在上標標記中，以說明縮寫。

重要的是要注意，`superscript` 元素應僅用於排版原因。如果你想要將一段文字樣式設為上升基線，則應該使用 CSS，而不是 `superscript` 元素。

要在 HTML 中表述化學方程式，你會使用下標元素。此元素使用下標，顯示較小文字且基線降低。

這裡是一個使用下標元素來顯示二氧化碳化學表現的範例（要查看預覽，請啟用互動式編輯器）：

:::interactive_editor

```html
<p>CO<sub>2</sub></p>
```

:::

數字二被包裹在 `sub` 頁籤中，以說明該字元應為下標。

下標元素的常見使用案例包括化學式、註腳和變數下標。
