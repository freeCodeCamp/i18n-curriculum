# --description--

`margin-bottom` を `5px` に変更すると見た目が良くなります。しかし、`Cinnamon Roll` メニュー項目と2番目の `hr` 要素の間のスペースが、上部の `hr` 要素と `Coffee` 見出しの間のスペースと一致していません。

`bottom-line` プロパティに `25px` を使って、`margin-top` というクラスを作成し、スペースをもう少し増やしてください。

```js
const hasBottomLine = new __helpers.CSSHelp(document).getStyle('.bottom-line');
```

```js
const hasMarginTop = new __helpers.CSSHelp(document).getCSSRules().some(x => x.style['margin-top'] === '25px');
```
