# --description--

Ein Schaltjahr ist ein Jahr, das durch `4` teilbar ist, außer für Jahre, die durch `100` teilbar sind und nicht durch `400`. Zum Beispiel ist `2000` ein Schaltjahr, aber `1900` nicht. Außerdem hat ein Schaltjahr einen zusätzlichen Tag im Februar, nämlich den 29. Tag des Monats.

**Ziel:** Erfüllen Sie die untenstehenden User Stories und bestehen Sie alle Tests, um das Labor abzuschließen.

**User Stories:**

1. Definieren Sie eine Funktion namens `isLeapYear`, die eine Zahl als Argument annimmt.
2. Deklarieren Sie außerhalb der Funktion eine Variable `year`, die den Wert des Jahres speichert, das Sie überprüfen möchten.
3. Verwenden Sie innerhalb der Funktion eine `if/ else`-Anweisung oder einen ternären Operator, um zu prüfen, ob das Jahr ein Schaltjahr ist.
4. Um zu prüfen, ob das Jahr ein Schaltjahr ist, erfüllen Sie die folgenden Bedingungen:

- Wenn das Jahr durch `4` teilbar ist, dann ist es ein Schaltjahr.
   - Es sei denn, das Jahr ist auch durch `100` teilbar, dann ist es kein Schaltjahr.
   - Es sei denn, das Jahr ist auch durch `400` teilbar, dann ist es ein Schaltjahr.

5. Wenn das Jahr ein Schaltjahr ist, geben Sie `[year] is a leap year.` zurück. Andernfalls geben Sie `[year] is not a leap year.` zurück. Sie ersetzen `[year]` durch den in der `isLeapYear`-Funktion definierten Parameter.
6. Sie sollten die `isLeapYear`-Funktion mit `year` als Argument aufrufen und das Ergebnis einer Variablen namens `result` zuweisen.
7. Sie sollten die `result`-Variable mit `console.log()` in der Konsole ausgeben.

function isLeapYear(year) {
  if ((year % 4 === 0 && year % 100 !== 0) || year % 400 === 0) {
    return(year + " is a leap year.");
  } else {
    return(year + " is not a leap year.");
  }
}

const year = 2024;
const result = isLeapYear(year);
