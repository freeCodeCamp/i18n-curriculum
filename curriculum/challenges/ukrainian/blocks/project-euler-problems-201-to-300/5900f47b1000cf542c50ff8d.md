# --description--

Для додатного числа $n$ визначимо $S(n)$ як суму цілих чисел $x$, для яких $1 < x < n$ та $x^3 \equiv 1\bmod n$.

Коли $n = 91$, існує 8 можливих значень для $x$, а саме: 9, 16, 22, 29, 53, 74, 79, 81. Отже, $S(91) = 9 + 16 + 22 + 29 + 53 + 74 + 79 + 81 = 363$.

Знайдіть $S(13\\,082\\,761\\,331\\,670\\,030)$.
