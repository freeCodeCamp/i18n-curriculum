# --description--

Lernen wir etwas über das HTML-Boilerplate.

Was ist das HTML-Boilerplate, fragen Sie? Es ist wie eine fertige Vorlage für Ihre Webseiten. Denken Sie daran wie an das Fundament eines Hauses. Ein Boilerplate enthält die Grundstruktur und die wesentlichen Elemente, die jedes HTML-Dokument benötigt. Es spart Ihnen Zeit und hilft sicherzustellen, dass Ihre Seiten richtig eingerichtet sind. Hier ist ein Beispiel:

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta
       name="viewport"
       content="width=device-width, initial-scale=1.0" />
    <title>freeCodeCamp</title>
    <link rel="stylesheet" href="./styles.css" />
  </head>
  <body>
  </body>
</html>
```

Lassen Sie uns die wichtigsten Teile dieses Boilerplates aufschlüsseln. Zuerst gibt es die `DOCTYPE`-Deklaration:

```html
<!DOCTYPE html>
```

Sie teilt den Browsern mit, welche HTML-Version Sie verwenden. Als Nächstes folgt das `html`-Tag:

```html
<!DOCTYPE html>
<html lang="en">
  <!--All other elements go inside here-->
</html>
```

Dieses umschließt Ihren gesamten Inhalt und kann die Sprache Ihrer Seite angeben. Innerhalb des `html`-Tags finden Sie zwei Hauptabschnitte – einen `head` und einen `body`:

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <!--Important metadata goes here-->
  </head>
  <body>
    <!--Headings, paragraphs, images, etc. go inside here-->
  </body>
</html>
```

Der `head`-Abschnitt enthält wichtige Informationen hinter den Kulissen:

```html
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Document Title Goes Here</title>
  <link rel="stylesheet" href="./styles.css" />
</head>
```

Die Metadaten Ihrer Seite, enthalten in `meta`-Elementen, geben Details an wie die Zeichencodierung und wie Websites wie Twitter den Link Ihrer Seite in der Vorschau anzeigen sollen. Der Titel Ihrer Seite, zu finden im `title`-Element, bestimmt den Text, der im Browser-Tab oder -Fenster erscheint. Schließlich verlinken Sie normalerweise die externen Stylesheets Ihrer Seite im `head`-Abschnitt mit `link`-Elementen.

Der `body`-Abschnitt ist der Bereich, in dem Ihr gesamter Inhalt steht:

```html
<body>
  <h1>I am a main title</h1>
  <p>Example paragraph text</p>
</body>
```

Warum ist ein Boilerplate wichtig? Es stellt sicher, dass Ihre Seiten korrekt strukturiert sind und in verschiedenen Browsern gut funktionieren. Die Verwendung eines Boilerplates hilft Ihnen, häufige Fehler zu vermeiden und bewährte Vorgehensweisen einzuhalten. Es ist ein großartiger Ausgangspunkt für jedes Web-Projekt. Denken Sie daran, dass Sie Ihr eigenes Boilerplate an Ihre Bedürfnisse anpassen können. Mit zunehmender Erfahrung fügen Sie vielleicht Ihre bevorzugten Elemente oder `meta`-Tags hinzu. Wenn Sie Ihr persönliches Boilerplate weiter verbessern, werden Sie feststellen, dass es Ihnen Zeit spart, wenn Sie neue Projekte starten.

Wenn Sie das nächste Mal eine neue HTML-Datei beginnen, sollten Sie ein Boilerplate verwenden. Es gibt Ihnen definitiv ein solides Fundament zum Aufbau.
