# --description--

このタスクは関係代名詞 `that` に焦点を当てています。これは名詞についての詳細を提供する節を導入するために使われます。

節とは、主語と動詞を少なくとも含む文の一部で、完全な意味を表すこともあれば、主文に追加情報を加えることもあります。

代名詞 `that` は、何かをより明確に特定または定義するのに役立ち、それを説明や必要な詳細に結びつけます。

例えば、`She read the book that won the Pulitzer Prize last year.` この文では、`that won the Pulitzer Prize last year` が `that` によって導入された関係節です。この節は本を説明または特定し、他の本と区別するのに役立ちます。ここで `that` は関係代名詞として節を名詞 `book` に結びつけ、彼女が読んだ特定の本を説明しています。
