# --description--

可以证明多项式 $n^4 + 4n^3 + 2n^2 + 5n$ 对于每个整数 $n$ 都是 6 的倍数。还可以证明 6 是满足此属性的最大整数。

定义 $M(a, b, c)$ 为满足多项式 $n^4 + an^3 + bn^2 + cn$ 对所有整数 $n$ 都是 $m$ 的倍数的最大 $m$。例如，$M(4, 2, 5) = 6$。

另外，定义 $S(N)$ 为所有 $0 < a, b, c ≤ N$ 的 $M(a, b, c)$ 之和。

我们可以验证 $S(10) = 1\\,972$ 和 $S(10\\,000) = 2\\,024\\,258\\,331\\,114$。

令 $F_k$ 为斐波那契序列：

- $F_0 = 0$，$F_1 = 1$ 且
- $F_k = F_{k - 1} + F_{k - 2}$，当 $k ≥ 2$ 时。

查找 $\sum S(F_k)$ 在 $2 ≤ k ≤ 1\,234\,567\,890\,123$ 范围内的最后 9 位数字。
