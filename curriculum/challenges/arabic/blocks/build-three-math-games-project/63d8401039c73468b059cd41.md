# --instructions--

لهذا التحدٍّ، تحتاج إلى إنشاء ثلاثة ألعاب رياضية باستخدام Python تقوم بما يلي:

- لعبة مخطط النقاط
  - توليد نقاط عشوائية على رسم بياني ويجب على اللاعب إدخال إحداثيات (x,y)
  - لزيادة الصعوبة، اجعل الرسم البياني أكبر
- لعبة تدريب الجبر
  - توليد مسائل بخطوة واحدة وخطوتين بقيم صحيحة عشوائية ويجب على اللاعب إدخال الإجابة
  - استخدم قيمًا موجبة وسالبة. لزيادة الصعوبة، اجعل الأرقام أكبر
- لعبة المقذوفات
  - عرض "جدار" بارتفاع وموقع عشوائي. يجب على اللاعب تحريك منزلقات لضبط مسار قطع مكافئ لتجاوز الجدار
  - لزيادة الصعوبة، اجعل مستوى ثاني حيث يدخل اللاعبون a و b و c بدون منزلقات

عند الانتهاء، أرسل عنوان URL لدفتر ملاحظات Colab العام على Google Drive الخاص بك.
