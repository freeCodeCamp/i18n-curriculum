# --description--

Come hai già visto, le funzioni possono restituire un valore usando la parola chiave `return`. La parola chiave `return` serve a restituire un valore da una funzione. Quando si usa la parola chiave `return`, la funzione smette di eseguire e restituisce il valore specificato dopo la parola chiave `return`.

```js
function add(a, b) {
  return a + b
}

console.log(add(2, 3)); // Output: 5
```

Ma cosa succede se la parola chiave `return` viene usata prima della fine della funzione?
Per rispondere a questa domanda, considera il seguente esempio:

```js
function add(a, b) {
  if(a > 2){
    return b;
  }

  return a + b;
}

console.log(add(3, 7)); // Output: 7
```

Nell'esempio sopra, la funzione `add` ha un'istruzione condizionale che verifica se il valore di `a` è maggiore di `2`. Se la condizione è vera, la funzione restituirà il valore di `b` e smetterà di eseguire. Se la condizione non è vera, la funzione restituirà la somma di `a` e `b`.
