# --description--

Sasa baada ya kuelewa sintaksia ya msingi ya HTML na CSS, tutaingia kwenye mambo ya kina. Ujuzi muhimu zaidi unahitaji kuutawala katika CSS ni upangaji na mpangilio. Kubadilisha fonti na rangi ni ujuzi muhimu, lakini uwezo wa kuweka vitu mahali hasa unapotaka kwenye ukurasa wa mtandao ni muhimu zaidi. Baada ya yote, ni kurasa ngapi za mtandao unazoweza kupata ambapo kila kipengele kimewekwa tu kimoja juu ya kingine?

Kujifunza kuweka vipengele kwenye ukurasa wa mtandao si vigumu sana mara tu unapofahamu dhana chache muhimu. Kwa bahati mbaya, wanafunzi wengi hukimbilia kujifunza HTML na CSS ili kufikia JavaScript na hatimaye hukosa dhana hizi za msingi. Hii husababisha kuchanganyikiwa, maumivu, (na gif za kuchekesha) kwa sababu ujuzi wote wa JavaScript duniani hauna maana ikiwa huwezi kuweka vipengele vyako kwenye ukurasa mahali unapotakiwa. Kwa hivyo, tukizingatia hilo, tuanze sasa.
