# --description--

Em inglês, o tempo presente simples é usado para descrever ações habituais ou usuais.No entanto, quando você está falando sobre funções ou experiências passadas, como em uma entrevista, você deve usar o passado simples. 

O passado simples frequentemente envolve a mudança de verbos.Por exemplo, `is` se torna `was` para sujeitos singulares e `are` se torna `were` para sujeitos plurais.

Por exemplo, `I am a teacher` no presente se torna `I was a teacher` para experiências passadas.Da mesma forma, `They are engineers` muda para `They were engineers` quando se fala sobre o passado.
