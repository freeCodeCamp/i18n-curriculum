# --explanation--

片語 `correo electrónico` 的意思是「電子郵件」。

Sebastián 開頭用 `su` 回答，因為他指的是別人的電子郵件。在這個例子中，是 Carlos 的電子郵件。

所以 `su correo electrónico` 的意思是「他的電子郵件位址」。
