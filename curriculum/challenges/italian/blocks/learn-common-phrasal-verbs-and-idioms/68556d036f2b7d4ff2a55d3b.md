# --explanation--

Usare `that` dopo `I think` è opzionale in frasi come questa. Dire `I think that it's vital we make these updates.` è corretto. Nella conversazione quotidiana, però, è comune omettere `that` per una frase più fluida e veloce. Per esempio: 

`I think that we need more time to finish the project.` – Questa struttura è corretta. La maggior parte delle persone però direbbe `I think we need more time…`.
