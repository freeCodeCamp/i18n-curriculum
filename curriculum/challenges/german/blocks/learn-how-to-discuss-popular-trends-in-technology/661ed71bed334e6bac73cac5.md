# --description--

`Augmented Reality` (abgekürzt als `AR`) ist wie Magie für Ihre Geräte. Es ist eine spezielle Technologie, die digitale Informationen mit der realen Welt um Sie herum vermischt. Stellen Sie sich vor, Sie sehen hilfreiche Informationen, wie Wegbeschreibungen oder lustige virtuelle Kreaturen, direkt auf Ihrem Telefon oder Tablet, als wären sie Teil Ihrer alltäglichen Umgebung. Sie macht die Welt spannender, indem sie eine Schicht digitalen Zaubers zu dem hinzufügt, was Sie sehen.
