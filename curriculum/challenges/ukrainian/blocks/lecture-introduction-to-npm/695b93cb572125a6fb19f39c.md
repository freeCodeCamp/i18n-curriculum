# --description--

У попередньому уроці ви дізналися, що таке залежності та як обрати правильні для свого проєкту.

У цьому уроці ви навчитеся встановлювати та видаляти залежності за допомогою npm.

Щоб встановити залежність, використовуйте команду `npm install` ось так:

```bash
npm install dependencyName
```

Ви також можете скоротити `install` до `i` ось так:

```bash
npm i dependencyName
```

Після встановлення пакета npm додає новий розділ `dependencies` у ваш файл `package.json` (якщо його там ще немає) і перераховує встановлений пакет. Наприклад, ось як це виглядає, коли ви встановлюєте пакет `chalk`:

```json
{
  "name": "install-uninstall-deps",
  "version": "1.0.0",
  "description": "a simple demo on how to install and uninstall dependencies",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": ["npm", "deps"],
  "author": "Kolade Chris",
  "license": "ISC",
  "dependencies": {
    "chalk": "^5.6.2"
  }
}
```

Коли ви встановлюєте залежність, у кореневій папці вашого проєкту з’являються дві нові речі:

* `node_modules/`: папка, що містить фактичний код встановлених пакетів.
    
* `package-lock.json`: файл, який зберігає детальну інформацію про точні версії цих пакетів, включно з їхніми власними залежностями. (Про цей файл ви дізнаєтеся більше в одному з наступних уроків.)
    

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/how-can-you-install-and-remove-dependencies-1.png" alt="">

Більшість проєктів потребують більше ніж одну залежність. Замість того, щоб встановлювати їх по одній, npm дозволяє встановити кілька пакетів одночасно, ввівши список імен пакетів, розділених пробілами:

```bash
npm install express lodash bcrypt
```

Як і раніше, коли ви встановлюєте кілька залежностей одночасно, npm оновлює ваші файли `package.json` і `package-lock.json`, а також ваш `node_modules`, щоб включити їх. Ось як виглядає `package.json`:

```json
{
  "name": "install-uninstall-deps",
  "version": "1.0.0",
  "description": "a simple demo on how to install and uninstall dependencies",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": ["npm", "deps"],
  "author": "Kolade Chris",
  "license": "ISC",
  "dependencies": {
    "bcrypt": "^6.0.0",
    "chalk": "^5.6.2",
    "express": "^5.1.0",
    "lodash": "^4.17.21"
  }
}
```

Щоб встановити `devDependency`, додайте прапорець `--save-dev` до команди `install`. Ви також можете використати коротшу версію `-D`:

```bash
npm i nodemon -D
```

Після запуску цієї команди npm оновить ваш файл `package.json` і додасть пакет у розділ `devDependencies`. Ось приклад, як це виглядає:

```json
{
  "name": "install-uninstall-deps",
  "version": "1.0.0",
  "description": "a simple demo on how to install and uninstall dependencies",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": ["npm", "deps"],
  "author": "Kolade Chris",
  "license": "ISC",
  "dependencies": {
    "bcrypt": "^6.0.0",
    "chalk": "^5.6.2",
    "express": "^5.1.0",
    "lodash": "^4.17.21"
  },
  "devDependencies": {
    "nodemon": "^3.1.10"
  }
}
```

Кожного разу, коли ви запускаєте команду `npm install`, npm покаже повідомлення про те, чи була установка успішною. Повідомлення про успіх зазвичай виглядає приблизно так:

```bash

  run `npm fund` for details

user@Kolade ~/Desktop/fCC/script-code/node/npm/install-uninstall-deps  % npm install nodemon -D

  run `npm fund` for details

```

Щоб видалити або деінсталювати пакет, використовуйте команду `npm uninstall`:

```bash
npm uninstall dependencyName
```

Наприклад, щоб видалити `chalk` з вашого проєкту, виконайте цю команду:

```bash
npm uninstall chalk
```

Ви також можете деінсталювати кілька пакетів одночасно, перерахувавши їх через пробіли:

```bash
npm uninstall express lodash bcrypt
```

Ви отримаєте повідомлення про те, чи вдалося виконати команду деінсталяції:

```bash
user@Kolade ~/Desktop/fCC/script-code/node/npm/install-uninstall-deps  % npm uninstall chalk

added 1 package, removed 1 package, and audited 101 packages in 1s

20 packages are looking for funding
  run `npm fund` for details

found 0 vulnerabilities

user@Kolade ~/Desktop/fCC/script-code/node/npm/install-uninstall-deps  % npm uninstall express lodash bcrypt

removed 70 packages, and audited 31 in 1s

4 packages are looking for funding
  run `npm fund` for details

found 0 vulnerabilities
```

І ваш файл `package.json` буде оновлено автоматично:

```json
{
  "name": "install-uninstall-deps",
  "version": "1.0.0",
  "description": "a simple demo on how to install and uninstall dependencies",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": ["npm", "deps"],
  "author": "Kolade Chris",
  "license": "ISC",
  "devDependencies": {
    "nodemon": "^3.1.10"
  }
}
```

Щоб деінсталювати `devDependency`, просто виконайте:

```bash
npm uninstall nodemon
```

Зверніть увагу, що прапорець `-D` не потрібен під час деінсталяції. npm автоматично видалить пакет із правильного розділу, у цьому випадку `devDependencies`.

Після деінсталяції `devDependency` ваш `package.json` знову буде оновлено:

```json
{
  "name": "install-uninstall-deps",
  "version": "1.0.0",
  "description": "a simple demo on how to install and uninstall dependencies",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": ["npm", "deps"],
  "author": "Kolade Chris",
  "license": "ISC"
}
```
