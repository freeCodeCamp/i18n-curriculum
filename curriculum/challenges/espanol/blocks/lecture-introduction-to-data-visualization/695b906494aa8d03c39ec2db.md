# --description--

Cada industria, desde la música hasta la atención médica, trabaja con conjuntos de datos grandes y a menudo complejos. Estos pueden incluir registros médicos de pacientes, tendencias de listas de reproducción a lo largo del tiempo o métricas de interacción en redes sociales.

Para entender esta información, las empresas usan una variedad de herramientas para ayudar a sus equipos a analizar datos y tomar decisiones informadas sobre sus productos y servicios.

Una de las herramientas más poderosas para entender y comunicar estos datos es la **visualización de datos**.

La visualización de datos crea formatos visuales claros a partir de datos complejos. Ejemplos de estos tipos de visuales incluyen gráficos de pastel, diagramas de dispersión, gráficos de barras y más. Las buenas visualizaciones ayudan a las empresas a ahorrar tiempo identificando información clave, tendencias y relaciones en los datos.

Hay muchas herramientas populares de visualización de datos usadas en negocios, incluyendo Microsoft Excel, Power BI, Google Charts y Tableau.

En estas próximas lecciones y talleres, aprenderás a crear visualizaciones interactivas usando la biblioteca de JavaScript D3.js.

El objetivo de este módulo es introducirte a los fundamentos de la visualización de datos y guiarte en la creación de tus propias visualizaciones con JavaScript.
