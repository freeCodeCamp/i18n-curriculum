## Що таке Python?

- **Вступ**: Python — це загальна мова програмування, відома своєю простотою та легкістю у використанні. Python застосовують у багатьох сферах, таких як наука про дані та машинне навчання, веб-розробка, скриптування та автоматизація, вбудовані системи та Інтернет речей, і багато іншого.
- **Загальні випадки використання**: Python використовують у науці про дані, машинному навчанні, веб-розробці, кібербезпеці, автоматизації та на мікрокомп’ютерах, таких як Raspberry Pi і плати, сумісні з MicroPython.

## Python у вашому локальному середовищі

- **Встановлення**: Найкращий спосіб встановити Python на Windows, Mac і Linux — це скачати інсталятор з офіційного сайту Python (`https://www.python.org/`).

## Змінні

- **Оголошення змінних**: Щоб оголосити змінну, почніть із імені змінної, за яким іде оператор присвоєння (`=`) і потім значення. Це може бути число, рядок, булевий тип тощо. Ось кілька прикладів:

```py
name = 'John Doe'
age = 25
```

- **Конвенції іменування змінних**: Ось конвенції іменування, які слід використовувати для змінних:

  - Імена змінних можуть починатися лише з літери або підкреслення (_), але не з числа.
  - Імена змінних можуть містити лише алфавітно-цифрові символи (a-z, A-Z, 0-9) та підкреслення (_).
  - Імена змінних чутливі до регістру — `age`, `Age` і `AGE` вважаються різними.
  - Імена змінних не можуть бути одним із зарезервованих ключових слів Python, таких як `if`, `class` або `def`.
  - Імена змінних, що складаються з кількох слів, розділяються підкресленнями. Наприклад, `snake_case`.

## Коментарі

- **Однолінійні коментарі**: Ці типи коментарів слід використовувати для коротких нотаток, які ви хочете залишити у своєму коді.

```py
# This is a single line comment
```

- **Багатолінійні рядки**: Ці типи рядків можна використовувати для залишення більших нотаток або щоб перетворити на коментарі розділи коду.

```py
"""
This is a multi-line string.
Here is some code commented out.

name = 'John Doe'
age = 25
"""
```

- **`print()` функція**: Щоб вивести дані в консоль, ви можете використовувати функцію `print()` ось так:

```py
print('Hello world!') # Hello world!
```

## Загальні типи даних у Python

- **Вступ**: Python — це мова з динамічною типізацією, як і JavaScript, що означає, що вам не потрібно явно оголошувати типи для змінних. Мова визначає тип змінної на основі того, яке значення ви їй призначаєте.
- **Ціле число**: Ціле число без десяткових знаків:

```py
my_integer_var = 10
print('Integer:', my_integer_var) # Integer: 10
```

- **Десяткове число**: Число з десятковими знаками:

```py
my_float_var = 4.50
print('Float:', my_float_var) # Float: 4.5
```

- **Рядок**: Послідовність символів, укладена в лапки:

```py
my_string_var = 'hello'
print('String:', my_string_var) # String: hello
```

- **Булевий**: Значення, що представляє або `True`, або `False`:

```py
my_boolean_var = True
print('Boolean:', my_boolean_var) # Boolean: True
```

- **Множина**: Неупорядкована колекція унікальних елементів:

```py
my_set_var = {7, 5, 8}
print('Set:', my_set_var) # Set: {7, 5, 8}
```

- **Словник**: Колекція пар ключ-значення, укладена у фігурні дужки:

```py
my_dictionary_var = {"name": "Alice", "age": 25}
print('Dictionary:', my_dictionary_var) # Dictionary: {'name': 'Alice', 'age': 25}
```

- **Кортеж**: Незмінна впорядкована колекція, укладена в круглі дужки:

```py
my_tuple_var = (7, 5, 8)
print('Tuple:', my_tuple_var) # Tuple: (7, 5, 8)
```

- **Послідовність**: Послідовність чисел, часто використовується в циклах:

```py
my_range_var = range(5)
print(my_range_var) # range(0, 5)
```

- **Список**: Впорядкована колекція елементів, що підтримує різні типи даних:

```py
my_list = [22, 'Hello world', 3.14, True]
print(my_list) # [22, 'Hello world', 3.14, True]
```

- **None**: Спеціальне значення, що представляє відсутність значення:

```py
my_none_var = None
print('None:', my_none_var) # None: None
```

## Незмінні та змінні типи

- **Незмінні типи**: Ці типи не можуть змінюватися після оголошення, хоча ви можете вказати їхнім змінним на щось нове, що називається переназначенням. До них належать ціле число, float, complex, булевий, рядок, кортеж, range і `None`.
- **Змінні типи**: Ці типи можуть змінюватися після оголошення. Ви можете додавати, видаляти або оновлювати їхні елементи. До них належать колекційні типи, такі як list, set і dictionary.
- **Функція `type()`**: Щоб побачити тип змінної, ви можете використовувати функцію `type()` ось так:

```py
greeting = 'Hello there!'
age = 21

print(type(greeting)) # <class 'str'>
print(type(age)) # <class 'int'>
```

- **Функція `isinstance()`**: Вона використовується для перевірки, чи відповідає змінна певному типу даних:

```py
greeting = 'Hello world'
name = 'John Doe'

print(isinstance(greeting, str)) # True
print(isinstance(name, int)) # False
```

## Робота з рядками

- **Визначення**: Як ви пам’ятаєте з JavaScript, рядки є незмінними, тобто ви не можете змінити їх після створення. У Python можна використовувати як одинарні, так і подвійні лапки. Рекомендується обрати одне правило і дотримуватися його:

```py
developer = 'Jessica'
city = "Los Angeles"
```

- **Доступ до символів рядка**: Ви можете отримати доступ до символів рядка, використовуючи дужкову нотацію ось так:

```py
my_str = 'Hello world'

print(my_str[0])  # H
print(my_str[6])  # w

print(my_str[-1])  # d
print(my_str[-2]) # l
```

- **Екранізація рядків**: Ви можете використовувати зворотну скісну риску (`\`), якщо ваш рядок містить лапки, ось так:

```py
msg = 'It\'s a sunny day'
quote = "She said, \"Hello!\""
```

- **Конкатенація рядків**: Щоб об’єднати рядки, можна використати оператор `+` ось так:

```py
developer = 'Jessica'
print('My name is ' + developer + '.') # My name is Jessica.
```

Інший спосіб конкатенації рядків — використання оператора `+=`. Він виконує конкатенацію та присвоєння в одному кроці, ось так:

```py
greeting = 'My name is '
developer = 'Jessica.'

greeting += developer
print(greeting) # My name is Jessica.
```

- **`f-strings`**: Це скорочення для форматованих рядкових літералів. Вони дозволяють виконувати інтерполяцію та деяку конкатенацію з компактним і читабельним синтаксисом:

```py
developer = 'Jessica'
greeting = f'My name is {developer}.'
print(greeting) # My name is Jessica.
```

- **Вирізання рядків (string slicing)**: Це коли ви можете витягти частини рядка. Ось базовий синтаксис:

```py
str[start:stop:step]
```

Початкова позиція — це індекс, з якого починається вирізання. Кінцева позиція — де вирізання має закінчитися. Ця позиція не включається. Позиція кроку означає інтервал, з яким відбувається вирізання. Ось кілька прикладів:

```py
message = 'Python is fun!'

print(message[0:6])  # Python
print(message[7:])  # is fun!
print(message[::2])  # Pto sfn
```

- **Отримання довжини рядка**: Функція `len()` використовується, щоб повернути кількість символів у рядку:

```py
developer = 'Jessica'

print(len(developer)) # 7
```

## Робота з оператором `in`

- **Оператор `in`**: він повертає булеве значення, яке вказує, чи існує символ або символи у рядку: 

```py
my_str = 'Hello world'

print('Hello' in my_str)  # True
print('hey' in my_str)    # False
print('hi' in my_str)    # False
print('e' in my_str)  # True
print('f' in my_str)  # False
```

## Загальні методи рядків

- **`upper()`**: Цей метод рядка повертає новий рядок, у якому всі символи перетворені на верхній регістр:

```py
developer = 'Jessica'

print(developer.upper()) # JESSICA
```

- **`lower()`**: Цей метод рядка повертає новий рядок, у якому всі символи перетворені на нижній регістр:

```py
developer = 'Jessica'

print(developer.lower()) # jessica
```

- **`strip()`**: Цей метод рядка повертає копію рядка з видаленими вказаними початковими та кінцевими символами (якщо аргумент не передано, видаляє початкові та кінцеві пробіли).

```py
greeting = '  hello world  '

trimmed_my_str = greeting.strip()
print(trimmed_my_str)  # 'hello world'
```

- **`replace()`**: Цей метод повертає новий рядок, у якому всі входження старого рядка замінені на новий.

```py
greeting = 'hello world'

replaced_my_str = greeting.replace('hello', 'hi')
print(replaced_my_str)  # 'hi world'
```

- **`split()`**: Цей метод використовується для розбиття рядка на список за вказаним роздільником. Роздільник — це рядок, який визначає, де має відбутися розбиття.

```py
dashed_name = 'example-dashed-name'

split_words = dashed_name.split('-')
print(split_words)  # ['example', 'dashed', 'name']
```

- **`join()`**: Цей метод використовується для об’єднання елементів ітерованого об’єкта в рядок із роздільником. Ітерований об’єкт — це колекція елементів, по яких можна пройтися циклом, наприклад список, рядок або кортеж.

```py
example_list = ['example', 'dashed', 'name']

joined_str = ' '.join(example_list)
print(joined_str)  # example dashed name
```

- **`startswith(prefix)`**: Цей метод рядка повертає булеве значення, яке вказує, чи починається рядок із вказаного префікса:

```py
developer = 'Naomi'

result = developer.startswith('N')
print(result)  # True
```

- **`endswith(suffix)`**: Цей метод рядка повертає булеве значення, яке вказує, чи закінчується рядок вказаним суфіксом:

```py
developer = 'Naomi'

result = developer.endswith('N')
print(result)  # False
```

- **`find()`**: Цей метод рядка повертає індекс першого входження підрядка. Якщо підрядок не знайдено, повертається `-1`:

```py
developer = 'Naomi'

result = developer.find('N')
print(result)  # 0

city = 'Los Angeles'
print(city.find('New')) # -1
```

- **`count(substring)`**: Цей метод рядка рахує, скільки разів підрядок зустрічається в рядку:

```py
city = 'Los Angeles'
print(city.count('e')) # 2
```

- **`capitalize()`**: Цей метод рядка повертає новий рядок, у якому перший символ написаний з великої літери, а всі інші — у нижньому регістрі:

```py
dessert = 'chocolate cake'
print(dessert.capitalize()) # Chocolate cake
```

- **`isupper()`**: Цей метод рядка повертає `True`, якщо всі літери в рядку у верхньому регістрі, і `False`, якщо ні:

```py
dessert = 'chocolate cake'
print(dessert.isupper()) # False
```

- **`islower()`**: Цей метод рядка повертає `True`, якщо всі літери в рядку у нижньому регістрі, і `False`, якщо ні:

```py
dessert = 'chocolate cake'
print(dessert.islower()) # True
```

- **`title()`**: Цей метод рядка повертає новий рядок, у якому перша літера кожного слова написана з великої літери:

```py
city = 'los angeles'
print(city.title()) # Los Angeles
```

- **`maketrans()`**: Цей метод рядка використовується для створення таблиці відповідностей символів для трансляції. Часто його застосовують разом із методом `translate()`, який застосовує цю таблицю до рядка і повертає перекладений результат.

```py
trans_table = str.maketrans('abc', '123')
print(trans_table) # {97: 49, 98: 50, 99: 51}

result = 'abcabc'.translate(trans_table)
print(result)  # 123123
```

## Загальні операції з цілими числами та числами з плаваючою комою

- **Основні математичні операції**: у Python можна виконувати основні математичні операції з цілими числами та числами з плаваючою комою, включно з додаванням, відніманням, множенням і діленням:

```py
int_1 = 56
int_2 = 12
float_1 = 5.4
float_2 = 12.0

# Addition

print('Integer Addition:', int_1 + int_2) # Integer Addition: 68
print('Float Addition:', float_1 + float_2) # Float Addition: 17.4

# Subtraction

print('Int Subtraction:', int_1 - int_2) # Int Subtraction: 44
print('Float Subtraction:',  float_2 - float_1) # Float Subtraction: 6.6

# Multiplication

print('Int Multiplication:', int_1 * int_2) # Int Multiplication: 672
print('Float Multiplication:', float_2 * float_1) # Float Multiplication: 64.80000000000001

# Division

print('Int Division:', int_1 / int_2) # Int Division: 4.666666666666667
print('Float Division:', float_2 / float_1) # Float Division: 2.222222222222222
```

Коли ви додаєте число з плаваючою комою і ціле число, результат буде перетворено на число з плаваючою комою ось так:

```py
int_1 = 56
float_1 = 5.4

print(int_1 + float_1) # 61.4
```

- **Оператор за модулем (`%`)**: він повертає остачу від ділення одного числа на інше:

```py
int_1 = 56
int_2 = 12

print(int_1 % int_2) # 8
```

- **Цілочисельне ділення (`//`)**: цей оператор використовується для ділення двох чисел із заокругленням вниз до найближчого цілого числа:

```py
int_1 = 56
int_2 = 12

print(int_1 // int_2) # 4
```

- **Оператор піднесення до степеня (`**`)**: цей оператор використовується для піднесення числа до степеня іншого числа:

```py
int_1 = 4
int_2 = 2

print(int_1 ** int_2) # 16
```

- **Функція `float()`**: ви можете використати цю функцію, щоб перетворити ціле число на число з плаваючою комою.

```py
num = 4

print(float(num)) # 4.0
```

- **Функція `int()`**: ви можете використати цю функцію, щоб перетворити число з плаваючою комою на ціле число.

```py
num = 4.0

print(int(num)) # 4
```

- **Функція `round()`**: ця функція використовується для заокруглення числа до найближчого цілого числа:

```py
num_1 = 3.4
num_2 = 7.7

print(round(num_1)) # 3
print(round(num_2)) # 8
```

- **Функція `abs()`**: ця функція використовується для отримання абсолютного значення числа:

```py
num = -13

print(abs(num)) # 13
```

- **Функція `bin()`**: ця функція використовується для перетворення цілого числа у його двійкове представлення у вигляді рядка:

```py
num = 56

print(bin(num))  # 0b111000
```

- **Функція `oct()`**: ця функція використовується для перетворення цілого числа у його вісімкове представлення у вигляді рядка:

```py
num = 56

print(oct(num))  # 0o70
```

- **Функція `hex()`**: ця функція використовується для перетворення цілого числа у його шістнадцяткове представлення у вигляді рядка:

```py
num = 56

print(hex(num))  # 0x38
```

- **Функція `pow()`**: ця функція використовується для піднесення числа до степеня іншого числа:

```py
result = pow(2, 3) 
print(result)  # 8
```

## Розширені присвоєння

- **Визначення**: Розширене присвоєння поєднує бінарну операцію з присвоєнням в одному кроці. Воно бере змінну, застосовує до неї операцію з іншим значенням і зберігає результат назад у ту саму змінну.

```py
# Addition assignment 
my_var = 10
my_var += 5

print(my_var) # 15

# Subtraction assignment
count = 14
count -= 3

print(count) # 11

# Multiplication assignment 
product = 65
product *= 7

print(product) # 455

# Division assignment 
price = 100
price /= 4

print(price) # 25.0

# Floor Division assignment 
total_pages = 23
total_pages //= 5

print(total_pages) # 4

# Modulo assignment 
bits = 35
bits %= 2

print(bits) # 1

# Exponentiation assignment 
power = 2
power **= 3

print(power) # 8
```

Існують також інші оператори розширеного присвоєння, наприклад, для побітових операторів. Вони включають `&=`, `^=`, `>>=` і `<<=`.

## Робота з функціями

- **Визначення**: Функції — це багаторазові фрагменти коду, які приймають вхідні дані (аргументи) і повертають вихідні дані. Щоб викликати функцію, потрібно звернутися до її імені, після якого йде множина дужок:

```py
# Defining a function

def get_sum(num_1, num_2):
    return num_1 + num_2

result = get_sum(3, 4) # function call
print(result) # 7
```

Якщо функція явно не повертає значення, тоді повернене значення за замовчуванням — `None`:

```py
def greet():
    print('hello') 

result = greet() # hello
print(result) # None
```

Ви також можете задавати значення за замовчуванням для параметрів ось так:

```py
def get_sum(num_1, num_2=2):
    return num_1 + num_2

result = get_sum(3) 
print(result) # 5
```

Якщо викликати функцію без правильної кількості аргументів, ви отримаєте `TypeError`:

```py
def calculate_sum(a, b):
    print(a + b)

calculate_sum()

# TypeError: calculate_sum() missing 2 required positional arguments: 'a' and 'b'
```

```py
name = input('What is your name?') # User types 'Kolade' and presses Enter  
print('Hello', name) # Hello Kolade
```

```py
print(int(3.14)) # 3
print(int('42')) # 42
print(int(True)) # 1
print(int(False)) # 0
```

## Область видимості в Python

- **Локальна область видимості**: це коли змінна, оголошена всередині функції або класу, доступна лише в межах цієї функції або класу.

```py
def my_func():
    num = 10
    print(num)
```

- **Огортаюча область видимості**: це коли функція, вкладена всередину іншої функції, може отримувати доступ до змінних тієї функції, в якій вона вкладена.

```py
def outer_func():
    msg = 'Hello there!'
    
    def inner_func():
        print(msg)
    inner_func()

print(outer_func()) # Hello there!
```

- **Глобальна область видимості**: це змінні, оголошені поза межами будь-яких функцій або класів, до яких можна отримати доступ з будь-якої частини програми.

```py
tax = 0.70 

def get_total(subtotal):
    total = subtotal + (subtotal * tax)
    return total

print(get_total(100))  # 170.0
```

- **Вбудована область видимості**: зарезервовані імена в Python для заздалегідь визначених функцій, модулів, ключових слів і об’єктів.

```py
print(str(45)) # '45'
print(type(3.14)) # <class 'float'>
print(isinstance(3, str)) # False
```

```py
print(3 == 4) # False
```

- **Нерівність (`!=`)**: Перевіряє, чи два значення не рівні:

```py
print(3 != 4) # True
```

```py
print(3 > 4) # False
```

- **Строго менше ніж (`<`)**: Перевіряє, чи одне значення менше за інше:

```py
print(3 < 4) # True
```

- **Більше або дорівнює (`>=`)**: Перевіряє, чи одне значення більше або дорівнює іншому:

```py
print(3 >= 4) # False
```

- **Менше або дорівнює (`<=`)**: Перевіряє, чи одне значення менше або дорівнює іншому:

```py
print(3 <= 4) # True
```

## Робота з інструкціями `if`, `elif` та `else`

- **Інструкції `if`**: Це умови, які використовують для визначення, чи є щось істинним чи ні. Якщо умова оцінюється як `True`, тоді цей блок коду буде запущений.

```py
age = 18

if age >= 18:
    print('You are an adult') # You are an adult
```

- **Умова `elif`**: Це умови, які йдуть після інструкції `if`. Якщо умова `elif` оцінюється як `True`, тоді цей блок коду буде запущений.

```py
age = 16

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')  # You are a teenager
```

- **Умова `else`**: Цей блок буде запущений, якщо жодна інша умова не оцінюється як `True`.

```py
age = 12

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')
else:
    print('You are a child')  # You are a child
```

Ви також можете використовувати вкладені інструкції `if` ось так:

```py
is_citizen = True
age = 25

if is_citizen:
    if age >= 18:
        print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

## Істинні та хибні значення

- **Визначення**: У Python кожне значення має вбудоване булеве значення, або вбудоване відчуття того, чи слід його розглядати як `True` чи `False` у логічному контексті. Багато значень вважаються істинними, тобто вони оцінюються як `True` у логічному контексті. Інші є хибними, що означає, що вони оцінюються як `False`. Ось кілька прикладів хибних значень:

```md
None
False
Integer 0
Float 0.0
Empty strings ''
```

Інші значення, як-от ненульові числа та непорожні рядки, є істинними.

## Робота з функцією `bool()`

- **Визначення**: Якщо ви хочете перевірити, чи є значення правдивим або хибним, ви можете використати вбудовану функцію `bool()`. Вона явно перетворює значення на його булевий еквівалент і повертає `True` для правдивих значень і `False` для хибних значень. Ось кілька прикладів:

```py
print(bool(False)) # False
print(bool(0))  # False
print(bool('')) # False

print(bool(True)) # True
print(bool(1)) # True
print(bool('Hello')) # True
```

## Булеві оператори та коротке замикання

- **Визначення**: Це спеціальні оператори, які дозволяють поєднувати кілька виразів для створення більш складної логіки прийняття рішень у вашому коді. У Python є три булеві оператори: `and`, `or` та `not`.
- **Оператор `and`**: Цей оператор приймає два операнди і повертає перший операнд, якщо він є хибним значенням, інакше повертає другий операнд. Обидва операнди мають бути правдивими, щоб вираз дав правдиве значення.

```py
is_citizen = True
age = 25

print(is_citizen and age) # 25
```

Ви також можете використовувати оператор `and` у умовах так:

```py
is_citizen = True
age = 25

if is_citizen and age >= 18:
    print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

- **Оператор `or`**: Цей оператор повертає перший операнд, якщо він є правдивим значенням, інакше повертає другий операнд. Вираз `or` дає правдиве значення, якщо принаймні один операнд є правдивим. Ось приклад:

```py
age = 19
is_employed = False

print(age or is_employed) # 19
```

Так само, як і з оператором `and`, ви можете використовувати оператор `or` у умовах так:

```py
age = 19
is_student = True

if age < 18 or is_student:
    print('You are eligible for a student discount') # You are eligible for a student discount
else:
    print('You are not eligible for a student discount')
```

- **Коротке замикання**: Оператори `and` та `or` відомі як оператори короткого замикання. Коротке замикання означає, що Python перевіряє значення зліва направо і зупиняється, щойно визначає фінальний результат.
- **Оператор `not`**: Цей оператор приймає один операнд і інвертує його булеве значення. Він перетворює правдиві значення на `False` і хибні значення на `True`. На відміну від попередніх операторів, які ми розглядали, `not` завжди повертає `True` або `False`. Ось кілька прикладів:

```py
print(not '') # True, because empty string is falsy
print(not 'Hello') # False, because non-empty string is truthy
print(not 0) # True, because 0 is falsy
print(not 1) # False, because 1 is truthy
print(not False) # True, because False is falsy
print(not True) # False, because True is truthy
```

Ось приклад використання оператора `not` у умові:

```py
is_admin = False

if not is_admin:
    print('Access denied for non-administrators.') # Access denied for non-administrators.
else:
    print('Welcome, Administrator!')
```

## Списки в Python

- **Вступ**: У Python тип даних список — це впорядкована послідовність елементів, які можуть складатися з рядків, чисел або навіть інших списків. Списки змінні та мають індексацію, починаючи з нуля.

```python
cities = ['Los Angeles', 'London', 'Tokyo']
```

- **Доступ до елементів списку**: Щоб отримати доступ до елемента списку `cities`, ви можете звернутися до його номера індексу в послідовності:

```python
cities = ['Los Angeles', 'London', 'Tokyo']
cities[0] # Los Angeles
```

- **Доступ до елементів за допомогою від’ємної індексації**: Щоб отримати доступ до останнього елемента будь-якого списку, можна використати `-1` як номер індексу:

```python
cities = ['Los Angeles', 'London', 'Tokyo']
cities[-1] # Tokyo
```

- Від’ємна індексація використовується для доступу до елементів, починаючи з кінця списку, а не з початку за індексом `0`.

- **Створення списків за допомогою конструктора `list()`**: Списки також можна створювати за допомогою конструктора `list()`. Конструктор `list()` використовується для перетворення ітерованого об’єкта на список:

```python
developer = 'Jessica'

print(list(developer)) 
# Result: ['J', 'e', 's', 's', 'i', 'c', 'a']
```

- **Знаходження довжини списку**: Ви можете використати функцію `len()`, щоб отримати довжину списку:

```python
numbers = [1, 2, 3, 4, 5]
len(numbers) # 5
```

- **Змінність списків**: Списки змінні, тобто ви можете оновити будь-який елемент у списку, якщо передати дійсний номер індексу. Щоб оновити список за певним індексом, можна призначити нове значення цьому індексу:

```python
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[0] = 'JavaScript'
print(programming_languages) # ['JavaScript', 'Java', 'C++', 'Rust']
```

- **Помилка виходу за межі індексу**: Якщо ви передасте індекс (позитивний або від’ємний), який виходить за межі списку, ви отримаєте `IndexError`:

```python
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[10] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: list assignment index out of range
"""
```

- **Видалення елементів зі списку**: Елементи можна видалити зі списку за допомогою ключового слова `del`:

```python
developer = ['Jane Doe', 23, 'Python Developer']
del developer[1]
print(developer) # ['Jane Doe', 'Python Developer']
```

- **Перевірка наявності елемента у списку**: Ключове слово `in` можна використати, щоб перевірити, чи існує елемент у списку:

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

- **Вкладені списки**: Списки можуть бути вкладені всередину інших списків:

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
```

- Щоб отримати доступ до вкладеного списку, потрібно звернутися до нього за індексом `2`, оскільки списки мають індексацію, починаючи з нуля.

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2] # ['Python', 'Rust', 'C++']
```

- Щоб далі отримати доступ до другої мови з цього вкладеного списку, потрібно звернутися до неї за індексом `1`:

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2][1] # Rust
```

- **Розпакування значень зі списку:** Розпакування значень зі списку — це техніка, яка використовується для призначення значень зі списку новим змінним. Ось приклад розпакування списку `developer` у нові змінні `name`, `age` і `job` таким чином:

```py
developer = ['Alice', 34, 'Rust Developer']
name, age, job = developer
```

- Якщо кількість змінних зліва від оператора присвоєння не співпадає з загальною кількістю елементів у списку, ви отримаєте `ValueError`.

- **Збирання залишкових елементів зі списку**: Щоб зібрати будь-які залишкові елементи зі списку, можна використати оператор зірочки (`*`) таким чином:

```py
developer = ['Alice', 34, 'Rust Developer']
name, *rest = developer
```

- **Зрізи списків**: Зріз — це концепція доступу до частини списку за допомогою оператора зрізу `:`. Щоб зробити зріз списку, який починається з індексу `1` і закінчується перед індексом `3`, можна використати такий синтаксис:

```python
desserts = ['Cake', 'Cookies', 'Ice Cream', 'Pie']
desserts[1:3] # ['Cookies', 'Ice Cream']
```

- **Кроки інтервалу**: Також можна вказати крок інтервалу, який визначає, на скільки збільшувати індекси між кроками. Ось приклад, якщо ви хочете отримати список лише парних чисел за допомогою зрізу:

```python
numbers = [1, 2, 3, 4, 5, 6]
numbers[1::2] # [2, 4, 6]
```

## Методи списків

- **append()**: Використовується, щоб додати елемент у кінець списку. Ось приклад використання методу `append()` для додавання числа `6` до цього списку `numbers`:

```py
numbers = [1, 2, 3, 4, 5]
numbers.append(6)
print(numbers) # [1, 2, 3, 4, 5, 6]
```

- **Додавання списків**: Метод `append()` також можна використати, щоб додати один список у кінець іншого:

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.append(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, [6, 8, 10]]
```

- **extend()**: Використовується, щоб додати кілька елементів у кінець списку. Ось приклад додавання чисел `6`, `8` і `10` у кінець списку `numbers`:

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.extend(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, 6, 8, 10]
```

- **insert()**: Використовується, щоб вставити елемент у певний індекс списку. Ось приклад використання методу `insert()`:

```py
numbers = [1, 2, 3, 4, 5]
numbers.insert(2, 2.5)

print(numbers) # [1, 2, 2.5, 3, 4, 5]
```

- **remove():** Використовується, щоб видалити елемент зі списку. Метод `remove()` видалить лише перше входження елемента у списку:

```py
numbers = [1, 2, 3, 4, 5, 5, 5]
numbers.remove(5)

print(numbers) # [1, 2, 3, 4, 5, 5]

```

- **pop()**: Використовується, щоб видалити певний елемент зі списку та повернути його:

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop(1) # The number 2 is returned
```

- Якщо не вказати елемент для методу `pop`, буде видалено останній елемент.

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop() # The number 5 is returned
```

- **clear()**: Використовується, щоб видалити всі елементи зі списку:

```py
numbers = [1, 2, 3, 4, 5]
numbers.clear()

print(numbers) # []
```

- **sort()**: Метод `sort()` використовується для сортування елементів на місці. Ось приклад сортування випадкового списку `numbers` на місці:

```py
numbers = [19, 2, 35, 1, 67, 41]
numbers.sort()

print(numbers) # [1, 2, 19, 35, 41, 67]
```

- **sorted()**: Вбудована функція, яка повертає новий відсортований список замість зміни оригінального списку:

```py
numbers = [19, 2, 35, 1, 67, 41]
sorted_numbers = sorted(numbers)

print(sorted_numbers) # [1, 2, 19, 35, 41, 67]
print(numbers) # [19, 2, 35, 1, 67, 41]
```

- **reverse()**: Використовується, щоб змінити порядок елементів у списку на зворотний:

```py
numbers = [6, 5, 4, 3, 2, 1]
numbers.reverse()

print(numbers) # [1, 2, 3, 4, 5, 6]
```

- **index()**: Використовується, щоб знайти перший індекс, де можна знайти елемент у списку:

```python
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('Java') # 1
```

- Якщо елемент не можна знайти за допомогою методу `index()`, результатом буде `ValueError`.

## Кортежі в Python

- **Визначення**: Кортеж — це тип даних у Python, який використовується для створення впорядкованої послідовності значень. Кортежі можуть містити змішаний набір типів даних:

```py
developer = ('Alice', 34, 'Rust Developer')
```

- Кортежі є незмінними, тобто елементи в кортежі не можна змінювати після створення. Якщо ви спробуєте оновити один із елементів кортежу, ви отримаєте `TypeError`:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')
programming_languages[0] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: "tuple" object does not support item assignment
"""
```

- **Доступ до елементів кортежу**: Щоб отримати доступ до елемента кортежу, використовуйте дужкову нотацію та номер індексу:

```py
developer = ('Alice', 34, 'Rust Developer')
developer[1] # 34
```

- Від’ємна індексація може використовуватися для доступу до елементів, починаючи з кінця кортежу:

```py
numbers = (1, 2, 3, 4, 5)
numbers[-2] # 4
```

- Якщо ви спробуєте передати номер індексу, що перевищує або дорівнює довжині кортежу, ви отримаєте `IndexError`:

```py
numbers = (1, 2, 3, 4, 5)
numbers[7]

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: tuple index out of range
"""
```

- Кортеж також можна створити за допомогою конструктора `tuple()`. У конструктор можна передати різні ітеровані об’єкти, такі як рядки, списки та навіть інші кортежі.

```python
developer = 'Jessica'

print(tuple(developer)) 
# Result: ('J', 'e', 's', 's', 'i', 'c', 'a')
```

- **Перевірка наявності елементів у кортежі**: Щоб перевірити, чи є елемент у кортежі, можна використати ключове слово `in` ось так:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

- **Розпакування кортежів**: Елементи можна розпакувати з кортежу ось так:

```py
developer = ('Alice', 34, 'Rust Developer')
name, age, job = developer
```

- Якщо потрібно зібрати всі залишкові елементи з кортежу, можна використати оператор зірочки (`*`) ось так:

```py
developer = ('Alice', 34, 'Rust Developer')
name, *rest = developer
```

- **Зрізи кортежів**: Зрізи можна використовувати, щоб витягти частину кортежу. Наприклад, елементи `pie` і `cookies` можна вирізати в окремий кортеж:

```py
desserts = ('cake', 'pie', 'cookies', 'ice cream')
desserts[1:3] # ('pie', 'cookies')
```

- **Видалення елементів із кортежів**: Видалення елемента з кортежу викличе `TypeError`, оскільки кортежі незмінні:

```py
developer = ('Jane Doe', 23, 'Python Developer')
del developer[1]

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: "tuple" object doesn't support item deletion
"""
```

- **Коли використовувати кортеж замість списку?**: Якщо вам потрібна динамічна колекція елементів, у якій можна додавати, видаляти та оновлювати елементи, слід використовувати список. Якщо ви знаєте, що працюєте з фіксованою та незмінною колекцією даних, тоді слід використовувати кортеж.

## Загальні методи для кортежів

- **count()**: Використовується, щоб визначити, скільки разів елемент зустрічається в кортежі. Наприклад, ви можете перевірити, скільки разів мова `'Rust'` зустрічається в кортежі:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('Rust') # 2
```

- Якщо вказаний елемент у методі `count()` зовсім відсутній у кортежі, тоді повернене значення буде `0`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('JavaScript') # 0
```

- Якщо до методу `count()` не передати жодних аргументів, Python поверне `TypeError`.

- **index()**: Використовується, щоб знайти індекс, де певний елемент присутній у кортежі. Ось приклад використання методу `index()` для знаходження індексу мови `'Java'`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('Java') # 1
```

- Якщо вказаний елемент не може бути знайдений, Python поверне `ValueError`.

- Ви можете передати необов’язковий початковий індекс у метод `index()`, щоб вказати, з якого місця починати пошук елемента в кортежі:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
programming_languages.index('Python', 3) # 5
```

- Також можна передати необов’язковий кінцевий індекс у метод `index()`, щоб вказати, де припинити пошук елемента в кортежі:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python', 'JavaScript', 'Python')
programming_languages.index('Python', 2, 5) # 2
```

- **sorted()**: Використовується для сортування елементів будь-якого ітерованого об’єкта та повертає новий відсортований список. Ось приклад створення нового списку чисел за допомогою функції `sorted()`:

```py
numbers = (13, 2, 78, 3, 45, 67, 18, 7)
sorted(numbers) # [2, 3, 7, 13, 18, 45, 67, 78]
```

- **Зміна поведінки сортування**: Ви можете налаштувати поведінку сортування для ітерованого об’єкта, використовуючи необов’язкові аргументи `reverse` та `key`. Ось приклад використання аргументу `key` для сортування елементів кортежу за довжиною:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
sorted(programming_languages, key=len)

# Result
# ['C++', 'Rust', 'Java', 'Rust', 'Python', 'Python']
```

- Ви можете створити новий список значень у зворотному порядку, використовуючи аргумент `reverse` ось так:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')

print(sorted(programming_languages, reverse=True))

# Result
# ['Rust', 'Rust', 'Python', 'Python', 'Java', 'C++']
```

## Цикли в Python

- **Визначення**: Цикли використовуються для повторення блоку коду певну кількість разів.

- **Цикл `for`**: Використовується для ітерування над послідовністю (наприклад, списком, кортежем або рядком) і виконання блоку коду для кожного елемента цієї послідовності. Ось приклад використання циклу `for` для ітерування по списку та виведення кожної мови в консоль:

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']

for language in programming_languages:
    print(language)

"""
Result 

Rust
Java
Python
C++
"""
```

- Ось приклад використання циклу `for` для ітерування по рядку `code` та виведення кожного символу:

```py
for char in 'code':
    print(char)

"""
Result 

c
o
d
e
"""
```

- Цикли `for` можуть бути вкладеними. Ось приклад використання вкладеного циклу `for`:

```py
categories = ['Fruit', 'Vegetable']
foods = ['Apple', 'Carrot', 'Banana']

for category in categories:
    for food in foods:
        print(category, food)

"""
Result

Fruit Apple
Fruit Carrot
Fruit Banana
Vegetable Apple
Vegetable Carrot
Vegetable Banana
"""

```

- **Цикл `while`**: Повторює блок коду, доки умова не стане `False`. Ось приклад використання циклу `while` для гри в відгадування:

```py
secret_number = 3
guess = 0

while guess != secret_number:
    guess = int(input('Guess the number (1-5): '))
    if guess != secret_number:
        print('Wrong! Try again.')

print('You got it!')

"""
Result

Guess the number (1-5): 2
Wrong! Try again.
Guess the number (1-5): 1
Wrong! Try again.
Guess the number (1-5): 3
You got it!
"""

```

- **Інструкції `break` та `continue`**: Використовуються в циклах для зміни виконання циклу.

- Інструкція `break` використовується для негайного виходу з циклу, коли певна умова виконується. Ось приклад використання інструкції `break` для списку `developer_names`:

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        break
    print(developer)
```

- Інструкція `continue` використовується, щоб пропустити поточну ітерацію і перейти до наступної ітерації циклу. Ось приклад використання інструкції `continue` замість інструкції `break`:

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        continue
    print(developer)
```

- Цикли `for` та `while` можна поєднувати з умовою `else`, яка виконується лише тоді, коли цикл не був завершений інструкцією `break`:

```python
words = ['sky', 'apple', 'rhythm', 'fly', 'orange']

for word in words:
    for letter in word:
        if letter.lower() in 'aeiou':
            print(f"'{word}' contains the vowel '{letter}'")
            break
    else:
        print(f"'{word}' has no vowels")
```

## Діапазони та їх використання в циклах

- **Функція `range()`**: використовується для генерації послідовності цілих чисел.

```py
range(start, stop, step)
```

- Обов’язковий аргумент `stop` — це ціле число (не включно), яке представляє кінцеву точку для послідовності чисел, що генерується. Ось приклад використання функції `range()`:

```py
for num in range(3):
    print(num)
```

- Якщо аргумент `start` не вказано, тоді за замовчуванням буде `0`. За замовчуванням послідовність цілих чисел збільшується на `1`. Ви можете використати необов’язковий аргумент `step`, щоб змінити значення кроку за замовчуванням. Ось приклад генерації послідовності парних чисел від 2 до 11, не включно (тобто включає 10):

```py
for num in range(2, 11, 2):
    print(num)
```

- Якщо ви не надасте жодних аргументів функції `range()`, тоді отримаєте `TypeError`.

- Функція `range()` приймає лише цілі числа як аргументи, а не числа з плаваючою комою. Використання чисел з плаваючою комою також призведе до `TypeError`:

```py
ERROR!
Traceback (most recent call last):
  File "<main.py>", line 1, in <module>
TypeError: 'float' object cannot be interpreted as an integer
```

- Ви можете використати від’ємне ціле число для аргументу `step`, щоб згенерувати послідовність цілих чисел у спадному порядку:

```py
for num in range(40, 0, -10):
    print(num)
```

- Функцію `range()` також можна використати для створення списку цілих чисел, застосувавши її з конструктором `list`. Конструктор `list` використовується для перетворення ітерованого об’єкта на список. Ось приклад генерації списку парних чисел від 2 до 10 включно:

```py
numbers = list(range(2, 11, 2))
print(numbers) # [2, 4, 6, 8, 10]
```

## Функції `enumerate()` і `zip()` у Python

- **`enumerate()`**: використовується для ітерування над послідовністю з одночасним відстеженням індексу кожного елемента в цій послідовності. Функція `enumerate()` приймає ітерований об’єкт як аргумент і повертає об’єкт `enumerate`, який складається з індексу та значення кожного елемента в ітерованому об’єкті.

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for index, language in enumerate(languages):
    print(f'Index {index} and language {language}')

# Result
# Index 0 and language Spanish
# Index 1 and language English
# Index 2 and language Russian
# Index 3 and language Chinese

```

- Функцію `enumerate()` також можна використовувати поза межами циклу `for`:

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

print(list(enumerate(languages)))
# [(0, 'Spanish'), (1, 'English'), (2, 'Russian'), (3, 'Chinese')]
```

- Функція `enumerate()` також приймає необов’язковий аргумент `start`, який визначає початкове значення для лічильника. Якщо цей аргумент опущено, лічильник починається з `0`.

-  **`zip()`** : використовується для ітерування над кількома ітерованими об’єктами паралельно. Ось приклад використання функції `zip()` для ітерування над `developers` і `ids`:

```python
developers = ['Naomi', 'Dario', 'Jessica', 'Tom']
ids = [1, 2, 3, 4]

for name, id in zip(developers, ids):
    print(f'Name: {name}')
    print(f'ID: {id}')


"""
Result

Name: Naomi
ID: 1
Name: Dario
ID: 2
Name: Jessica
ID: 3
Name: Tom
ID: 4
"""

```

## Генератори списків у Python

- **Визначення**: Генератор списку дозволяє створити новий список в одному рядку, поєднуючи цикл і умову безпосередньо в квадратних дужках. Це робить код коротшим і часто легшим для читання.

```py
even_numbers = [num for num in range(21) if num % 2 == 0]
print(even_numbers)
```

## Функції для ітерованих об’єктів

- **`filter()`**: Використовується для фільтрації елементів з ітерованого об’єкта на основі умови. Повертає ітератор, який містить лише ті елементи, що задовольняють умову. Ось приклад створення нового списку лише зі словами довжиною більше чотирьох символів:

```py
words = ['tree', 'sky', 'mountain', 'river', 'cloud', 'sun']

def is_long_word(word):
    return len(word) > 4

long_words = list(filter(is_long_word, words))
print(long_words) # ['mountain', 'river', 'cloud']
```

- **`map()`**: Використовується для застосування функції до кожного елемента ітерованого об’єкта та повертає новий ітерований об’єкт з результатами. Ось приклад використання функції `map()` для перетворення списку температур у градусах Цельсія на Фаренгейти:

```py
celsius = [0, 10, 20, 30, 40]

def to_fahrenheit(temp):
    return (temp * 9/5) + 32

fahrenheit = list(map(to_fahrenheit, celsius))
print(fahrenheit) # [32.0, 50.0, 68.0, 86.0, 104.0]
```

- **`sum()`**: Використовується для отримання суми з ітерованого об’єкта, такого як список або кортеж. Ось приклад використання функції `sum()`:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers)
print(total) # Result: 50
```

- Ви також можете передати необов’язковий аргумент `start`, який встановлює ініціальне значення для суми. Ось оновлений приклад із використанням аргументу `start` як позиційного аргументу:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, 10) # positional argument
print(total) # 60
```

- Ви також можете вибрати використання аргументу `start` як ключового слова ось так:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, start=10) # keyword argument
print(total) # 60
```

## Лямбда-функції

- **Визначення**: Лямбда-функція в Python — це стислий спосіб створити функцію без імені (анонімну функцію).
- Лямбда-функції часто використовують як аргумент для іншої функції. Ось приклад лямбда-функції:

```py
numbers = [1, 2, 3, 4, 5]

even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(even_numbers)  # [2, 4]
```

- Найкращі практики використання лямбда-функцій включають уникання призначення їх змінній, збереження простоти та читабельності, а також використання для коротких, одноразових функцій.

## Словники

- **Словники**: Словники — це вбудовані структури даних, які зберігають колекції пар ключ-значення. Ключі мають бути незмінними типами даних. Ось загальний синтаксис словника в Python:

```python
dictionary = {
    key1: value1,
    key2: value2
}
```

- **Конструктор**: Конструктор — це альтернативний спосіб створення словника. Ви передаєте список кортежів як аргумент конструктору. Ці кортежі містять ключ як перший елемент і значення як другий елемент.

```python
pizza = dict([('name', 'Margherita Pizza'), ('price', 8.9), ('calories_per_slice', 250), ('toppings', ['mozzarella', 'basil'])])
```

- **Дужкова нотація**: Щоб отримати доступ до значення пари ключ-значення, можна використовувати синтаксис, відомий як дужкова нотація.

```python
dictionary[key]
```

## Загальні методи словника

- **`get()` метод**: Метод `get()` отримує значення, пов’язане з ключем. Це схоже на дужкову нотацію, але дозволяє встановити значення за замовчуванням, що запобігає помилкам, якщо ключ не існує.

```python
dictionary.get(key, default)
```

- **`keys()` та `values()` методи**: Методи `keys()` та `values()` повертають об’єкт перегляду з усіма ключами та значеннями словника відповідно. Об’єкт перегляду — це спосіб побачити вміст словника без створення окремої копії даних.

```python
pizza = {
    'name': 'Margherita Pizza',
    'price': 8.9,
    'calories_per_slice': 250
}

pizza.keys()
# dict_keys(['name', 'price', 'calories_per_slice'])

pizza.values()
# dict_values(['Margherita Pizza', 8.9, 250])
```

- **`items()` метод**: Метод `items()` повертає об’єкт перегляду з усіма парами ключ-значення у словнику, включно з ключами та значеннями.

```python
pizza.items()
# dict_items([('name', 'Margherita Pizza'), ('price', 8.9), ('calories_per_slice', 250)])
```

- **`clear()` метод**: Метод `clear()` видаляє всі пари ключ-значення зі словника.

```python
pizza.clear()
```

- **`pop()` метод**: Метод `pop()` видаляє пару ключ-значення з ключем, вказаним як перший аргумент, і повертає її значення. Якщо ключ не існує, він повертає значення за замовчуванням, вказане як другий аргумент. Якщо ключ не існує і значення за замовчуванням не вказано, виникає `KeyError`.

```python
pizza.pop('price', 10)
pizza.pop('total_price') # KeyError
```

- **`popitem()` метод**: У Python 3.7 і вище метод `popitem()` видаляє останній доданий елемент.

```python
pizza.popitem()
```

- **`update()` метод**: Метод `update()` оновлює пари ключ-значення парами іншого словника. Якщо ключі збігаються, їхні значення перезаписуються. Нові ключі будуть додані до словника як нові пари ключ-значення.

```python
pizza.update({ 'price': 15, 'total_time': 25 })
```

## Ітерування над словником

- **Ітерування над значеннями**: Якщо потрібно ітерувати над значеннями у словнику, ви можете написати цикл `for` з `values()`, щоб отримати всі значення словника.

```python
products = {
    'Laptop': 990,
    'Smartphone': 600,
    'Tablet': 250,
    'Headphones': 70,
}

for price in products.values():
    print(price)
```

Вихідні дані:

```md
990
600
250
70
```

- **Ітерування над ключами**: Якщо потрібно ітерувати над ключами словника `products` вище, ви можете написати `products.keys()` або `products` безпосередньо.

```python
for product in products.keys():
    print(product)
    
# Or

for product in products:
    print(product)
```

Вихідні дані:

```md
Laptop
Smartphone
Tablet
Headphones
```

- **Ітерування над парами ключ-значення**: Якщо потрібно ітерувати одночасно над ключами та їх відповідними значеннями, ви можете ітерувати над `products.items()`. Ви отримуєте окремі кортежі з ключами та їх відповідними значеннями.

```python
for product in products.items():
    print(product)
```

Вихідні дані:

```md
('Laptop', 990)
('Smartphone', 600)
('Tablet', 250)
('Headphones', 70)
```

Щоб зберегти ключ і значення в окремих змінних циклу, потрібно розділити їх комою. Перша змінна зберігає ключ, а друга — значення.

```python
for product, price in products.items():
    print(product, price)
```

Вихідні дані:

```md
Laptop 990
Smartphone 600
Tablet 250
Headphones 70
```

- **Функція `enumerate()`**: Якщо потрібно ітерувати над словником, відстежуючи лічильник, можна викликати функцію `enumerate()`. Функція повертає об’єкт `enumerate`, який призначає ціле число кожному елементу, як лічильник. Лічильник можна почати з будь-якого числа, але за замовчуванням він починається з 0.

Призначення індексу та елемента окремим змінним циклу — це загальний спосіб використання `enumerate()`. Наприклад, з `products.items()` ви можете отримати всю пару ключ-значення разом з індексом:

```python
for index, product in enumerate(products.items()):
    print(index, product)
```

Вихідні дані:

```md
0 ('Laptop', 990)
1 ('Smartphone', 600)
2 ('Tablet', 250)
3 ('Headphones', 70)
```

Щоб налаштувати початкове значення лічильника, можна передати другий аргумент у `enumerate()`. Наприклад, тут ми починаємо лічильник з 1.

```python
for index, product in enumerate(products.items(), 1):
    print(index, product)
```

Вихідні дані:

```md
1 ('Laptop', 990)
2 ('Smartphone', 600)
3 ('Tablet', 250)
4 ('Headphones', 70)
```

## Множини

- **Множини**: множини — це вбудовані структури даних у Python, які не допускають повторюваних значень. Множини змінні та неупорядковані, тобто їхні елементи не зберігаються у певному порядку, тому ви не можете використовувати індекси або ключі для доступу до них. Також множини можуть містити лише значення незмінних типів даних, таких як числа, рядки та кортежі.

- **Визначення множини**: щоб визначити множину, потрібно записати її елементи у фігурних дужках, розділяючи їх комами.

```python
my_set = {1, 2, 3, 4, 5}
```

- **Визначення порожньої множини**: якщо потрібно визначити порожню множину, слід використати функцію `set()`. Просто написати порожні фігурні дужки автоматично створить словник.

```python
set() # Set
{}    # Dictionary
```

## Загальні методи множини

- **Метод `add()`**: Ви можете додати елемент до множини за допомогою методу `add()`, передаючи новий елемент як аргумент.

```python
my_set.add(6)
```

- **Методи `remove()` та `discard()`**: Щоб видалити елемент із множини, ви можете використати або метод `remove()`, або метод `discard()`, передаючи елемент, який хочете видалити, як аргумент. Метод `remove()` викличе помилку `KeyError`, якщо елемент не знайдено, тоді як метод `discard()` цього не зробить.

```python
my_set.remove(4)
my_set.discard(4)
```

- **Метод `clear()`**: Метод `clear()` видаляє всі елементи з множини.

```python
my_set.clear()
```

## Математичні операції над множинами

- **Методи `issubset()` і `issuperset()`**: Методи `issubset()` і `issuperset()` перевіряють, чи є множина підмножиною або надмножиною іншої множини відповідно.

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 5}

print(your_set.issubset(my_set)) # True
print(my_set.issuperset(your_set)) # True
```

- **Метод `isdisjoint()`**: Метод `isdisjoint()` перевіряє, чи дві множини не мають спільних елементів (чи є вони диз’юнктними).

```python
my_set = {1, 2, 3}
your_set = {4, 5, 6}

print(my_set.isdisjoint(your_set)) # True
```

- **Оператор об’єднання (`|`)**: Оператор об’єднання `|` повертає нову множину з усіма елементами обох множин.

```python
my_set = {1, 2, 3}
your_set = {4, 5, 6}

my_set | your_set # {1, 2, 3, 4, 5, 6}
```

- **Оператор перетину (`&`)**: Оператор перетину `&` повертає нову множину лише з тими елементами, які є спільними для обох множин.

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

my_set & your_set # {2, 3, 4}
```

- **Оператор різниці (`-`)**: Оператор різниці `-` повертає нову множину з елементами першої множини, яких немає в другій.

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

my_set - your_set # {1, 5}
```

- **Оператор симетричної різниці (`^`)**: Оператор симетричної різниці `^` повертає нову множину з елементами, які є або в першій, або в другій множині, але не в обох одночасно.

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

my_set ^ your_set # {1, 5, 6}
```

- **Оператор `in`**: Ви можете перевірити, чи належить елемент множині, за допомогою оператора `in`.

```python
print(5 in my_set)
```

## Стандартна бібліотека Python

- **Стандартна бібліотека Python**: бібліотека надає вам заздалегідь написаний і багаторазово використовуваний код, такий як функції, класи та структури даних, які ви можете використовувати у своїх проєктах. У Python є обширна стандартна бібліотека з вбудованими модулями, які реалізують стандартизовані рішення для багатьох проблем і завдань. Деякі приклади популярних вбудованих модулів — це `math`, `random`, `re` (скорочення від "регулярні вирази") і `datetime`.

## Інструкція імпорту

- **Інструкція імпорту**: Щоб отримати доступ до елементів, визначених у вбудованих модулях, ви використовуєте інструкцію імпорту. Інструкції імпорту зазвичай пишуться на початку файлу. Інструкції імпорту працюють однаково для функцій, класів, констант, змінних і будь-яких інших елементів, визначених у модулі.

- **Базова інструкція імпорту**: Ви можете використати ключове слово `import`, за яким іде назва модуля:

```python
import module_name
```

Потім, якщо потрібно викликати функцію з цього модуля, ви використовуєте точкову нотацію: спочатку назва модуля, а потім назва функції.

```python
module_name.function_name()
```

Наприклад, у вашому коді ви можете написати так, щоб імпортувати модуль `math` і отримати квадратний корінь з 36:

```python
import math

math.sqrt(36)
```

- **Імпорт модуля з іншим іменем**: Якщо потрібно імпортувати модуль під іншим ім’ям (так званий "псевдонім"), ви можете використати `as`, а потім псевдонім у кінці інструкції імпорту. Це часто застосовується для довгих назв модулів або щоб уникнути конфліктів імен.

```python
import module_name as module_alias
```

Наприклад, щоб у коді посилатися на модуль `math` як `m`, можна призначити псевдонім так:

```python
import math as m
```

Потім ви можете отримувати доступ до елементів модуля, використовуючи цей псевдонім:

```python
m.sqrt(36)
```

- **Імпорт конкретних елементів**: Якщо вам не потрібні всі елементи модуля, ви можете імпортувати конкретні елементи за допомогою `from`. У цьому випадку інструкція імпорту починається з `from`, потім назва модуля, далі ключове слово `import` і нарешті імена елементів, які ви хочете імпортувати.

```python
from module_name import name1, name2
```

Потім ви можете використовувати ці імена без префікса модуля у вашому Python-скрипті. Наприклад:

```python
from math import radians, sin, cos

angle_degrees = 40
angle_radians = radians(angle_degrees)

sine_value = sin(angle_radians)
cos_value = cos(angle_radians)

print(sine_value) # 0.6427876096865393
print(cos_value)  # 0.766044443118978
```

Це зручно, але може призвести до конфліктів імен, якщо у вас уже є функції або змінні з такими ж іменами. Враховуйте це, обираючи тип інструкції імпорту.

Якщо потрібно призначити псевдоніми цим іменам, ви також можете це зробити, використовуючи ключове слово `as` і псевдонім.

```python
from module_name import name1 as alias1, name2 as alias2
```

- **Інструкція імпорту з астериском (`*`)**: Астериск повідомляє Python, що ви хочете імпортувати все з цього модуля, але так, щоб не потрібно було використовувати назву модуля як префікс.

```python
from module_name import *
```

Наприклад, якщо ви використаєте це для імпорту модуля `math`, ви зможете викликати будь-яку функцію, визначену в цьому модулі, без вказування назви модуля як префікса.

```python
from math import *
print(sqrt(36))  # 6.0
```

Однак це зазвичай не рекомендується, бо може призвести до конфліктів у просторі імен і ускладнити розуміння, звідки беруться імена.

## `if __name__ == '__main__'`

- **`__name__` змінна**: `__name__` — це спеціальна вбудована змінна в Python. Коли Python-файл запускається безпосередньо, Python встановлює значення цієї змінної як рядок `"__main__"`. Але якщо Python-файл імпортується як модуль в інший Python-скрипт, значення змінної `__name__` встановлюється як ім'я цього модуля.

Ось чому ви часто знайдете цю умову в Python-скриптах. Вона містить код, який ви хочете запускати **тільки** якщо Python-скрипт виконується як основна програма.

```python
if __name__ == '__main__': 
    # Code
```

## Загальні помилки в Python

- **SyntaxError**: помилка, яку Python видає, коли ваш код не відповідає його правилам синтаксису. Наприклад, код `print("Hello there"` призведе до синтаксичної помилки з повідомленням `SyntaxError: '(' was never closed`, тому що в коді відсутня закриваюча дужка.
- **NameError**: Python видає `NameError`, коли ви намагаєтеся звернутися до змінної або функції, яку не визначили. Наприклад, якщо у вашому коді є рядок `print(username)`, але змінна `username` не була визначена раніше, ви отримаєте помилку імені з повідомленням `NameError: name 'username' is not defined`.
- **TypeError**: ця помилка виникає, коли ви виконуєте операцію над двома або більше несумісними типами даних. Наприклад, якщо ви спробуєте додати рядок до числа, отримаєте помилку `TypeError: can only concatenate str (not "int") to str`.
- **IndexError**: ви отримаєте `IndexError`, якщо звернетеся до індексу, якого немає в списку або інших послідовностях, таких як кортеж чи рядок. Наприклад, у рядку `Hello world` індекс останнього символу — це `10`. Якщо ви спробуєте звернутися до символу таким чином, `greet = "hello world"; print(greet[11])`, отримаєте помилку з повідомленням `IndexError: string index out of range`.
- **AttributeError**: Python видає цю помилку, коли ви намагаєтеся використати метод або властивість, які не існують у об’єкта цього типу. Наприклад, виклик `.append()` для рядка `"hello".append("!")` призведе до помилки з повідомленням `AttributeError: 'str' object has no attribute 'append'`.

## Обробка виключень

- **`try...except`**: Використовується для виконання блоку коду, який може викликати виключення. Блок `try` — це місце, де ви очікуєте, що може виникнути помилка, а блок `except` обробляє вказане виключення, якщо воно виникає. Ось приклад:

  ```py
  try:
    print(22 / 0)
  except ZeroDivisionError:
    print('You can\'t divide by zero!')
    # You can't divide by zero!
  ```

  Ви також можете ланцюжити кілька блоків `except`, щоб обробляти різні типи виключень:

  ```py
  try:
    number = int(input('Enter a number: '))
    print(22 / number)
  except ZeroDivisionError:
    print('You cannot divide by zero!')
    # You cannot divide by zero! prints when you enter 0
  except ValueError:
    print('Please enter a valid number!')
    # Please enter a valid number! prints when you don't enter an integer
  ```

- **`else` та `finally`**: Ці блоки розширюють `try...except`. Якщо виключення не виникає, виконується блок `else`. Блок `finally` виконується завжди, незалежно від помилок.

  ```py
  try:
    result = 100 / 4
  except ZeroDivisionError:
    print('You cannot divide by zero!') # This will not run
  else:
    print(f'Result is {result}') # Result is 25.0
  finally:
    print('Execution complete!') # Execution complete!
  ```

- **Об’єкт виключення**: Дозволяє отримати доступ до самого виключення для кращого налагодження та виведення безпосереднього повідомлення про помилку. Щоб отримати об’єкт виключення, потрібно використати ключове слово `as`. Ось приклад:

  ```py
  try:
      value = int('This will raise an error')
  except ValueError as e:
      print(f'Caught an error: {e}')
      # Caught an error: invalid literal for int() with base 10: 'This will raise an error'
  ```

- **Інструкція `raise`**: Дозволяє вручну викликати виключення. Ви можете використати її, щоб кинути виключення, коли виконується певна умова. Ось приклад:

  ```py
  def divide(a, b):
      if b == 0:
          raise ZeroDivisionError('You cannot divide by zero')
      return a / b
  ```

## Сигналізація виключень
  
  Інструкція `raise` також корисна, коли ви створюєте власні кастомні виключення, оскільки з її допомогою можна викликати виключення з кастомним повідомленням. Ось приклад цього:

  ```py
  class InvalidCredentialsError(Exception):
      def __init__(self, message="Invalid username or password"):
          self.message = message
          super().__init__(self.message)

  def login(username, password):
      stored_username = "admin"
      stored_password = "password123"
      
      if username != stored_username or password != stored_password:
          raise InvalidCredentialsError()
      
      return f"Welcome, {username}!"
  ```  

  Ось як ви можете використати функцію `login` з виключенням `InvalidCredentialsError`:

  ```py
  # невдала спроба входу
  try:
      message = login("user", "wrongpassword")
  except InvalidCredentialsError as e:
      print(f"Login failed: {e}")
  else:
      print(message)

  # успішна спроба входу
  try:
      message = login("admin", "password123")
  except InvalidCredentialsError as e:
      # Цей блок не виконується, бо вхід був успішним
      print(f"Login failed: {e}")
  else:
      # Блок else виконується, якщо блок 'try' завершився без виключення
      print(message)
  ```

  Інструкцію `raise` також можна використовувати з ключовим словом `from` для ланцюжка виключень, що показує зв’язок між різними помилками:

  ```py
  def parse_config(filename):
    try:
        with open(filename, 'r') as file:
            data = file.read()
            return int(data)
    except FileNotFoundError:
        raise ValueError('Configuration file is missing') from None
    except ValueError as e:
        raise ValueError('Invalid configuration format') from e

  config = parse_config('config.txt')
  ```

## Класи та об’єкти в Python

- **Визначення класу**: Клас — це шаблон для створення об’єктів. Він визначає поведінку об’єкта через його атрибути та методи. Ось базовий приклад визначення класу в Python:

```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        print(f'{self.name.upper()} says woof woof!')
```

- **Створення об’єктів**: Об’єкти — це екземпляри класу. Їх створюють, викликаючи клас із необхідними аргументами.

```python
dog1 = Dog('Jack', 3)
dog2 = Dog('Thatcher', 5)

dog1.bark()  # JACK says woof woof!
dog2.bark()  # THATCHER says woof woof!
```

- **Виклик методів об’єктів**: Ви можете викликати методи об’єктів, щоб виконувати дії або отримувати інформацію.

```python
object_name1.method_name()
object_name2.method_name()
```

- **Різниця між класом і об’єктом**: Клас — це багаторазовий шаблон, а об’єкт — конкретний екземпляр цього класу з реальними даними.

## Атрибути

- **Атрибути екземпляра**: визначаються в `__init__()` за допомогою `self` і є унікальними для кожного об’єкта.
- **Атрибути класу**: визначаються безпосередньо всередині класу і спільні для всіх екземплярів.

```python
class Dog:
    species = 'French Bulldog'  # Class attribute

    def __init__(self, name):
        self.name = name  # Instance attribute

print(Dog.species) # French Bulldog

jack = Dog('Jack')
print(jack.name)     # Jack
print(jack.species)  # French Bulldog
```

## Методи

- **Методи**: Функції, визначені всередині класу, які працюють з атрибутами об’єкта.

```python
class Car:
    def __init__(self, color, model):
        self.color = color
        self.model = model

    def describe(self):
        return f'This car is a {self.color} {self.model}'

my_car_1 = Car('red', 'Tesla Model S')
print(my_car_1.describe())  # This car is a red Tesla Model S
```

- **Доступ до методів**: Викликайте методи об’єктів за допомогою точкової нотації. Ось приклад виклику методу `describe` для двох різних об’єктів car:

```python
class Car:
    def __init__(self, color, model):
        self.color = color  
        self.model = model  

    def describe(self):
        return f'This car is a {self.color} {self.model}'

my_car_1 = Car('red', 'Tesla Model S')
my_car_2 = Car('green', 'Lamborghini Revuelto')

print(my_car_1.describe()) # Calling method using the dot notation

print(my_car_2.describe()) # Calling method using the dot notation
```

## Методи dunder (магічні методи)

- **Визначення**: Спеціальні методи, які починаються і закінчуються подвійним підкресленням (наприклад, `__init__`, `__len__`, `__str__`, `__eq__`). Python використовує їх внутрішньо для вбудованих операцій.

```python
class Book:
    def __init__(self, title, pages):
        self.title = title
        self.pages = pages

    def __len__(self):
        return self.pages

    def __str__(self):
        return f"'{self.title}' has {self.pages} pages"

    def __eq__(self, other):
        return self.pages == other.pages

book1 = Book('Built Wealth Like a Boss', 420)
print(len(book1))        # 420
print(str(book1))        # 'Built Wealth Like a Boss' has 420 pages
```

- **Непрямий виклик методів dunder**: Вам не потрібно викликати методи dunder безпосередньо. Замість цього Python автоматично викликає їх, коли відбуваються певні дії. Ці операції включають:

  - **арифметичні дії, такі як додавання, віднімання, множення, ділення та інші**. `__add__()` викликається для додавання, `__sub__()` для віднімання, `__mul__()` для множення і `__truediv__()` для ділення.
        
  - **операції з рядками, такі як конкатенація, повторення, форматування та перетворення у текст**. `__add__()` викликається для конкатенації, `__mul__()` для повторення, `__format__()` для форматування, `__str__()` і `__repr__()` для перетворення у текст тощо.
        
  - **операції порівняння, такі як перевірка на рівність, менше за, більше за та інші**. `__eq__()` викликається для перевірки рівності, `__lt__()` для менше за, `__gt__()` для більше за тощо.
        
  - **операції ітерації, такі як зробити об’єкт ітерованим і переходити між елементами**. `__iter__()` викликається, щоб повернути ітератор, а `__next__()` — щоб отримати наступний елемент.

## Приклад із реального світу: кошик для покупок

- **Клас Cart із dunder-методами**: дозволяє додавати, видаляти, ітерувати та перевіряти вміст із вбудованою поведінкою.

```python
class Cart:
    def __init__(self):
        self.items = []

    def add(self, item):
        self.items.append(item)

    def remove(self, item):
        if item in self.items:
            self.items.remove(item)
        else:
            print(f'{item} is not in cart')

    def list_items(self):
        return self.items

    def __len__(self):
        return len(self.items)

    def __getitem__(self, index):
        return self.items[index]

    def __contains__(self, item):
        return item in self.items

    def __iter__(self):
        return iter(self.items)

cart = Cart()
cart.add('Laptop')
print(len(cart))        # 1
print('Laptop' in cart) # True
```

## Що таке об’єктно-орієнтоване програмування?

- **Об’єктно-орієнтоване програмування**: стиль програмування, у якому розробники розглядають усе у своєму коді як реальний об’єкт. Його часто називають OOP. Чотири ключові принципи, які допомагають ефективно організовувати та керувати кодом, — це **інкапсуляція**, **успадкування**, **поліморфізм** та **абстракція**.
- **Класи**: шаблон для створення об’єктів. Кожен об’єкт, створений із класу, має атрибути, які визначають дані, і методи, що визначають поведінку об’єкта.

## Що таке інкапсуляція?

- **Інкапсуляція**: Об’єднання атрибутів і методів об’єкта в єдину одиницю. Вона дозволяє приховати внутрішній стан об’єкта за простим набором публічних методів і атрибутів, які діють як двері. За цими дверима знаходяться приватні атрибути і методи, які контролюють, як змінюються дані і хто може їх бачити.
- **Приклад інкапсуляції**: Якщо ви хочете відстежувати баланс гаманця, ви дозволите внесення і зняття коштів, але не захочете, щоб хтось міг безпосередньо змінювати баланс гаманця:

```py
class Wallet:
   def __init__(self, balance):
       self.__balance = balance # Private attribute

   def deposit(self, amount):
       if amount > 0:
           self.__balance += amount # Add to the balance safely

   def withdraw(self, amount):
       if 0 < amount <= self.__balance:
           self.__balance -= amount # Remove from the balance safely

account = Wallet(500)
print(account.__balance) # AttributeError: 'Wallet' object has no attribute '__balance'
```

- **Різниця між префіксами атрибутів з одним і подвійним підкресленням**: Префікс з одним підкресленням означає, що атрибути і методи призначені для внутрішнього використання. Це конвенція, яка не забороняє доступ до атрибутів ззовні. Префікс з подвійним підкресленням ефективно забороняє доступ до них поза межами їхнього класу.

## Що таке геттери і сеттери?

- **Геттери і сеттери**: Методи, які дозволяють контролювати, як отримують доступ і змінюють атрибути класу. Ви отримуєте значення за допомогою геттерів і призначаєте значення за допомогою сеттерів.
- **Властивості**: Вони поєднують геттери і сеттери та дозволяють отримувати доступ до даних. Вони запускають додаткову логіку за лаштунками, коли ви отримуєте, призначаєте або видаляєте значення.
- **Чому властивості замість методів**: Властивості використовують замість методів для кращої читабельності та чистішого коду. Вони дозволяють отримувати доступ до значень через точкову нотацію, як до звичайних атрибутів, без круглих дужок.
- **Створення геттера**: Щоб створити геттер, використовуйте декоратор `@property`. Ось геттер, який отримує радіус кола:

```py
class Circle:
    def __init__(self, radius):
        self.radius = radius # Calling the setter

    @property
    def radius(self): # A getter to get the radius
        return self._radius
  
    @property
    def area(self):  # A getter to calculate area
        return 3.14 * (self._radius ** 2)

my_circle = Circle(3)

print(my_circle.radius) # 3
print(my_circle.area) # 28.26
```

Використання self.radius всередині `__init__()` гарантує, що сеттер виконається під час створення об’єкта, тому недійсні значення радіуса будуть одразу виявлені.

- **Створення сеттера**: Щоб створити сеттер, який призначатиме радіус, потрібно визначити інший метод з таким самим ім’ям і використати `@<property_name>.setter` над ним:

```py
class Circle:
    def __init__(self, radius):
        self.radius = radius # Calling the setter

    @property
    def radius(self):  # A getter to get the radius
        return self._radius

    @radius.setter
    def radius(self, value):  # A setter to set the radius
        if value <= 0:
            raise ValueError('Radius must be positive')
        self._radius = value

my_circle = Circle(3)
print('Initial radius:', my_circle.radius) # Initial radius: 3

my_circle.radius = 8
print('After modifying the radius:', my_circle.radius) # After modifying the radius: 8
```

- **Як Python обробляє геттери і сеттери**: Після визначення геттерів і сеттерів Python автоматично викликає їх за лаштунками щоразу, коли ви використовуєте звичайний синтаксис атрибутів таким чином:

```py
my_circle.radius # This will call the getter
my_circle.radius = 4 # This will call the setter
```

У сеттері не слід призначати значення властивості безпосередньо їй самій, бо це спричинить `RecursionError`.

- **Делетер**: Після призначення і отримання значення за допомогою сеттера і геттера ви можете контролювати, як його видалити, за допомогою `deleter`. Делетер запускає кастомну логіку, коли ви використовуєте інструкцію `del` для властивості. Щоб створити делетер, використовуйте декоратор `@<property_name>.deleter`.

```py
  # Deleter
    @radius.deleter
    def radius(self):
        print("Deleting radius...")
        del self._radius
```

## Що таке успадкування і як воно сприяє повторному використанню коду?

- **Успадкування**: процес, за якого дочірній клас використовує атрибути та методи батьківського класу. Успадкування сприяє повторному використанню коду, забезпечує чітку ієрархію та дозволяє кастомізувати поведінку без переписування всього заново. Щоб реалізувати успадкування, дочірній клас приймає ім’я батьківського класу:

```py
class Parent:
    # Parent attributes and methods

class Child(Parent):
    # Child inherits, extends, and/or overrides where necessary
```

- **Одинарне та множинне успадкування**: коли дочірній клас успадковує властивості та методи від одного батьківського класу, як показано вище, цей процес називається **одинарним успадкуванням**. Коли дочірній клас успадковує властивості та методи від кількох батьківських класів, це називається **множинним успадкуванням**. Ось синтаксис для цього:

```py
class Parent:
    # Attributes and methods for Parent

class Child:
    # Attributes and methods for Child

class GrandChild(Parent, Child):
    # GrandChild inherits from both Parent and Child
    # GrandChild can combine or override behavior from each
```

- **`super()` функція**: функція, яка дозволяє **викликати** метод батьківського класу, коли клас має іншу реалізацію цього методу або розширює його, без дублювання коду.

## Що таке поліморфізм і як він сприяє повторному використанню коду?

- **Поліморфізм**: принцип ООП, який дозволяє різним класам використовувати однакову назву методу, але кожен клас реалізує його по-своєму під час виклику. Ось синтаксис для цього:

```py
class A:
   def action(self): ...

class B:
   def action(self): ...

class C:
   def action(self): ...

Class().method()  # Works for A, B, or C
```

- **Поліморфізм на основі успадкування**: батьківський елемент задає метод, а кожен дочірній клас адаптує його під свої потреби.

## Що таке name mangling і як він працює?

- **Name mangling**: процес, під час якого Python внутрішньо перейменовує атрибут із подвійним префіксом підкреслення, додаючи підкреслення та ім’я класу як префікс, перетворюючи `__attribute` на `_ClassName__attribute`.
- **Мета name mangling**: основна мета name mangling — запобігти випадковому перевизначенню атрибутів і методів під час використання успадкування. Ось код, який допоможе це краще зрозуміти:

```py
class Parent:
    def __init__(self):
        self.__data = 'Parent data'

class Child(Parent):
    def __init__(self):
        super().__init__()
        self.__data = 'Child data'

c = Child()
print(c.__dict__) # {'_Parent__data': 'Parent data', '_Child__data': 'Child data'}
```

## Що таке абстракція і як вона допомагає тримати складні системи організованими?

- **Абстракція**: концепція програмування, у якій приховуються складні деталі реалізації об’єкта або системи, а показуються лише суттєві функції. У Python та інших мовах програмування абстракція спрощує складні системи, підвищуючи їхню повторну використаність.
- **Приклад абстракції**: гарним прикладом абстракції в повсякденному житті є автомобіль, який дозволяє користуватися лише кермом, педалями та важелем перемикання передач, не знаючи, як працює двигун або гальма.
- **Як Python реалізує абстракцію**: Python реалізує абстракцію через модуль `abc`. Цей модуль надає клас `ABC` (абстрактний базовий клас) і декоратор `@abstractmethod`. Абстрактний базовий клас (ABC) визначає загальні методи та властивості, які мають реалізувати підкласи. Його не можна інстанціювати.
- **Як визначається абстрактний метод**: абстрактний метод визначається за допомогою `@abstractmethod` і має бути перевизначений у підкласах, навіть якщо має реалізацію за замовчуванням. Основний синтаксис абстракції виглядає так:

```py
from abc import ABC, abstractmethod

# Define an abstract base class
class AbstractClass(ABC):
    @abstractmethod
    def abstract_method(self):
        pass

# Concrete subclass that implements the abstract method
class ConcreteClassOne(AbstractClass):
    def abstract_method(self):
        print('Implementation in ConcreteClassOne')

# Another concrete subclass
class ConcreteClassTwo(AbstractClass):
    def abstract_method(self):
        print('Implementation in ConcreteClassTwo')
```

## Алгоритми та нотація великого O

- **Алгоритми**: Набір однозначних інструкцій для розв’язання задачі або виконання завдання. Алгоритми мають завершуватися за скінченну кількість кроків, і кожен крок має бути точним і однозначним.

- **Нотація великого O**: Описує найгірший випадок продуктивності або швидкість зростання алгоритму при збільшенні розміру вхідних даних. Вона фокусується на тому, як зростає використання ресурсів із збільшенням вхідних даних, ігноруючи константні множники та члени нижчого порядку.

### Загальні часові складності

- **O(1) — константний час**: Алгоритм виконується за однаковий час незалежно від розміру вхідних даних.

```python
def check_even_or_odd(number):
    if number % 2 == 0:
        return 'Even'
    else:
        return 'Odd'
```

- **O(log n) — логарифмічний час**: Час зростає повільно зі збільшенням вхідних даних. Зустрічається в алгоритмах, які багаторазово зменшують розмір задачі на частку (наприклад, бінарний пошук).

- **O(n) — лінійний час**: Час виконання зростає пропорційно до розміру вхідних даних.

```python
for grade in grades:
    print(grade)
```

- **O(n log n) — лог-лінійний час**: Загальна складність ефективних алгоритмів сортування, таких як Merge Sort і Quick Sort.

- **O(n²) — квадратичний час**: Час виконання зростає квадратично. Часто зустрічається у вкладених циклах.

```python
for i in range(n):
    for j in range(n):
        print("Hello, World!")
```

### Просторова складність

- **O(1) — константний обсяг пам’яті**: Алгоритм використовує однакову кількість пам’яті незалежно від розміру вхідних даних.
- **O(n) — лінійний обсяг пам’яті**: Використання пам’яті зростає пропорційно до розміру вхідних даних.
- **O(n²) — квадратичний обсяг пам’яті**: Використання пам’яті зростає квадратично зі збільшенням розміру вхідних даних.

- **Розуміння проблеми**: Прочитайте умову задачі кілька разів. Визначте вхідні дані, очікуваний вихід і як перетворити вхідні дані у вихідні.

- **Псевдокод**: Опис логіки алгоритму на високому рівні, незалежний від мови програмування. Використовує загальновживану мову, змішану з програмними конструкціями, такими як `IF`, `ELSE`, `FOR`, `WHILE`.

```md
GET original_string
SET reversed_string = ""
FOR EACH character IN original_string:
  ADD character TO THE BEGINNING OF reversed_string
DISPLAY reversed_string
```

- **Крайні випадки**: Конкретні, дійсні вхідні дані, що трапляються на межах того, що алгоритм має обробляти. Завжди враховуйте та тестуйте крайні випадки.

## Масиви

- **Статичні масиви**: мають фіксований розмір, визначений під час ініціалізації. Елементи зберігаються у суміжних ділянках пам’яті. Розмір не можна змінити під час виконання програми.

- **Динамічні масиви**: можуть автоматично збільшуватися або зменшуватися під час виконання програми. Зміна розміру відбувається через автоматичне копіювання у більші масиви за потреби.

### Списки Python (динамічні масиви)

```python
numbers = [3, 4, 5, 6]

# Access elements
numbers[0]  # 3

# Update elements
numbers[2] = 16

# Add elements
numbers.append(7)
numbers.insert(3, 15)  # Insert at specific index

# Remove elements
numbers.pop(2)  # Remove at specific index
numbers.pop()   # Remove last element
```

### Часові складності для динамічних масивів

- **Доступ**: O(1)
- **Вставка в кінець**: O(1) у середньому, O(n) при необхідності зміни розміру
- **Вставка посередині**: O(n)
- **Видалення**: O(n) для середини, O(1) для кінця

## Стек

- **Стек**: структура даних типу «останній доданий — перший вилучений» (LIFO). Елементи додаються і вилучаються лише зверху.

- **Операція push**: додавання елемента на вершину стеку. Часова складність: O(1).

- **Операція pop**: вилучення елемента з вершини стеку. Часова складність: O(1).

```python
# Using Python list as stack
stack = []

# Push operations
stack.append(1)
stack.append(2)
stack.append(3)

# Pop operations
top_element = stack.pop()  # Returns 3
```

## Черги

- **Черги**: структура даних типу «перший прийшов — перший пішов» (FIFO). Елементи додаються в кінець і видаляються з початку.

- **Операція додавання в чергу (enqueue)**: додавання елемента в кінець черги. Часова складність: O(1).

- **Операція вилучення з черги (dequeue)**: видалення елемента з початку черги. Часова складність: O(1).

```python
from collections import deque

# Using deque for efficient queue operations
queue = deque()

# Enqueue operations
queue.append(1)
queue.append(2)
queue.append(3)

# Dequeue operations
first_element = queue.popleft()  # Returns 1
```

## Зв’язані списки

- **Зв’язані списки**: Лінійна структура даних, де кожен вузол містить дані та референс на наступний вузол. Вузли з’єднані як ланцюг.

### Однозв’язні списки

- **Структура**: Кожен вузол має дані та один референс на наступний вузол.
- **Перебір**: Можна рухатися лише вперед від голови до хвоста.
- **Головний вузол**: Перший вузол у списку, зазвичай єдиний вузол з прямим доступом.
- **Хвостовий вузол**: Останній вузол у списку, вказує на `None`.

### Операції та часові складності

- **Вставка на початок**: O(1)
- **Вставка в кінець**: O(n) — потрібно пройти до кінця
- **Вставка посередині**: O(n) — потрібно пройти до позиції
- **Видалення з початку**: O(1)
- **Видалення з кінця**: O(n) — потрібно пройти, щоб знайти попередній вузол
- **Видалення посередині**: O(n) — потрібно пройти, щоб знайти вузол

### Двозв’язні списки

- **Структура**: Кожен вузол має дані та два референси: на наступний і попередній вузол.
- **Перебір**: Можна рухатися в обох напрямках.
- **Пам’ять**: Потребує більше пам’яті, ніж однозв’язні списки через додатковий референс.

## Хеш-мапи та множини

### Мапи та хеш-мапи

- **Мапа (абстрактний тип даних)**: Керує колекціями пар ключ-значення. Кожен ключ має бути унікальним, але значення можуть повторюватися.

- **Хеш-мапа**: Конкретна реалізація мапи як АТД із використанням техніки хешування. Використовує хеш-функцію для генерації хеш-значень ключів, які визначають місце зберігання у базовому масиві.

### Словники Python (хеш-мапи)

```python
# Creating dictionaries
my_dictionary = {
    "A": 1,
    "B": 2, 
    "C": 3
}

# Alternative creation
my_dictionary = dict(A=1, B=2, C=3)

# Access and modify
value = my_dictionary["A"]  # 1
my_dictionary["A"] = 4      # Update value
del my_dictionary["A"]      # Remove key-value pair

# Check membership
"C" in my_dictionary

# Get keys, values, items
my_dictionary.keys()
my_dictionary.values()
my_dictionary.items()
```

### Часова складність для хеш-мап

- **Середній випадок**: O(1) для додавання, отримання, видалення
- **Найгірший випадок**: O(n), коли відбувається багато хеш-колізій

### Множини

- **Множини**: Неупорядковані колекції унікальних елементів. Дублікати не допускаються, певний порядок не підтримується.

- **Тільки незмінні елементи**: Множини можуть містити лише незмінні типи даних (числа, рядки, кортежі), оскільки хеш-значення мають залишатися сталими.

```python
# Creating sets
numbers = {1, 2, 3, 4}
empty_set = set()  # Must use set(), not {}

# Add and remove elements
numbers.add(5)
numbers.remove(4)      # Raises KeyError if not found
numbers.discard(4)     # No error if not found

# Set operations
set_a = {1, 2, 3, 4}
set_b = {2, 3, 4, 5, 6}

# Union, intersection, difference, symmetric difference
set_a.union(set_b)                    # or set_a | set_b
set_a.intersection(set_b)             # or set_a & set_b
set_a.difference(set_b)               # or set_a - set_b
set_a.symmetric_difference(set_b)     # or set_a ^ set_b

# Subset and superset checks
set_a.issubset(set_b)
set_a.issuperset(set_b)
set_a.isdisjoint(set_b)

# Membership testing
5 in numbers
```

### Часова складність для множин

- **Середній випадок**: O(1) для додавання, видалення, перевірки належності
- **Найгірший випадок**: O(n) через хеш-колізії

## Колізії хешування

- **Колізія хешування**: Виникає, коли два різні ключі дають однакове хеш-значення.

- **Стратегії розв’язання колізій**:
  - **Зв’язування**: Кожен індекс масиву вказує на зв’язаний список, який зберігає всі елементи з однаковим хеш-значенням
  - **Відкрите адресування**: Пошук наступного доступного індексу за допомогою заздалегідь визначеної послідовності

## Коли використовувати кожну структуру даних

- **Списки**: коли потрібен впорядкований, індексований доступ і розмір невідомий заздалегідь  
- **Стек**: для операцій LIFO (функціонал скасування, обчислення виразів, повернення назад)  
- **Черги**: для операцій FIFO (планування завдань, пошук у ширину)  
- **Зв’язані списки**: коли часто потрібно вставляти або видаляти на початку, розмір невідомий, випадковий доступ не потрібен  
- **Хеш-мапи**: для швидкого пошуку за ключем, підрахунку кількості, кешування  
- **Множини**: для перевірки унікальності, математичних операцій над множинами, видалення дублікатів

## Алгоритми пошуку

Алгоритми пошуку дозволяють шукати ціль у певному списку елементів.

У комп’ютерних науках існують два алгоритми пошуку, які варто знати. Це **лінійний пошук** та **бінарний пошук**. Важливо розуміти відмінності між цими двома алгоритмами та коли слід використовувати кожен із них.

### Лінійний пошук

- Лінійний пошук ітерує список елементів, перевіряючи кожен елемент з початку, доки не знайде цільовий елемент.
- Якщо цільовий елемент знайдено, повертається індекс, де він розташований у списку.
- Якщо ціль не знайдена, повертається `-1`, що означає **недійсний індекс** у більшості мов програмування.
- Оскільки лінійний пошук перевіряє кожен елемент, доки не знайде ціль, він не є ефективним для великого списку елементів.
- Часова складність лінійного пошуку — `O(n)`, бо час пошуку зростає лінійно зі збільшенням розміру списку.
- Просторова складність лінійного пошуку — `O(1)`, бо він не потребує додаткового простору для пошуку у списку.

### Бінарний пошук

- Бінарний пошук працює, ділячи список елементів навпіл і перевіряючи, чи знаходиться цільове значення посередині списку.
- Умова для роботи бінарного пошуку — елементи списку мають бути впорядковані за зростанням.
- Бінарний пошук є більш ефективним алгоритмом для пошуку у великому списку, бо він ділить список навпіл і ігнорує ту половину, де ціль не знайдена.
- Якщо цільовий елемент знайдено посередині списку, повертається індекс цільового елемента.
- Якщо елемент не знайдено, алгоритм перевіряє, чи знаходиться ціль у лівій чи правій половині списку.
- Він продовжує ділити залишкові частини списку навпіл, доки цільовий елемент не буде знайдений.
- Якщо цільовий елемент так і не знайдено у списку, повертається `-1`.
- Часова складність бінарного пошуку — `O(log n)`, бо час пошуку зростає логарифмічно зі збільшенням розміру списку.
- Просторова складність бінарного пошуку — `O(1)`, бо він не потребує додаткового простору для пошуку у списку.

### Відмінності лінійного пошуку від бінарного

- Бінарний пошук краще підходить для великого списку елементів у порівнянні з лінійним пошуком.
- Часова складність лінійного пошуку — `O(n)`, бо час пошуку зростає лінійно зі збільшенням розміру списку.
- Часова складність бінарного пошуку — `O(log n)`, бо час пошуку зростає логарифмічно зі збільшенням розміру списку.

## Алгоритми сортування та техніка розділяй і володарюй

У комп’ютерних науках техніка розділяй і володарюй використовується для розбиття задачі на менші підзадачі, щоб їх було легше розв’язати. Рекурсія — це техніка, яку часто застосовують у розділяй і володарюй, і ця стратегія є потужним інструментом для реалізації багатьох ефективних алгоритмів сортування, таких як сортування злиттям.

### Сортування злиттям

- Сортування злиттям — це алгоритм сортування, який базується на підході розділяй і володарюй.
- Він працює шляхом рекурсивного поділу списку на менші підсписки, доки кожен підсписок не містить лише один елемент.
- Потім він багаторазово об’єднує підсписки назад у відсортованому порядку.
- Часова складність сортування злиттям — `O(n log n)`, оскільки список безперервно ділиться навпіл `(log n)`, а потім об’єднується `(O(n))`.
- Просторова складність сортування злиттям — `O(n)`, бо це не алгоритм сортування на місці.

## Огляд графів

Граф — це множина вузлів (вершин), з’єднаних ребрами (зв’язками). Кожен вузол може бути з’єднаний з кількома іншими вузлами, утворюючи мережу. Різновиди графів включають:

- Орієнтований: ребра мають напрямок (від одного вузла до іншого), часто представлені прямими лініями зі стрілками.
- Неорієнтований: ребра не мають напрямку, представлені простими лініями.
- Вершинний: кожен вузол пов’язаний з міткою або ідентифікатором.
- Циклічний: містить цикли (шлях, що починається і закінчується в одному й тому самому вузлі).
- Ациклічний (DAG): не містить циклів.
- З мітками ребер: кожне ребро має мітку, зазвичай розташовану поруч із відповідним ребром.
- Зважений: ребра мають ваги (значення), які можна використовувати для виконання арифметичних дій.
- Роз’єднаний: містить два або більше вузлів, які не з’єднані жодними ребрами.

Графи використовують у різних застосунках, таких як карти, мережі, системи рекомендацій, розв’язання залежностей.

## Обхід графа

Це передбачає відвідування всіх вузлів у графі. Два основні алгоритми:

- **Пошук у ширину (BFS)**
  - Використовує чергу.
  - Досліджує рівень за рівнем.
  - Знаходить найкоротший шлях у невагомих графах.

- **Пошук у глибину (DFS)**
  - Використовує стек (або рекурсію).
  - Повністю досліджує гілку перед поверненням назад.
  - Корисний для виявлення циклів і пошуку шляху.

## Представлення графів

Графи можна представити двома основними способами:

- **Список суміжності**
  - Кожен вузол має список своїх сусідів.
  - Економить простір для розріджених графів.
  - Легко ітерувати над сусідами.

- **Матриця суміжності**
  - Двовимірний масив, де рядки та стовпці представляють вузли.
  - Вимагає багато місця для великих графів.
  - Швидко перевірити, чи існує ребро між двома вузлами.

## Дерева

Дерево — це особливий тип графа, який є ациклічним і зв’язним. Основні властивості включають:

- Вони не мають циклів або кіл (шляхів, де початковий і кінцевий вузли однакові).
- Вони мають бути зв’язними (кожен вузол можна досягти з будь-якого іншого вузла).

### Загальні типи дерев

Найпоширеніші типи дерев:

- Двійкові дерева
  - Кожен вузол має не більше двох дочірніх елементів: лівий і правий.

- Двійкові дерева пошуку (BST)
  - Двійкове дерево, у якому кожен лівий дочірній елемент менший за свого батьківського елемента, а кожен правий дочірній елемент більший за свого батьківського елемента.

## Префіксні дерева

Також відомі як префіксні дерева, вони використовуються для зберігання множин рядків, де кожен вузол представляє символ.

Спільні префікси зберігаються лише один раз, що робить їх ефективними для завдань, таких як автозаповнення та перевірка орфографії.

Операції пошуку та вставки мають часову складність O(L), де L — довжина рядка.

## Пріоритетні черги

Пріоритетна черга — це абстрактний тип даних, у якому кожен елемент має свій пріоритет.

Черги та стеки враховують лише порядок вставки, тоді як пріоритетні черги враховують пріоритет елементів.

Стандартні черги працюють за принципом FIFO (перший прийшов — перший вийшов), а стеки — за принципом LIFO (останній прийшов — перший вийшов). Однак у пріоритетній черзі елементи з вищим пріоритетом обслуговуються раніше за ті, що мають нижчий пріоритет, незалежно від порядку їх вставки.

## Купи

Це спеціалізована дерево-подібна структура даних із дуже специфічною властивістю, яка називається властивістю купи.

Властивість купи визначає відношення між батьківськими та дочірніми вузлами. Існує два типи куп:

- Макс-купа
  - Значення кожного батьківського вузла більше або дорівнює значенням його дочірніх вузлів.
  - Найбільший елемент знаходиться в корені.

- Мін-купа
  - Значення кожного батьківського вузла менше або дорівнює значенням його дочірніх вузлів.
  - Найменший елемент знаходиться в корені.

### Приклад модуля Python `heapq`

```py
import heapq

# Create empty heap
my_heap = []

# Insert elements
heapq.heappush(my_heap, 9)
heapq.heappush(my_heap, 3)
heapq.heappush(my_heap, 5)
print(my_heap) # [3, 9, 5]

# Remove smallest element
print(heapq.heappop(my_heap))  # 3
print(my_heap) # [5, 9]

# Push + Pop in one step
print(heapq.heappushpop(my_heap, 7)) # 5
print(my_heap) # [7, 9]

# Transform list into heap
nums = [5, 7, 3, 1]
heapq.heapify(nums)
```

### Використання пріоритетів

```py
my_heap = []
heapq.heappush(my_heap, (3, "A"))
heapq.heappush(my_heap, (2, "B"))
heapq.heappush(my_heap, (1, "C"))

# Removes lowest number = highest priority
print(heapq.heappop(my_heap))  # (1, "C")
```

## Вступ до динамічного програмування

- **Визначення**: Динамічне програмування — це алгоритмічна техніка, яка розв’язує складні задачі, розбиваючи їх на простіші підзадачі та зберігаючи результати, щоб уникнути повторних обчислень.  
- **Перекривні підзадачі**: Одні й ті самі менші задачі з’являються кілька разів під час розв’язання більшої задачі. Замість того, щоб повторно обчислювати ці підзадачі, ми зберігаємо їхні розв’язки.  
- **Оптимальна підструктура**: Оптимальний розв’язок задачі містить оптимальні розв’язки її підзадач. Це означає, що ми можемо побудувати найкращий розв’язок, поєднуючи найкращі розв’язки менших частин.

## Рішення за допомогою динамічного програмування

- **Мемоізація (підхід зверху вниз)**: Мемоізація зберігає результати дорогих викликів функції і повертає кешований результат, коли ті самі вхідні дані з’являються знову.

```py
def climb_stairs_memo(n, memo={}):
    """Dynamic programming with memoization"""
    # Check if we've already calculated this value
    if n in memo:
        return memo[n]  # Return cached result - O(1) lookup!
    
    # Base cases
    if n <= 2:
        return n
    
    # Calculate once and store in memo for future use
    memo[n] = climb_stairs_memo(n-1, memo) + climb_stairs_memo(n-2, memo)
    return memo[n]
```

- **Табуляція (підхід знизу вгору)**: Табуляція будує розв’язок знизу вгору, заповнюючи таблицю розв’язками підзадач.

```py
def climb_stairs_tabulation(n):
    """Dynamic programming with tabulation"""
    if n <= 2:
        return n
    
    # Create array to store results for all steps from 0 to n
    dp = [0] * (n + 1)
    dp[1] = 1  # 1 way to reach step 1
    dp[2] = 2  # 2 ways to reach step 2
    
    # Build up the solution iteratively
    for i in range(3, n + 1):
        # Ways to reach step i = ways to reach (i-1) + ways to reach (i-2)
        dp[i] = dp[i-1] + dp[i-2]
    
    return dp[n]
```

## Реальні застосунки динамічного програмування

- **Оптимізація маршрутів**: GPS-системи використовують алгоритми динамічного програмування, щоб знайти найкоротші шляхи між локаціями.
