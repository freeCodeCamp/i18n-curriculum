# --explanation--

`Hands on deck` significa che servono più persone per aiutare con un compito o una situazione. Deriva dalla navigazione, dove tutti i membri dell'equipaggio dovevano essere sul ponte della nave per gestire una situazione urgente. Per esempio:

`We have a tight deadline, so we need all hands on deck.` - Questo significa che tutti devono aiutare per completare il lavoro in tempo.
