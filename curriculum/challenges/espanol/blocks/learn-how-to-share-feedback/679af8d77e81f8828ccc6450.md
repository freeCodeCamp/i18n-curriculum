# --explanation--

`Perhaps` se usa para indicar incertidumbre o para sugerir algo sin ser definitivo. Por ejemplo:

`Perhaps we should wait before making a decision.` - Esto significa que el hablante no está completamente seguro pero piensa que esperar podría ser una buena idea.

`To make sure` significa asegurarse de que algo ocurra o se haga correctamente. Por ejemplo:

Por favor, asegúrate de que los documentos se envíen antes de la fecha límite.
