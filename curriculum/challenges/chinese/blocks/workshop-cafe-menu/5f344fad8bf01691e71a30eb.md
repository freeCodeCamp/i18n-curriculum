# --description--

到目前为止，你对内容的展示和外观的控制有限。 为了改变这种情况，在 `head` 元素中创建一个 `style` 元素。

# --hints--

你应该有一个 `<style>` 开始标签。

```js
assert.match(code, /<style\s*>/i);
```

你应该有一个 `</style>` 结束标签。

```js
assert.match(code, /<\/style\s*>/);
```

你的 `style` 元素都应嵌套在 `head` 元素中。

```js
assert.match(code, /<head\s*>[\w\W\s]*<style\s*>[\w\W\s]*<\/style\s*>[\w\W\s]*<\/head\s*>/i)
```
