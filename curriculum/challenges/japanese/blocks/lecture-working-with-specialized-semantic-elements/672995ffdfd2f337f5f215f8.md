# --interactive--

superscript要素は、テキストの一部を上付き文字として表示するために使います。上付き文字とは、通常の文字の行よりも上に印刷される記号や文字のことです。

以下は、指数を示すためにsuperscript要素を使った例です（プレビューを見るにはインタラクティブエディターを有効にしてください）：

:::interactive_editor

```html
<p>2<sup>2</sup> (2 squared) is 4.</p>
```

:::

この例では、数字の2が`sup`タグで囲まれており、段落内で上付き文字を表しています。プレビュー画面では、2つ目の数字の2が最初の数字の2よりも小さく、少し上に表示されているのがわかります。

superscript要素の一般的な使用例には、指数、上付き文字、序数があります。以下は、上付き文字を使った例です（プレビューを見るにはインタラクティブエディターを有効にしてください）：

:::interactive_editor

```html
<p>
  Monseigneur is often written as <strong>M<sup>gr</sup></strong>.
</p>
```

:::

上付き文字とは、通常は略語を示すために上付きで書かれる文字のことです。この例では、`g`と`r`の文字が上付きタグで囲まれて略語を示しています。

重要なのは、superscript要素はあくまで組版上の理由で使うべきだということです。もしテキストを視覚的に上げたいだけなら、superscript要素ではなくCSSを使うべきです。

HTML内で化学式を表すには、subscript要素を使います。この要素は下付き文字を使い、文字の基線を下げて小さな文字で表示します。

以下は、二酸化炭素の化学式を示すためにsubscript要素を使った例です（プレビューを見るにはインタラクティブエディターを有効にしてください）：

:::interactive_editor

```html
<p>CO<sub>2</sub></p>
```

:::

数字の2が`sub`タグで囲まれており、その文字が下付き文字であることを示しています。

subscript要素の一般的な使用例には、化学式、脚注、変数の添え字があります。
