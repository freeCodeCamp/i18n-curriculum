# --description--

React ni maktaba ya mtazamo ya chanzo huru iliyoundwa na kudumishwa na Facebook. Ni chombo kizuri kuonyesha Kiolesura cha Mtumiaji (UI) cha programu za mtandao za kisasa.

React hutumia nyongeza ya sintaksia ya JavaScript inayoitwa JSX inayokuwezesha kuandika HTML moja kwa moja ndani ya JavaScript. Hii ina faida kadhaa. Inakuwezesha kutumia nguvu kamili ya programu ya JavaScript ndani ya HTML, na husaidia kuweka msimbo wako usomeka. Kwa sehemu kubwa, JSX ni sawa na HTML ambayo tayari umejifunza, hata hivyo kuna tofauti chache muhimu zitakazojadiliwa katika changamoto hizi.

Kwa mfano, kwa sababu JSX ni nyongeza ya sintaksia ya JavaScript, unaweza kuandika JavaScript moja kwa moja ndani ya JSX. Ili kufanya hivyo, unajumuisha msimbo unaotaka uchukuliwe kama JavaScript ndani ya mabano ya mviringo: `{ 'this is treated as JavaScript code' }`. Kumbuka hili, kwani linatumika katika changamoto kadhaa zijazo.

Hata hivyo, kwa sababu JSX si JavaScript halali, msimbo wa JSX lazima ukusanywe kuwa JavaScript. Transpiler Babel ni chombo maarufu kwa mchakato huu. Kwa urahisi wako, tayari imeongezwa nyuma ya pazia kwa changamoto hizi. Ikiwa utatengeneza JSX isiyo sahihi kisintaksia, utaona jaribio la kwanza katika changamoto hizi lishindikane.

Inafaa kutambua kuwa chini ya pazia changamoto hizi zinawito `ReactDOM.render(JSX, document.getElementById('root'))`. Wito huu wa kitendakazi ndio unaoweka JSX yako katika uwakilishi mwepesi wa React wa DOM. React hutumia picha za snapshot za DOM yake kuboresha sasisho kwa sehemu maalum tu za DOM halisi.

# --instructions--

Msimbo wa sasa unatumia JSX kuweka kipengele cha `div` kwa thabiti `JSX`. Badilisha `div` na kipengele cha `h1` na ongeza maandishi `Hello JSX!` ndani yake.
