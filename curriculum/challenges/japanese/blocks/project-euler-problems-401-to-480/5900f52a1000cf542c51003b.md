# --description--

$p$ 人のグループが円卓に座り、宝くじチケットの交換ゲームをすることにしました。各人はランダムに代入された、未削りの宝くじチケットを持ってゲームを始めます。各チケットは削ると、£1 から £$p$ までの整数ポンドの賞金が明らかになり、同じ賞金のチケットはありません。ゲームの目的は、各人がゲームを終えるときに自分のチケットの賞金を最大化することです。

任意の人が最初のプレイヤーに選ばれます。テーブルを回りながら、各プレイヤーは次の2つのうちどちらかの選択肢しかありません。

1. プレイヤーは自分のチケットを削り、その価値をテーブルの全員に明らかにできます。
2. プレイヤーは未削りの自分のチケットを、以前のプレイヤーの削られたチケットと交換し、そのチケットを持ってゲームから退出します。以前のプレイヤーは新しく得たチケットを削り、その価値をテーブルの全員に明らかにします。

すべてのチケットが削られた時点でゲームは終了します。テーブルに残っているすべてのプレイヤーは、現在持っているチケットを持って退出しなければなりません。

各プレイヤーは自分のチケットの期待値を最大化する最適戦略を使うと仮定します。

$p$ 人のゲームでゲーム終了時にテーブルに残っているプレイヤーの期待人数を $E(p)$ と表します（例：$E(111) = 5.2912$ は有効数字5桁で四捨五入した値です）。

$S_1(N) = \displaystyle\sum_{p = 1}^N E(p)$ とします。

$k > 1$ の場合、$S_k(N) = \displaystyle\sum_{p = 1}^N S_{k - 1}(p)$ とします。

$S_{20}({10}^{14})$ を求め、答えを有効数字10桁で四捨五入した科学的記数法の文字列で書いてください。小文字の `e` を仮数部と指数部の区切りに使います。例えば $S_3(100)$ の答えは `5.983679014e5` となります。
