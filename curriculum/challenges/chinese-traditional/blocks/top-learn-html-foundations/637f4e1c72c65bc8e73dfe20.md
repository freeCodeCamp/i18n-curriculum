# --description--

要示範 HTML 樣板，你首先需要一個 HTML 檔案來使用。

在你的電腦上創建一個新資料夾並命名為 `html-boilerplate`。在該資料夾內創建一個新檔案並命名為 `index.html`。

你可能已經熟悉許多不同型別的檔案，例如 doc、pdf 和影像檔案。

要讓電腦知道你想要創建一個 HTML 檔案，你需要像創建 `index.html` 檔案時一樣，將檔名附加 `.html` 擴充套件。

值得注意的是，你將你的 HTML 檔案命名為 index。你應該始終將包含你網站首頁的 HTML 檔案命名為 `index.html`。這是因為網頁伺服端預設會在使用者進入你網站時尋找 `index.html` 頁面－如果沒有這個檔案，將會造成重大問題。

## DOCTYPE

每個 HTML 頁面都以 doctype 宣告開始。doctype 的目的是告訴瀏覽器應該使用哪個版本的 HTML 來呈現文件。最新版本的 HTML 是 HTML5，而該版本的 doctype 就是 `<!DOCTYPE html>`。

舊版本 HTML 的 doctype 較為複雜。 例如，這是 HTML4 的 doctype 宣告：

```html
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
```

不過，你大概永遠不會想使用舊版的 HTML，因此你會一直使用 `<!DOCTYPE html>`。

在你先前創建的 `index.html` 檔案中使用文字編輯器開啟，並在第一行加入 `<!DOCTYPE html>`。
