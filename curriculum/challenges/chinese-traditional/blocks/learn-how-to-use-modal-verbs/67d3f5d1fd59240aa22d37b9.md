## --text--

哪一句語法正確？

## --answers--

`You must to complete the security training.`

### --feedback--

`Must` 是一個情態動詞，且在基本動詞前不使用 `to`。

---

`You must completed the security training.`

### --feedback--

`must` 後面的動詞應該使用原形（`complete`），而不是過去式（`completed`）。

---

`You must complete the security training.`

---

`You must completing the security training.`

### --feedback--

`Must` 後面接原形動詞，而非 `-ing` 形式。

# --explanation--

`Must` 是一個情態動詞，因此它遵循以下規則：

- 它後面總是接動詞的原形（不帶 `to`）。

- 第三人稱單數不加 `-s`。

- 在疑問句或否定句中不使用 `do/does`。
