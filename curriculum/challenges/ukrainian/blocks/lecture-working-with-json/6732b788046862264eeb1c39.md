# --description--

Що таке JSON і як отримати доступ до значень за допомогою дужкової та точкової нотації?

JSON означає JavaScript Object Notation. Це легкий текстовий формат даних, який часто використовується для обміну даними між сервером і веб-застосунком.

Одна з причин популярності JSON у веб-розробці полягає в тому, що він одночасно машинно-зрозумілий і зручний для читання людиною.

Оскільки JSON не залежить від мови програмування, ви можете легко передавати JSON-дані з Java-застосунку до Python-застосунку або з JavaScript-застосунку до C#-застосунку.

JSON підтримує багато типів даних, включно з об’єктами, масивами, рядками, булевими значеннями, null і числами.

Ось приклад JSON-об’єкта:

```js
{
  "name": "Alice",
  "age": 30,
  "isStudent": false,
  "list of courses": ["Mathematics", "Physics", "Computer Science"]
}
```

Як бачите, JSON використовує пари ключ-значення для зберігання інформації, і кожна пара розділена комою. Кожен ключ повинен бути укладений у подвійні лапки, інакше виникне помилка.

Щоб отримати доступ до даних у JSON-об’єкті, ви можете використовувати точкову або дужкову нотацію. У цьому прикладі ми використовуємо точкову нотацію, щоб отримати доступ до `age` з JSON-об’єкта:

```js
import data from "./example.json" with { type: "json" };

console.log(data.age);
```

Цей приклад використовує те, що називається `import` інструкцією, яка імпортує JSON-об’єкт у цей файл, щоб ми мали до нього доступ. Ви дізнаєтеся більше про `import` інструкцію в одному з наступних уроків.

Ви також можете використовувати дужкову нотацію для доступу до інформації з JSON-об’єктів. Ось приклад доступу до масиву `list of courses`:

```js
import data from "./example.json" with { type: "json" };

console.log(data["list of courses"]);
```

Використання дужкової нотації тут особливо корисне, оскільки ключ містить кілька слів, розділених пробілами. Якби ми спробували використати точкову нотацію, це призвело б до помилки.

Підсумовуючи, JSON — це універсальний формат, який може зберігати багато типів даних, включно з масивами та вкладеними об’єктами. Використовуючи точкову або дужкову нотацію, ви легко можете отримати доступ до значень, що зберігаються в JSON-об’єкті.
