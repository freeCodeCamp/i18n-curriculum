# --description--

このプロジェクトでは、freeCodeCamp News の著者に関するデータを取得します。
オンラインの情報源からデータを取得したい場合は、`API`（アプリケーションプログラミングインターフェース）を使う必要があります。
`API`は、組織の外部の人が内部データを取得できるようにします。

`fetch`からデータを受け取るために、`API`リクエストを送信してコードがデータを受け取れるようにする `GET` というメソッドがあります。

`GET`メソッドで `fetch()`リクエストを作成する方法は次の通りです。

```js
fetch("url-goes-here")
```

この URL に `GET`リクエストを送ってください：`"https://cdn.freecodecamp.org/curriculum/news-author-page/authors.json"`。まだコードの最後にセミコロンを付けないでください。
