# --description--

함수 $lcm(a, b)$는 $a$와 $b$의 최소공배수를 나타냅니다.

$A(n)$을 $1 ≤ i ≤ n$에 대해 $lcm(n, i)$ 값들의 평균이라고 하겠습니다.

예를 들어: $A(2) = \frac{2 + 2}{2} = 2$이고 $A(10) = \frac{10 + 10 + 30 + 20 + 10 + 30 + 70 + 40 + 90 + 10}{10} = 32$입니다.

$S(n) = \sum A(k)$를 $1 ≤ k ≤ n$에 대해 정의합니다.

$S(100) = 122\\,726$입니다.

$S(99\\,999\\,999\\,019)\bmod 999\\,999\\,017$를 구하세요.
