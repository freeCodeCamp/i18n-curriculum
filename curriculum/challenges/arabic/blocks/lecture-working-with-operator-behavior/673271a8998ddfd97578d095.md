# --interactive--

إذا كنت تعمل مع أرقام وتحتاج إلى زيادة أو تقليل قيمة بمقدار واحد، فإن معامِلات التزايد والتقليل تُسهّل المهمة. دعنا نشرحها بطريقة بسيطة.

تمثّل معامِلات التزايد والتقليل بـ `++` و `--` على التوالي. كلاهما يسمح لك بضبط قيمة متغير بمقدار `1`.

بدلاً من كتابة شيء مثل `x = x + 1` أو `x = x - 1`، يمكنك ببساطة استخدام `x++` لإضافة `1`، أو `x--` لطرح `1`. هذا أسرع وأنظف وأسهل للقراءة.

هناك نقطة مهمة في كيفية عمل معامِلات التزايد والتقليل: تأتي في شكلين، بادئة ولاحقة، والفرق هو متى يتم تحديث القيمة. بالنسبة لمعامل التزايد، البادئة هي `++x` واللاحقة هي `x++`.

البادئة (`++x`) تزيد قيمة المتغير أولاً، ثم تُرجع القيمة الجديدة. اللاحقة (`x++`) تُرجع القيمة الحالية للمتغير أولاً، ثم تزيدها:

:::interactive_editor

```js
let x = 5;

console.log(++x); // 6
console.log(x); // 6
```

:::

في الشفرة أعلاه، تعني `++x` "زد `x` أولاً، ثم استخدمه". لذا عند تسجيل `++x`، تحصل فورًا على القيمة بعد الزيادة، وهي `6`.

الآن، لننظر إلى مثال باستخدام الشكل اللاحق:

:::interactive_editor

```js
let y = 5;

console.log(y++); // 5
console.log(y); // 6
```

:::

في هذا المثال، تعني `y++` "استخدم `y` أولاً، ثم زدها". عند تسجيل `y++`، تحصل على `5`، لكن `y` تصبح `6` بعد هذا السطر من الشفرة.

معامل التقليل يعمل بنفس طريقة التزايد، لكنه يقلل القيمة بمقدار `1`. مرة أخرى، هناك شكلان: البادئة (`--x`) تقلل قيمة المتغير أولاً، ثم تُرجع القيمة الجديدة. واللاحقة (`x--`) تُرجع القيمة الحالية أولاً، ثم تقللها:

:::interactive_editor

```js
let x = 5;
console.log(--x); // 4
console.log(x); // 4

let y = 5;
console.log(y--); // 5
console.log(y); // 4
```

:::

فأي شكل يجب أن تستخدم: بادئة أم لاحقة؟ في كثير من الحالات، لا يهم أيهما تستخدم. كلاهما يؤدي المهمة. لكن إذا كنت تستخدم القيمة فورًا في تعبير، يصبح الفرق مهمًا. لننظر إلى هذا المثال:

:::interactive_editor

```js
let a = 5;
let b = ++a;
console.log(b); // 6 (a was incremented before assignment)

let c = 5;
let d = c++;
console.log(d); // 5 (c was incremented after assignment)
```

:::

إذا كنت تحتاج القيمة المحدثة فورًا، استخدم البادئة. إذا أردت القيمة الحالية أولاً ولا تهتم بالزيادة إلا لاحقًا، استخدم اللاحقة.
