# --description--

三角錐は球状のボールを使って構成されており、それぞれのボールはちょうど次の下のレベルの3つのボールの上に乗っています。

<img alt="四層の球状ボールで構成された三角錐" src="https://cdn.freecodecamp.org/curriculum/project-euler/exploring-pascals-pyramid.png" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

次に、頂点から各位置に至る経路の数を計算します。経路は頂点から始まり、現在の位置の真下にある3つの球のいずれかに向かって下へ進みます。したがって、ある位置に到達する経路の数は、その位置のすぐ上にある数の合計になります（位置によっては最大3つの数が上にあります）。

この結果がパスカルのピラミッドであり、各レベル n の数は三項展開 ${(x + y + z)}^n$ の係数です。

${(x + y + z)}^{200000}$ の展開において、${10}^{12}$ の倍数である係数はいくつありますか？
