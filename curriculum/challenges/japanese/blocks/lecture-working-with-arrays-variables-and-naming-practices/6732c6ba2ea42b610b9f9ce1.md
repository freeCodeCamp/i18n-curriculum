# --description--

前のレッスンで`length`プロパティを紹介しました。このプロパティは配列内の要素数を返します。ここでその動作を簡単におさらいしましょう。

```js
const fruits = ['apple', 'banana', 'orange'];
console.log(fruits.length); // 3
```

空のスロットを持つ配列も存在します。空のスロットとは、何も入っていないスロットのことです。これは値が`undefined`の配列とは異なります。このような配列はスパース配列と呼ばれます。例を示します。

```js
const sparseArray = [1, , , 4];
console.log(sparseArray.length); // 4
```

この場合、定義されている要素は`1`と`4`の2つだけですが、長さは`4`です。なぜなら最大のインデックス（`3`）に`1`を足したものが長さの`4`になるからです。

次に、固定長の空配列を作成する方法について説明します。JavaScriptにはいくつか方法がありますが、一般的な方法の一つは数値の引数を使って`Array()`コンストラクタを使うことです。`Array()`コンストラクタは`new`キーワードと一緒に使って新しい配列を作成できます。例を示します。

```js
const emptyArray = new Array(5);
console.log(emptyArray.length); // 5
console.log(emptyArray); // [ , , , , ]
```

この例では`Array(5)`を使って新しい配列を作成しています。これは長さが`5`で全てのスロットが空のスパース配列を作成します。

固定長の空配列を作成する別の方法は、長さの引数を使って`Array.from()`メソッドを使うことです。`new Array(n)`とは異なり、このメソッドは指定した長さの配列を作成し、すべての要素が存在し値は`undefined`になります。

```js
const fixedLengthArray = Array.from({ length: 5 });
console.log(fixedLengthArray.length); // 5
console.log(fixedLengthArray); // [undefined, undefined, undefined, undefined, undefined]
```

特定の長さの配列を作成し、デフォルト値で埋めたい場合は`Array.fill()`メソッドを使えます。

```js
const filledArray = new Array(3).fill(0);
console.log(filledArray); // [0, 0, 0]
```

これは長さ3の配列を作成し、すべての要素を`0`の値で埋めます。注意：オブジェクトで埋める場合、すべてのスロットは同じオブジェクトを参照します。独立したコピーが必要な場合はコールバックや`Array.from()`を使ってください。

配列の長さを取得し、固定長の配列を作成する方法を理解することは、多くのプログラミングタスクで重要です。特に特定のアルゴリズムやデータ構造のために配列を初期化する必要がある場合に役立ちます。
