# --explanation--

Um `log` é um registro de eventos, ações ou mensagens criadas por um sistema de computador ou software.Por exemplo:

`The system logs showed several errors before the crash.` – Isso significa que os registros tinham informações sobre problemas que ocorreram.

Um `crash` é quando um sistema de computador ou programa para de funcionar corretamente de repente.Por exemplo:

`The application crash caused all unsaved data to be lost.` – Isso significa que o programa falhou inesperadamente e os dados não foram salvos.
