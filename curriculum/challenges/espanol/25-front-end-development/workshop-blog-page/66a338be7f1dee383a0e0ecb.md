# --description--

Para la segunda entrada del blog, necesitará agregar otro elemento `article`.

Dentro del elemento `article`, agregue un elemento `h3` con el texto de `Mr. Whiskers' First Bath`.

Debajo de su elemento `h3`, agregue dos párrafos de texto lorem ipsum.
