# --description--

Un `challenge` indica un'attività o un problema che richiede abilità e impegno per essere risolto. Nel coding, un `coding challenge` è un problema o un'attività che necessita di competenze di programmazione per essere completato.

`Fresh` in questo contesto significa nuovo o diverso. Chiedere un `fresh perspective` significa cercare un punto di vista nuovo o diverso su un problema o un'attività.
