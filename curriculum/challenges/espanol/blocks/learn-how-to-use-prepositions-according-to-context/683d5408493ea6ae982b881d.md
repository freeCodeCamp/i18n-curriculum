# --questions--

## --text--

¿Qué oración usa correctamente la palabra `near`?

## --answers--

`The developer near fixed the bug this morning.`

### --feedback--

No se debe usar `Near` antes de un verbo de esta manera.

---

`He near completed the project.`

### --feedback--

Esta no es la estructura correcta. Usa `almost` u otra palabra, no `near`, en este contexto.

---

`The printer is near the manager's office.`

---

`She is near finishing the user guide.`

### --feedback--

La frase debería ser `almost finished` o `close to finishing`, no `near finishing`.

## --video-solution--

3

# --explanation--

`Near` se usa para describir la cercanía física o la corta distancia entre objetos o lugares.

En la oración correcta, se muestra claramente la posición de la impresora en relación con la oficina del gerente. Este es el uso intencionado y adecuado de `near` como preposición para describir la ubicación.
