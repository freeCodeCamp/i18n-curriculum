# --questions--

## --text--

以下 XML 有什麼問題？:

```xml
<person>
  <name>Chuck</name>
  <phone type="intl">
    +1 734 303 4456
  <email hide="yes" />
</person>
```

## --answers--

Email 頁籤缺少結束標籤。

---

空白會導致 XML 無效。

---

電話頁籤缺少結束標籤。

---

純文字應使用 UTF-8 編碼。
