# --explanation--

`Should` ni kitenzi cha modal kinachodhihirisha ushauri, mapendekezo, au njia bora ya kufanya — lakini si wajibu.

- `You should update your password regularly.` - Ni wazo zuri kubadilisha nenosiri lako mara kwa mara kwa usalama.

- `You should check for errors in the code before running it.` - Ni desturi nzuri kutafuta makosa kabla ya kutekeleza msimbo.

Kwa kuwa Jake anasema `we should focus on the critical systems first`, anatoa ushauri mzito, lakini timu haina wajibu mkali wa kufanya hivyo.
