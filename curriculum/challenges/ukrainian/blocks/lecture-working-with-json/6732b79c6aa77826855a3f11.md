# --interactive--

Існують два потужні методи в JavaScript для роботи з JSON-даними: `JSON.parse()` та `JSON.stringify()`. Ці методи часто використовують для перетворення між JSON-рядками та JavaScript-об’єктами.

`JSON.stringify()` використовується для перетворення JavaScript-об’єкта у JSON-рядок. Це корисно, коли потрібно зберегти або передати дані у форматі, який легко поділитися або передати між системами.

Ось як ви можете використати метод `JSON.stringify()`:

:::interactive_editor

```js
const user = {
  name: "John",
  age: 30,
  isAdmin: true
};

const jsonString = JSON.stringify(user);
console.log(jsonString);
```

:::

Метод `JSON.stringify()` також приймає необов’язковий параметр, який називається replacer, і може бути функцією або масивом. Ось приклад використання масиву для необов’язкового параметра replacer:

:::interactive_editor

```js
const developerObj = {
  firstName: "Jessica",
  isAwesome: true,
  isMusician: true,
  country: "USA",
};

// result: {"firstName":"Jessica","country":"USA"}
console.log(JSON.stringify(developerObj, ["firstName", "country"]));
```

:::

У цьому прикладі у нас є `developerObj` з чотирма властивостями. Коли ми використовуємо метод `JSON.stringify()`, ми можемо передати масив як другий параметр і вказати, які властивості ми хочемо перетворити у рядок. Результатом буде рядок, що містить лише властивості `firstName` та `country`.

Ще одним необов’язковим параметром для методу `JSON.stringify()` є параметр spacer. Він дозволяє контролювати відступи у рядку, що повертається:

:::interactive_editor

```js
const developerObj = {
  firstName: "Jessica",
  isAwesome: true,
  isMusician: true,
  country: "USA",
};

console.log(JSON.stringify(developerObj, null, 2));

/* result
{
  "firstName": "Jessica",
  "isAwesome": true,
  "isMusician": true,
  "country": "USA"
}
*/
```

:::

Зазвичай ви не будете використовувати ці необов’язкові параметри для методу `JSON.stringify()`, але корисно знати про них.

Ще один метод, який ви часто використовуватимете у програмуванні, — це метод `JSON.parse()`. `JSON.parse()` перетворює JSON-рядок назад у JavaScript-об’єкт. Це корисно, коли ви отримуєте JSON-дані з веб-сервера або з `localStorage` і вам потрібно опрацьовувати ці дані у своєму застосунку. Ви дізнаєтеся більше про `localStorage` у майбутньому уроці.

Ось приклад, як працювати з методом `JSON.parse()`:

:::interactive_editor

```js
const jsonString = '{"name":"John","age":30,"isAdmin":true}';
const userObject = JSON.parse(jsonString);
console.log(userObject);

// Result:
// { name: 'John', age: 30, isAdmin: true }
```

:::

Це дозволяє працювати з даними у вашій програмі як з звичайним JavaScript-об’єктом, що полегшує їх обробку та використання.

У майбутніх модулях ми продовжимо вивчати, як працювати з JSON та методами `JSON.parse()` і `JSON.stringify()`.
