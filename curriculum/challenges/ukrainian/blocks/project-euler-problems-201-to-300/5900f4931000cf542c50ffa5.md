# --description--

Для натурального числа $k$ визначимо $d(k)$ як суму цифр числа $k$ у його звичайному десятковому представленні. Тобто $d(42) = 4 + 2 = 6$.

Для натурального числа $n$ визначимо $S(n)$ як кількість натуральних чисел $k &lt; {10}^n$, які мають такі властивості:

- $k$ ділиться на 23 без остачі, та
- $d(k) = 23$.

Відомо, що $S(9) = 263\\,626$ і $S(42) = 6\\,377\\,168\\,878\\,570\\,056$.

Знайдіть $S({11}^{12})$ і подайте відповідь за модулем ${10}^9$.
