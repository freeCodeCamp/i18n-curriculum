# --interactive--

Die Eigenschaften `grid-column` und `grid-row` ermöglichen es Ihnen, die horizontale und vertikale Platzierung von Grid-Elementen innerhalb eines Grid-Layouts festzulegen.

Mit anderen Worten: Beide erlauben es Ihnen, zu steuern, wo ein Grid-Element beginnt und endet, indem Sie sich auf Gitterlinien beziehen. Diese Gitterlinien sind die Grenzen, die die Reihen und Spalten trennen, die Sie bereits mit den Eigenschaften `grid-template-rows` und `grid-template-columns` definiert haben.

Schauen wir uns die Syntax beider Eigenschaften `grid-row` und `grid-column` sowie ein praktisches Beispiel an.

Hier ist die Syntax der Eigenschaft `grid-row`:

```css
grid-row: <start-line> / <end-line>;
```

Und von `grid-column`:

```css
grid-column: <start-line> / <end-line>;
```

`<start-line>` ist die Gitterlinie, an der das Element beginnt, und `<end-line>` ist die Gitterlinie, an der das Element endet. Beide sind 1-basiert, das heißt, Sie zählen sie ab 1, nicht ab 0.

Denken Sie daran, dass die Gitterlinien für Reihen basierend auf der Anzahl der Reihen generiert werden, die in der Eigenschaft `grid-template-rows` angegeben sind. Dasselbe gilt für Spalten mit der Eigenschaft `grid-template-columns`.

Für das praktische Beispiel beginnen wir mit den Linien, die der Browser generiert, wenn Sie sowohl die Eigenschaften `grid-template-rows` als auch `grid-template-columns` verwenden.

Hier ist ein Beispiel-Grid mit 4 Spalten und 3 Reihen:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}
```

:::

Die Inspektion des Grid-Containers (der Klasse `grid`) zeigt, dass jede Reihe und Spalte von zwei Linien begrenzt wird – einer Startlinie am Anfang der Reihe oder Spalte und einer Endlinie am Ende der Reihe oder Spalte.

Sie können diese Linien mit den Eigenschaften `grid-row` und `grid-column` ansprechen, um zu bestimmen, wo Sie ein Element platzieren sollten.

So können Sie das erste Grid-Element dazu bringen, die ersten zwei Spalten einzunehmen:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}

.item1 {
  grid-column: 1 / 3;
}
```

:::

Damit sagen Sie, dass das erste Grid-Element bei Spalte 1 beginnen und direkt vor Spalte 3 enden soll.

Das erste Element nimmt nun zwei Spalten ein, und das vierte Element wird in die zweite Reihe verschoben. Cool!

Wenn Sie möchten, dass das erste Element auch zwei Reihen einnimmt, können Sie eine `grid-row` von `1 / 3` angeben:

:::interactive_editor

```html
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;

  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}

.item1 {
  grid-column: 1 / 3;
  grid-row: 1 / 3;
}
```

:::

Sie können auch das Schlüsselwort `span` verwenden, um dem Grid-Element mitzuteilen, über wie viele Reihen und Spalten es sich erstrecken soll. Zum Beispiel ist `1 / 3` dasselbe wie `1 / span 2`:

```html
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}

.item1 {
  grid-column: 1 / span 2;
  grid-row: 1 / span 2;
}
```

:::

Sie können diese Technik auf jedes Element im Grid anwenden und es überall platzieren, wo Sie möchten.
