# --explanation--

`Real-time alerts` — це повідомлення або попередження, які з’являються одразу, як щось відбувається. Наприклад:

`The app sends real-time alerts if someone logs in from a new location.` – це означає, що щоразу, коли хтось входить з іншого місця, користувач отримує миттєве попередження.

`Graphics` — це зображення або візуальні елементи, які використовують для передачі інформації. Наприклад:

`The dashboard uses graphics to show project progress.` – це означає, що візуальні інструменти, як діаграми або іконки, використовують, щоб зробити дані легшими для розуміння.
