# --explanation--

`Build tools`는 코드 컴파일, 테스트, 소프트웨어 프로젝트 패키징 같은 반복 작업을 자동화하고 관리하는 데 도움을 주는 소프트웨어 프로그램입니다. 예를 들어:

`Maven is a popular build tool for Java projects, helping automate the compilation process.` - Maven은 Java 애플리케이션 빌드 과정을 간소화하여 코드 컴파일과 패키지 생성 같은 작업을 자동화한다는 뜻입니다.

`Built-in`는 소프트웨어에 이미 포함되어 있어 추가 설치나 설정이 필요 없는 기능을 의미합니다. 예를 들어:

`Visual Studio has built-in tools for debugging code.` - Visual Studio에 디버깅 도구가 내장되어 있어 사용자가 별도의 도구를 설치하지 않고도 디버깅을 시작할 수 있다는 뜻입니다.

`Version control`는 코드 변경 사항을 추적하고 관리하는 시스템으로, 팀이 같은 프로젝트에서 작업하면서 작업 손실이나 충돌 없이 협업할 수 있게 합니다. 예를 들어:

`Git is a widely-used version control system for tracking code changes and collaborating with teammates.` - Git은 개발자가 코드의 여러 버전을 관리하고 같은 프로젝트에서 함께 작업할 수 있도록 돕는다는 뜻입니다.
