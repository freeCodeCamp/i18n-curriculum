# --explanation--

`Should` es un verbo modal que expresa consejo, recomendación o la mejor estructura a seguir, pero no una obligación.

- `You should update your password regularly.` - Es una buena idea cambiar tu contraseña con frecuencia por seguridad.

- `You should check for errors in the code before running it.` - Es una buena práctica encontrar errores antes de ejecutar el código.

Como Jake dice `we should focus on the critical systems first`, está dando una recomendación fuerte, pero el equipo no está estrictamente obligado a hacerlo.
