# --description--

단어 `what`는 절을 도입하는 데 사용할 수 있습니다. 보통 명사절을 도입하며, 명사에 대한 추가 정보를 제공하거나 문장의 일부 역할을 합니다.

명사절은 더 큰 문장 내에서 명사 역할을 하는 절입니다. 보통 주어와 동사를 포함하며, 문장에서 주어, 목적어 또는 보어 역할을 할 수 있습니다.

예를 들어, 문장 `She doesn't know what to buy`에서 `what to buy`는 `what`가 구매를 결정해야 하는 대상을 나타내는 절입니다.

또 다른 예로 `Tell me what you heard.`가 있으며 `What you heard`가 절이고, `what`는 들은 구체적인 내용을 설명하며 동사 `heard`의 목적어 역할을 합니다.

`These`는 앞서 언급되었거나 문맥에서 이해된 특정 항목을 가리키는 복수 지시대명사입니다.
