# --description--

Sea $n$ un entero positivo. Un triple entero ($a$, $b$, $c$) se llama un triple de factorización de $n$ si:

- $1 ≤ a ≤ b ≤ c$
- $a \times b \times c = n$.

Define $f(n)$ como $a + b + c$ para la estructura de factorización ($a$, $b$, $c$) de $n$ que minimiza $\frac{c}{a}$. Se puede demostrar que esta estructura es única.

Por ejemplo, $f(165) = 19$, $f(100\,100) = 142$ y $f(20!) = 4\,034\,872$.

Encuentra $f(43!)$.
