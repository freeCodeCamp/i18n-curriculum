# --explanation--

`And when does the team perform the final checks?` використовує `Present Simple` час (`does perform`) для запитання про заплановану майбутню подію.

Правильна відповідь, `The final system checks are set for Friday night`, також використовує `Present Simple` для опису фіксованого плану.

`Present Simple` часто використовується для майбутніх подій, які є частиною розкладу або графіка. Наприклад:

- `The train departs at 8 PM tonight.` — це вказує на запланований від’їзд.

- `Our meeting starts tomorrow morning.` — це показує запланований час для майбутньої події.

У цьому контексті `Present Simple` робить відповідь лаконічною та доречною для обговорення фіксованої події.
