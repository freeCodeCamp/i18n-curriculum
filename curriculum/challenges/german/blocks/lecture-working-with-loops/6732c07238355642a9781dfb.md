# --interactive--

Eine `break`-Anweisung wird verwendet, um eine Schleife vorzeitig zu verlassen, während eine `continue`-Anweisung dazu dient, die aktuelle Iteration einer Schleife zu überspringen und mit der nächsten fortzufahren.

Hier ist ein Beispiel für die Verwendung einer `break`-Anweisung in einer `for`-Schleife:

:::interactive_editor

```js
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    break;
  }
  console.log(i);
}
```

:::

Im obigen Beispiel beginnt die Schleife bei `0` zu zählen, und solange `i` kleiner als `10` ist, läuft die Schleife weiter.

Innerhalb der Schleife prüfen wir, ob `i` gleich `5` ist. Falls ja, verwenden wir die `break`-Anweisung, um die Schleife vorzeitig zu verlassen. Wenn nicht, geben wir den Wert von `i` in der Konsole aus. Die Ausgabe des Codes zeigt also die Zahlen `0`, `1`, `2`, `3` und `4` an.

Die `break`-Anweisung ist nützlich, wenn Sie eine Schleife basierend auf einer bestimmten Bedingung vorzeitig verlassen möchten. Zum Beispiel, wenn Sie in einem Array nach einem bestimmten Wert suchen, können Sie eine `break`-Anweisung verwenden, um die Schleife zu verlassen, sobald Sie den Wert gefunden haben.

Manchmal möchten Sie eine bestimmte Iteration einer Schleife überspringen, ohne die Schleife vollständig zu verlassen. Hier kommt die `continue`-Anweisung ins Spiel. Hier ist ein Beispiel für die Verwendung einer `continue`-Anweisung in einer `for`-Schleife:

:::interactive_editor

```js
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    continue;
  }
  console.log(i);
}
```

:::

Wie zuvor haben wir `i` auf `0` initialisiert und eine Bedingung, die die Schleife so lange ausführt, wie `i` kleiner als `10` ist.

Innerhalb der Schleife verwenden wir die `i`-Anweisung, um die aktuelle Iteration zu überspringen und mit der nächsten fortzufahren, wenn `5` gleich `continue` ist.

Die Ausgabe dieses Codes zeigt die Zahlen `0`, `1`, `2`, `3`, `4`, `6`, `7`, `8` und `9` an. Die Zahl `5` wird wegen der `continue`-Anweisung übersprungen.

Eine weitere Möglichkeit bei der Verwendung sowohl der `break`- als auch der `continue`-Anweisungen besteht darin, Bezeichnungen zu verwenden, um anzugeben, welche Schleife Sie abbrechen oder fortsetzen möchten.

Das ist nützlich, wenn Sie verschachtelte Schleifen haben und den Ablauf der äußeren Schleife von der inneren Schleife aus steuern möchten.

Hier ist ein Beispiel für die Verwendung von Bezeichnungen mit der `break`-Anweisung:

:::interactive_editor

```js
outerLoop: for (let i = 0; i < 3; i++) {
  innerLoop: for (let j = 0; j < 3; j++) {
    if (i === 1 && j === 1) {
      break outerLoop;
    }
    console.log(`i: ${i}, j: ${j}`);
  }
}
```

:::

In diesem Beispiel haben wir eine äußere `for`-Schleife mit der Bezeichnung `outerLoop` und eine innere `for`-Schleife mit der Bezeichnung `innerLoop`.

Wenn `i` gleich `1` und `j` gleich `1` ist, verwenden wir die `break`-Anweisung mit der Bezeichnung `outerLoop`, um die äußere Schleife vorzeitig zu verlassen. Dadurch werden sowohl die innere als auch die äußere Schleife beendet.

Die Ausgabe dieses Codes wird Folgendes in der Konsole ausgeben:

```md
"i: 0, j: 0"
"i: 0, j: 1"
"i: 0, j: 2"
"i: 1, j: 0"
```

Meistens werden Sie keine Notwendigkeit finden, Bezeichnungen mit den `break`- und `continue`-Anweisungen zu verwenden, aber es ist gut zu wissen, dass Sie diese Option haben, falls Sie sie jemals brauchen.
