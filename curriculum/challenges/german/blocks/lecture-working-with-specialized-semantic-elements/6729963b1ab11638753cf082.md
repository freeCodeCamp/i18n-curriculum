# --interactive--

Das Element für unausgesprochene Annotationen, kurz `u`, wird verwendet, um inline Text darzustellen, dem nicht-textuelle Annotationen hinzugefügt wurden.

Hier ist ein Beispiel für die Verwendung des `u`-Elements, um verschiedene Rechtschreibfehler hervorzuheben:

:::interactive_editor

```html
<p>
  You can use the unarticulated annotation element to highlight
  <u>inccccort</u> <u>spling</u> <u>issses</u>.
</p>
```

:::

In dem Beispiel sind die Wörter `incorrect`, `spelling` und `issues` falsch geschrieben. Die Standardformatierung für das `u`-Element ist eine schwarze Unterstreichung unter dem Text.

In HTML4 wurde das `u`-Element für Styling-Zwecke verwendet. In HTML5 sollte das `u`-Element jedoch nur verwendet werden, um anzuzeigen, dass Text mit nicht-textuellen Annotationen versehen ist.

Wenn Sie einen Text unterstreichen möchten, sollten Sie stattdessen CSS verwenden und nicht HTML.

Das Durchstreich-Element, kurz `s`, sollte verwendet werden, um darzustellen, dass Text nicht mehr korrekt oder relevant ist. Hier ist ein Beispiel für die Verwendung des `s`-Elements, um die Absage einer Aktivität zu zeigen:

:::interactive_editor

```html
<p><s>Tomorrow's hike will be meeting at noon.</s></p>

<p>Due to unforeseen weather conditions, the hike has been canceled.</p>
```

:::

In diesem Beispiel ist der erste Satz durchgestrichen, weil die Wanderung aus Wettergründen abgesagt wurde.

Das `s`-Element sollte niemals nur verwendet werden, um Änderungen an einem Dokument zu zeigen. Passendere Elemente dafür sind das Element für gelöschten Text und das Element für eingefügten Text.

Das `ruby`-Element stellt kleinen Text dar, der über oder unter dem Haupttext angezeigt wird. Es wird typischerweise verwendet, um die Aussprache ostasiatischer Schriftzeichen zu zeigen. Hier ist ein Beispiel für das `ruby`-Element von der MDN-Web-Dokumentationsseite:

:::interactive_editor

```html
<ruby> 明日 <rp>(</rp><rt>Ashita</rt><rp>)</rp> </ruby>
```

:::

Das `rp`-Element, oder das Fallback-Klammer-Element für ruby, wird als Fallback für Browser verwendet, die keine Unterstützung für die Anzeige von ruby-Annotationen bieten.

Das `rt`-Element, oder ruby-Text-Element, wird verwendet, um den Text für die ruby-Annotation anzugeben. Dieser Text wird üblicherweise für Aussprache oder Übersetzungsdetails in der ostasiatischen Typografie verwendet.

Während das `ruby`-Element für andere Arten von Annotationen verwendet werden kann, ist der häufigste Anwendungsfall die ostasiatische Typografie.
