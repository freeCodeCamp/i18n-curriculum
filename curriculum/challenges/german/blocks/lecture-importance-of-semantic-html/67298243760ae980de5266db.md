# --description--

Semantik bezeichnet die Bedeutung von Wörtern oder Phrasen in einer Sprache. In HTML, das ebenfalls eine Sprache ist, haben Elemente auch ihre eigene semantische Bedeutung. Tatsächlich können Sie Ihr HTML-Dokument so betrachten wie ein Textdokument. Und ähnlich wie bei einem Textdokument können Sie Überschriften, Bilder, fett gedruckten Text und andere Formatierungen haben.

Die semantische Bedeutung eines Elements bezieht sich darauf, welche besondere Information dieses Element vermittelt. Die semantische Bedeutung eines `p`-Elements ist zum Beispiel ein Absatz mit Text:

```html
<p>
  Let me tell you about my fantastic holiday in Paris.
  I saw the impressive Eiffel Tower up close!
</p>
```

Die meisten Elemente haben eine semantische Bedeutung. Das `div`-Element ist eines der wenigen, die keine haben. Aber warum ist das wichtig?

Vor allem sorgt die Verwendung von richtigem semantischem HTML für die beste Erfahrung für Nutzer von Hilfstechnologien wie Screenreadern. Außerdem kann semantisches HTML Ihre Suchmaschinenplatzierungen verbessern. Das nennt man Suchmaschinenoptimierung oder SEO.

Schließlich kann die Verwendung korrekter semantischer Elemente Ihre Entwicklungserfahrung verbessern. Anstatt sich durch eine Menge `div`s zu wühlen, um Ihre Navigationsleiste zu finden, können Sie das `nav`-Element direkt bearbeiten und wissen genau, was Sie ändern. Im Verlauf dieses Abschnitts lernen Sie mehr über diese Themen, wie Sie semantische Elemente verwenden und warum semantisches HTML so wichtig ist.
