# --explanation--

`To speak with someone about something` bedeutet, ein Gespräch oder eine Diskussion mit einer Person über ein bestimmtes Thema zu führen. Zum Beispiel:

`She spoke with her manager about taking some days off.` – Das bedeutet, sie hatte ein Gespräch mit ihrem Manager über Urlaub.

Die Vergangenheitsform von `speak` ist `spoke`. In diesem Fall spricht Sophie über etwas, das bereits passiert ist – ihr Gespräch mit Jake.
