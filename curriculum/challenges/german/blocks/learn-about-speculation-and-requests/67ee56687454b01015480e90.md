# --explanation--

Eine `for loop` ist nützlich, wenn Sie wissen, wie oft die Schleife ausgeführt werden soll. Zum Beispiel:

- `A for loop is useful for going through a list.` – Sie können sie verwenden, um jedes Element in einer Liste zu überprüfen.

- `The for loop stops when it reaches the last item.` – Die Schleife endet, wenn keine Elemente mehr vorhanden sind.

Eine `while loop` ist besser, wenn die Anzahl der Wiederholungen von einer Bedingung abhängt. Zum Beispiel:

- `A while loop runs until a condition is false.` – Die Schleife läuft weiter, solange die Regel wahr ist.

- `Be careful, a while loop can run forever!` – Wenn Sie sie nicht stoppen, endet die Schleife nicht.
