## Orodha za Python

- **Utangulizi**: Katika Python, aina ya data ya orodha ni mfululizo uliopangwa wa vipengele ambavyo vinaweza kuwa mfuatano wa herufi, nambari au hata orodha nyingine. Orodha ni zinazobadilika na zina kielezo kuanzia sufuri.

```python
cities = ['Los Angeles', 'London', 'Tokyo']
```

- **Kupata vipengele katika orodha**: Ili kupata kipengele kutoka orodha ya `cities`, unaweza kurejea nambari ya kielezo katika mfululizo:

```python
cities = ['Los Angeles', 'London', 'Tokyo']
cities[0] # Los Angeles
```

- **Kupata vipengele kwa kutumia kielezo hasi**: Ili kupata kipengele cha mwisho cha orodha yoyote, unaweza kutumia `-1` kama nambari ya kielezo:

```python
cities = ['Los Angeles', 'London', 'Tokyo']
cities[-1] # Tokyo
```

- Kielezo hasi hutumika kupata vipengele kuanzia mwisho wa orodha badala ya mwanzo kwa kielezo cha `0`.

- **Kuunda orodha kwa kutumia kijenzi cha `list()`**: Orodha pia zinaweza kuundwa kwa kutumia kijenzi cha `list()`. Kijenzi cha `list()` hutumika kubadilisha iterable kuwa orodha:

```python
developer = 'Jessica'

print(list(developer)) 
# Result: ['J', 'e', 's', 's', 'i', 'c', 'a']
```

- **Kupata urefu wa orodha**: Unaweza kutumia kitendakazi cha `len()` kupata urefu wa orodha:

```python
numbers = [1, 2, 3, 4, 5]
len(numbers) # 5
```

- **Ubadilishaji wa orodha**: Orodha ni zinazobadilika, maana unaweza kusahihisha kipengele chochote ndani ya orodha mradi tu uweke nambari halali ya kielezo. Ili kusahihisha orodha kwa kielezo fulani, unaweza kuweka thamani mpya kwa kielezo hicho:

```python
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[0] = 'JavaScript'
print(programming_languages) # ['JavaScript', 'Java', 'C++', 'Rust']
```

- **Hitilafu ya Kielezo nje ya Mipaka**: Ikiwa utaweka kielezo (chenye thamani chanya au hasi) ambacho kiko nje ya mipaka ya orodha, basi utapokea `IndexError`:

```python
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[10] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: list assignment index out of range
"""
```

- **Kuondoa vipengele kutoka orodha**: Vipengele vinaweza kuondolewa kutoka orodha kwa kutumia neno kuu la `del`:

```python
developer = ['Jane Doe', 23, 'Python Developer']
del developer[1]
print(developer) # ['Jane Doe', 'Python Developer']
```

- **Kukagua kama kipengele kiko kwenye orodha**: Neno kuu la `in` linaweza kutumika kuangalia kama kipengele kiko kwenye orodha:

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

- **Kupanga ndani orodha**: Orodha zinaweza kupangwa ndani ya orodha nyingine:

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
```

- Ili kupata orodha iliyopangwa ndani, utahitaji kuipata kwa kutumia kielezo cha `2` kwa kuwa orodha zina kielezo kuanzia sufuri.

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2] # ['Python', 'Rust', 'C++']
```

- Ili kupata lugha ya pili kutoka orodha iliyopangwa ndani, utahitaji kuipata kwa kutumia kielezo cha `1`:

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2][1] # Rust
```

- **Kutoa thamani kutoka orodha (Unpacking)**: Kutoa thamani kutoka orodha ni mbinu inayotumika kuweka thamani kutoka orodha kwenye vigezo vipya. Hapa kuna mfano wa kutoa orodha ya `developer` kwenye vigezo vipya vinavyoitwa `name`, `age` na `job` kama ifuatavyo:

```py
developer = ['Alice', 34, 'Rust Developer']
name, age, job = developer
```

- Ikiwa idadi ya vigezo upande wa kushoto wa opereta ya mgawo haifai na idadi ya vitu kwenye orodha, basi utapokea `ValueError`.

- **Kukusanya vitu vilivyobaki kutoka orodha**: Ili kukusanya vitu vyote vilivyobaki kutoka orodha, unaweza kutumia opereta ya nyota (`*`) kama ifuatavyo:

```py
developer = ['Alice', 34, 'Rust Developer']
name, *rest = developer
```

- **Kuchukua sehemu ya orodha (Slicing)**: Kuchukua sehemu ni dhana ya kupata sehemu ya orodha kwa kutumia opereta ya kuchukua sehemu `:`. Ili kuchukua sehemu ya orodha kuanzia kielezo cha `1` na kuishia kabla ya kielezo cha `3`, unaweza kutumia sintaksia ifuatayo:

```python
desserts = ['Cake', 'Cookies', 'Ice Cream', 'Pie']
desserts[1:3] # ['Cookies', 'Ice Cream']
```

- **Muda wa hatua**: Pia inawezekana kubainisha muda wa hatua unaoamua kwa kiasi gani kuongezeka kati ya vigezo. Hapa kuna mfano ikiwa unataka kutoa orodha ya nambari za hata kwa kutumia kuchukua sehemu:

```python
numbers = [1, 2, 3, 4, 5, 6]
numbers[1::2] # [2, 4, 6]
```

## Njia za orodha

- **append()**: Inatumika kuongeza kipengele mwishoni mwa orodha. Hapa kuna mfano wa kutumia njia ya `append()` kuongeza nambari `6` kwenye orodha ya `numbers`:

```py
numbers = [1, 2, 3, 4, 5]
numbers.append(6)
print(numbers) # [1, 2, 3, 4, 5, 6]
```

- **Kuongeza orodha**: Njia ya `append()` pia inaweza kutumika kuongeza orodha moja mwishoni mwa nyingine:

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.append(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, [6, 8, 10]]
```

- **extend()**: Inatumika kuongeza vitu vingi mwishoni mwa orodha. Hapa kuna mfano wa kuongeza nambari `6`, `8`, na `10` mwishoni mwa orodha ya `numbers`:

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.extend(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, 6, 8, 10]
```

- **insert()**: Inatumika kuingiza kipengele kwenye kielezo maalum cha orodha. Hapa kuna mfano wa kutumia njia ya `insert()`:

```py
numbers = [1, 2, 3, 4, 5]
numbers.insert(2, 2.5)

print(numbers) # [1, 2, 2.5, 3, 4, 5]
```

- **remove()**: Inatumika kuondoa kipengele kutoka orodha. Njia ya `remove()` itaondoa tukio la kwanza tu la kipengele ndani ya orodha:

```py
numbers = [1, 2, 3, 4, 5, 5, 5]
numbers.remove(5)

print(numbers) # [1, 2, 3, 4, 5, 5]

```

- **pop()**: Inatumika kuondoa kipengele maalum kutoka orodha na kurudisha kipengele hicho:

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop(1) # The number 2 is returned
```

- Ikiwa hutaelezea kipengele kwa njia ya `pop`, basi kipengele cha mwisho kitatolewa.

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop() # The number 5 is returned
```

- **clear()**: Inatumika kuondoa vitu vyote kutoka orodha:

```py
numbers = [1, 2, 3, 4, 5]
numbers.clear()

print(numbers) # []
```

- **sort()**: Njia ya `sort()` hutumika kupanga vipengele mahali pale pale. Hapa kuna mfano wa kupanga orodha ya nasibu ya `numbers` mahali pale:

```py
numbers = [19, 2, 35, 1, 67, 41]
numbers.sort()

print(numbers) # [1, 2, 19, 35, 41, 67]
```

- **sorted()**: Kitendakazi cha ndani kinachorudisha orodha mpya iliyopangwa badala ya kubadilisha orodha ya awali:

```py
numbers = [19, 2, 35, 1, 67, 41]
sorted_numbers = sorted(numbers)

print(sorted_numbers) # [1, 2, 19, 35, 41, 67]
print(numbers) # [19, 2, 35, 1, 67, 41]
```

- **reverse()**: Inatumika kubadilisha mpangilio wa vipengele katika orodha:

```py
numbers = [6, 5, 4, 3, 2, 1]
numbers.reverse()

print(numbers) # [1, 2, 3, 4, 5, 6]
```

- **index()**: Inatumika kupata kielezo cha tukio la kwanza la kipengele katika orodha:

```python
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('Java') # 1
```

- Ikiwa kipengele hakiwezi kupatikana kwa njia ya `index()`, basi matokeo yatakuwa `ValueError`.

## Tuple katika Python

- **Maelezo**: Tuple ni aina ya data ya Python inayotumika kuunda mfululizo uliopangwa wa thamani. Tuple zinaweza kuwa na aina mchanganyiko ya data:

```py
developer = ('Alice', 34, 'Rust Developer')
```

- Tuple ni zisizobadilika, maana vipengele ndani ya tuple haviwezi kubadilishwa baada ya kuundwa. Ukijaribu kusahihisha kipengele ndani ya tuple, utapokea `TypeError`:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')
programming_languages[0] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: "tuple" object does not support item assignment
"""
```

- **Kupata vipengele kutoka tuple**: Ili kupata kipengele kutoka tuple, tumia uandishi wa mabano na nambari ya kielezo:

```py
developer = ('Alice', 34, 'Rust Developer')
developer[1] # 34
```

- Kielezo hasi kinaweza kutumika kupata vipengele kuanzia mwisho wa tuple:

```py
numbers = (1, 2, 3, 4, 5)
numbers[-2] # 4
```

- Ukijaribu kuweka kielezo kinachozidi au sawa na urefu wa tuple, basi utapokea `IndexError`:

```py
numbers = (1, 2, 3, 4, 5)
numbers[7]

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: tuple index out of range
"""
```

- Tuple pia inaweza kuundwa kwa kutumia kijenzi cha `tuple()`. Ndani ya kijenzi, unaweza kuweka iterables tofauti kama mfuatano wa herufi, orodha na hata tuple nyingine.

```python
developer = 'Jessica'

print(tuple(developer)) 
# Result: ('J', 'e', 's', 's', 'i', 'c', 'a')
```

- **Kukagua vitu katika tuple**: Ili kuangalia kama kipengele kiko katika tuple, unaweza kutumia neno kuu la `in` kama ifuatavyo:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

- **Kutoa vitu kutoka tuple**: Vitu vinaweza kutolewa kutoka tuple kama ifuatavyo:

```py
developer = ('Alice', 34, 'Rust Developer')
name, age, job = developer
```

- Ikiwa unahitaji kukusanya vitu vilivyobaki kutoka tuple, unaweza kutumia opereta ya nyota (`*`) kama ifuatavyo:

```py
developer = ('Alice', 34, 'Rust Developer')
name, *rest = developer
```

- **Kuchukua sehemu ya tuple**: Kuchukua sehemu kunaweza kutumika kutoa sehemu ya tuple. Kwa mfano, vitu `pie` na `cookies` vinaweza kuchukuliwa kuwa tuple tofauti:

```py
desserts = ('cake', 'pie', 'cookies', 'ice cream')
desserts[1:3] # ('pie', 'cookies')
```

- **Kuondoa vitu kutoka tuple**: Kuondoa kipengele kutoka tuple kutaongeza `TypeError` kwa kuwa tuple ni zisizobadilika:

```py
developer = ('Jane Doe', 23, 'Python Developer')
del developer[1]

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: "tuple" object doesn't support item deletion
"""
```

- **Lini utumie tuple badala ya orodha?**: Ikiwa unahitaji mkusanyiko unaobadilika wa vipengele ambapo unaweza kuongeza, kuondoa na kusahihisha vipengele, basi tumia orodha. Ikiwa unajua unafanya kazi na mkusanyiko thabiti na usiobadilika wa data, basi tumia tuple.

## Njia za kawaida za tuple

- **`count()`**: Inatumika kubaini ni mara ngapi kipengele kinaonekana katika tuple. Kwa mfano, unaweza kuangalia ni mara ngapi lugha ya `'Rust'` inaonekana katika tuple:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('Rust') # 2
```

- Ikiwa kipengele kilichotajwa katika njia ya `count()` hakipo kabisa katika tuple, basi thamani ya return itakuwa `0`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('JavaScript') # 0
```

- Ikiwa hakuna hoja zitakazopitishwa kwa njia ya `count()`, basi Python itarudisha `TypeError`.

- **index()**: Inatumika kupata kielezo ambacho kipengele fulani kiko katika tuple. Hapa kuna mfano wa kutumia njia ya `index()` kupata kielezo cha lugha ya `'Java'`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('Java') # 1
```

- Ikiwa kipengele kilichotajwa hakiwezi kupatikana, basi Python itarudisha hitilafu ya `ValueError`.

- Unaweza pia kupitisha kielezo cha kuanzia kwa njia ya `index()` ili kubainisha wapi kuanza kutafuta kipengele katika tuple:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
programming_languages.index('Python', 3) # 5
```

- Pia unaweza kupitisha kielezo cha mwisho kwa njia ya `index()` ili kubainisha wapi kuacha kutafuta kipengele katika tuple:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python', 'JavaScript', 'Python')
programming_languages.index('Python', 2, 5) # 2
```

- **`sorted()`**: Inatumika kupanga vipengele katika iterable yoyote na kurudisha orodha mpya iliyopangwa. Hapa kuna mfano wa kuunda orodha mpya ya nambari kwa kutumia kitendakazi cha `sorted()`:

```py
numbers = (13, 2, 78, 3, 45, 67, 18, 7)
sorted(numbers) # [2, 3, 7, 13, 18, 45, 67, 78]
```

- **Kubadilisha tabia ya kupanga**: Unaweza kubinafsisha tabia ya kupanga kwa iterable kwa kutumia hoja za hiari za `key` na `reverse`. Hapa kuna mfano wa kutumia hoja ya `key` kupanga vitu katika tuple kwa mujibu wa urefu:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
sorted(programming_languages, key=len)

# Result
# ['C++', 'Rust', 'Java', 'Rust', 'Python', 'Python']
```

- Unaweza pia kuunda orodha mpya ya thamani kwa mpangilio wa kinyume, ukitumia hoja ya `reverse` kama hii:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')

print(sorted(programming_languages, reverse=True))

# Result
# ['Rust', 'Rust', 'Python', 'Python', 'Java', 'C++']
```

## Mizunguko katika Python

- **Ufafanuzi**: Mizunguko hutumika kurudia sehemu ya msimbo kwa idadi fulani ya mara.

- **`for` loop**: Inatumika kurudia juu ya mfululizo (kama orodha, tuple au mfuatano wa herufi) na kutekeleza sehemu ya msimbo kwa kila kipengele katika mfululizo huo. Hapa kuna mfano wa kutumia `for` loop kurudia orodha na kuchapisha kila lugha kwenye konsoli:

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']

for language in programming_languages:
    print(language)

"""
Result 

Rust
Java
Python
C++
"""
```

- Hapa kuna mfano wa kutumia `for` loop kurudia mfuatano wa herufi `code` na kuchapisha kila herufi:

```py
for char in 'code':
    print(char)

"""
Result 

c
o
d
e
"""
```

- Mizunguko ya `for` inaweza kupangwa ndani ya mizunguko mingine. Hapa kuna mfano wa kutumia `for` loop iliyopangwa ndani:

```py
categories = ['Fruit', 'Vegetable']
foods = ['Apple', 'Carrot', 'Banana']

for category in categories:
    for food in foods:
        print(category, food)

"""
Result

Fruit Apple
Fruit Carrot
Fruit Banana
Vegetable Apple
Vegetable Carrot
Vegetable Banana
"""

```

- **`while` loop**: Inarudia sehemu ya msimbo hadi hali fulani iwe ya `False`. Hapa kuna mfano wa kutumia `while` loop kwa mchezo wa kubahatisha:

```py
secret_number = 3
guess = 0

while guess != secret_number:
    guess = int(input('Guess the number (1-5): '))
    if guess != secret_number:
        print('Wrong! Try again.')

print('You got it!')

"""
Result

Guess the number (1-5): 2
Wrong! Try again.
Guess the number (1-5): 1
Wrong! Try again.
Guess the number (1-5): 3
You got it!
"""

```

- **Kauli za `break` na `continue`**: Zinatumika katika mizunguko kubadilisha utekelezaji wa mzunguko.

- Kauli ya `break` hutumika kutoka mara moja kwenye mzunguko wakati hali fulani inatimia. Hapa kuna mfano wa kutumia kauli ya `break` kwa orodha ya `developer_names`:

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        break
    print(developer)
```

- Kauli ya `continue` hutumika kuruka kurudia kwa sasa na kuendelea na kurudia inayofuata ya mzunguko. Hapa kuna mfano wa kutumia kauli ya `continue` badala ya kauli ya `break`:

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        continue
    print(developer)
```

- Mizunguko ya `for` na `while` inaweza kuunganishwa na kifungu cha `else`, ambacho hutekelezwa tu wakati mzunguko haukuishwa kwa kauli ya `break`:

```python
words = ['sky', 'apple', 'rhythm', 'fly', 'orange']

for word in words:
    for letter in word:
        if letter.lower() in 'aeiou':
            print(f"'{word}' contains the vowel '{letter}'")
            break
    else:
        print(f"'{word}' has no vowels")
```

## Ranges na Matumizi Yake katika Mizunguko

- **Kitendakazi cha `range()`**: Kinatumika kuunda mfululizo wa nambari kamili.

```py
range(start, stop, step)
```

- Hoja muhimu ya `stop` ni nambari kamili (isiyo jumuishi) inayowakilisha mwisho wa mfululizo wa nambari unaozalishwa. Hapa kuna mfano wa kutumia kitendakazi cha `range()`:

```py
for num in range(3):
    print(num)
```

- Ikiwa hoja ya `start` haijatolewa, basi chaguo-msingi ni `0`. Kwa chaguo-msingi mfululizo wa nambari kamili utaongezeka kwa `1`. Unaweza kutumia hoja ya hiari ya `step` kubadilisha thamani ya ongezeko. Hapa kuna mfano wa kuunda mfululizo wa nambari kamili hata kuanzia 2 hadi 10 (si jumuishi 11):

```py
for num in range(2, 11, 2):
    print(num)
```

- Ikiwa hutoa hoja yoyote kwa kitendakazi cha `range()`, basi utapata `TypeError`.

- Kitendakazi cha `range()` kinakubali tu nambari kamili kwa hoja, si nambari za desimali. Kutumia nambari za desimali kutasababisha kosa la aina ya `TypeError`:

```py
ERROR!
Traceback (most recent call last):
  File "<main.py>", line 1, in <module>
TypeError: 'float' object cannot be interpreted as an integer
```

- Unaweza kutumia nambari kamili hasi kwa hoja ya `step` kuunda mfululizo wa nambari kamili unaopungua:

```py
for num in range(40, 0, -10):
    print(num)
```

- Kitendakazi cha `list` kinaweza pia kutumika kuunda orodha ya nambari kamili kwa kutumia `range()` kama kijenzi. Kijenzi cha `list` hutumika kubadilisha iterable kuwa orodha. Hapa kuna mfano wa kuunda orodha ya nambari kamili hata kati ya 2 na 10 jumuishi:

```py
numbers = list(range(2, 11, 2))
print(numbers) # [2, 4, 6, 8, 10]
```

## Kitendakazi cha `enumerate()` na `zip()` katika Python

- **`enumerate()`**: Inatumika kurudia juu ya mfululizo na kufuatilia kielezo cha kila kipengele katika mfululizo huo. Kitendakazi cha `enumerate()` kinachukua iterable kama hoja na kurudisha kitu cha `enumerate` kinachojumuisha kielezo na thamani ya kila kipengele katika iterable.

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for index, language in enumerate(languages):
    print(f'Index {index} and language {language}')

# Result
# Index 0 and language Spanish
# Index 1 and language English
# Index 2 and language Russian
# Index 3 and language Chinese

```

- Kitendakazi cha `enumerate()` kinaweza pia kutumika nje ya mzunguko wa `for`:

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

print(list(enumerate(languages)))
# [(0, 'Spanish'), (1, 'English'), (2, 'Russian'), (3, 'Chinese')]
```

- Kitendakazi cha `enumerate()` kinakubali pia hoja ya hiari ya `start` inayobainisha thamani ya kuanzia kwa kuhesabu. Ikiwa hoja hii haitolewi, kuhesabu huanza kwa `0`.

- **`zip()`**: Inatumika kurudia juu ya iterables nyingi sambamba. Hapa kuna mfano wa kutumia kitendakazi cha `zip()` kurudia juu ya orodha ya `developers` na `ids`:

```python
developers = ['Naomi', 'Dario', 'Jessica', 'Tom']
ids = [1, 2, 3, 4]

for name, dev_id in zip(developers, ids):
    print(f'Name: {name}')
    print(f'ID: {dev_id}')


"""
Result

Name: Naomi
ID: 1
Name: Dario
ID: 2
Name: Jessica
ID: 3
Name: Tom
ID: 4
"""

```

even_numbers = [num for num in range(21) if num % 2 == 0]

## Njia za vitu vinavyoweza kurudiwa

- **`filter()`**: Inatumika kuchuja vipengele kutoka iterable kulingana na hali fulani. Inarudisha iterator yenye vipengele tu vinavyokidhi hali. Hapa kuna mfano wa kuunda orodha mpya ya maneno marefu zaidi ya herufi nne:

```py
words = ['tree', 'sky', 'mountain', 'river', 'cloud', 'sun']

def is_long_word(word):
    return len(word) > 4

long_words = list(filter(is_long_word, words))
print(long_words) # ['mountain', 'river', 'cloud']
```

- **`map()`**: Inatumika kutumia kitendakazi kwa kila kipengele katika iterable na kurudisha iterable mpya yenye matokeo. Hapa kuna mfano wa kutumia kitendakazi cha `map()` kubadilisha orodha ya joto kutoka celsius hadi fahrenheit:

```py
celsius = [0, 10, 20, 30, 40]

def to_fahrenheit(temp):
    return (temp * 9/5) + 32

fahrenheit = list(map(to_fahrenheit, celsius))
print(fahrenheit) # [32.0, 50.0, 68.0, 86.0, 104.0]
```

- **`sum()`**: Inatumika kupata jumla kutoka iterable kama orodha au tuple. Hapa kuna mfano wa kutumia kitendakazi cha `sum()`:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers)
print(total) # Result: 50
```

- Unaweza pia kupitisha hoja ya hiari ya `start` ambayo inaweka thamani ya mwanzo ya jumla. Hapa kuna mfano uliosasishwa ukitumia hoja ya `start` kama hoja ya nafasi:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, 10) # positional argument
print(total) # 60
```

- Unaweza pia kutumia hoja ya `start` kama hoja ya neno kama hii badala yake:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, start=10) # keyword argument
print(total) # 60
```

## Kitendakazi cha lambda

- **Ufafanuzi**: Kitendakazi cha lambda katika Python ni njia fupi ya kuunda kitendakazi kisicho na jina (kitendakazi kisicho na jina). 
- Kitendakazi cha lambda mara nyingi hutumika kama hoja kwa kitendakazi kingine. Hapa kuna mfano wa kitendakazi cha lambda:

```py
numbers = [1, 2, 3, 4, 5]

even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(even_numbers)  # [2, 4]
```

- Mbinu bora za kutumia kitendakazi cha lambda ni kutoziweka kwenye kigezo, kuziweka rahisi na za kueleweka, na kuzitumia kwa vitendakazi vifupi, mara moja tu.
