# --explanation--

Un `code submission` è l'insieme di file di codice che un programmatore invia per la revisione. Per esempio:

- `Make sure your code submission follows the guidelines.` - Il codice deve rispettare le regole prima di inviarlo.

- `The professor reviewed my code submission and gave feedback.` - Il professore ha controllato il tuo codice e ha condiviso commenti.

Un `loop structure` esegue ripetutamente un blocco di codice finché non si soddisfa una certa condizione, risparmiando tempo automatizzando attività ripetitive, come i loop `for` o i loop `while`. Per esempio:

- `The loop structure in my code runs five times.` - Il tuo loop si ripete cinque volte nel programma.

- `A loop structure allows a program to repeat actions.` - Un loop aiuta un programma a fare la stessa cosa più volte.
