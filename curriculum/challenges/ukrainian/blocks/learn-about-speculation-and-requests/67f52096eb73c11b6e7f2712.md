### --feedback--

Ця двослівна фраза означає, що хтось фізично потрапляє до обмеженої області, а не зламує систему віддалено.

# --explanation--

`Physical access` означає отримання доступу до місця, наприклад, офісу або серверної кімнати.

`physical access breach` трапляється, коли хтось заходить у захищене приміщення без дозволу. Наприклад:

- `To avoid a physical access breach, we should require ID checks at the entrance.` — компанія може запобігти несанкціонованому доступу, перевіряючи ідентифікатори на вході.

- `A physical access breach can happen if someone enters the server room without permission.` — фізична проблема безпеки може виникнути, якщо хтось заходить у серверну кімнату без дозволу.
