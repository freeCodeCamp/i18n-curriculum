# --explanation--

`Want`は何かをしたいという願望を表します。過去形では、以前にその願望があったことを示すために`wanted`と言います。その後に動詞の不定詞（`to`付き）が続きます。例えば：

- `She wants to install the new software.` - 彼女は新しいソフトウェアをシステムに設定または追加したいと願っています。

- `They wanted to upgrade their system last week.` - 過去には、彼らはアップグレードするつもりでした。
