# --description--

正の整数 $k$ に対して、$d(k)$ を $k$ の通常の10進表記における桁の和と定義します。例えば $d(42) = 4 + 2 = 6$ です。

正の整数 $n$ に対して、$S(n)$ を次の条件を満たす $10^n$ 未満の正の整数 $k$ の個数と定義します：

- $k$ は23で割り切れること、
- $d(k) = 23$ であること。

$S(9) = 263\,626$ および $S(42) = 6\,377\,168\,878\,570\,056$ が与えられています。

$S(11^{12})$ を求め、その答えを $\bmod 10^9$ で答えてください。
