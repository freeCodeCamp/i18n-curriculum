# --description--

人は状況に応じてさまざまな方法でコミュニケーションを取ります。

`teléfono` を使って誰かと直接話すことができ、または電話で送られる短い書面のメッセージである `mensaje de texto` を使うこともできます。

さらに、`correo` は自宅の住所に送られる手紙のような伝統的な郵便を指します。今日では、多くのメッセージがオンラインで送られます。この場合は `correo electrónico` と呼ばれます。

`Electrónico` は技術や電子機器を使ったものを表します。
