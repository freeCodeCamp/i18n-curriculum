## Python ni nini?

- **Utangulizi**: Python ni lugha ya programu ya matumizi ya jumla inayojulikana kwa urahisi na unyumbufu wake. Python hutumika katika nyanja nyingi kama vile sayansi ya data na ujifunzaji wa mashine, ukuzaji wa mtandao, uandishi wa skripti na otomatiki, mifumo iliyojengwa ndani na IoT, na mengine mengi.
- **Matumizi ya Kawaida**: Python hutumika katika sayansi ya data, ujifunzaji wa mashine, ukuzaji wa mtandao, usalama wa mtandao, otomatiki na kompyuta ndogo kama Raspberry Pi na bodi zinazolingana na MicroPython.

## Python katika Mazingira Yako ya Ndani

- **Usakinishaji**: Njia bora ya kusakinisha Python kwenye Windows, Mac, na Linux ni kupakua kifurushi cha usakinishaji kutoka kwenye tovuti rasmi ya Python (`https://www.python.org/`).
- **Terminali**: Kiolesura cha maandishi kinachokuwezesha kuingiliana na kompyuta yako kwa kuandika amri. Kila mfumo wa uendeshaji huja na programu ya terminali ya msingi. Kwenye macOS, unaweza kutumia programu ya Terminal. Kwenye Windows, unaweza kutumia Command Prompt au PowerShell. Kwenye Linux, kila mazingira ya desktop yana programu yake ya terminali ya msingi kama GNOME Terminal au Konsole.
- **IDE**: IDE ni kifupi cha Integrated Development Environment na ina vipengele kama mhariri wa msimbo, zana za majaribio, terminali na zaidi.
- **Mhariri wa Msimbo na IDE Maarufu kwa Python**: VS Code, PyCharm, na Spyder
- **Kukimbia Msimbo Kwenye Kompyuta Yako**: Ili kukimbia msimbo wa Python, una chaguzi kadhaa. Chaguo la kwanza ni kubofya kitufe cha kuendesha kwenye VS Code ambacho kinaonekana kama kitufe cha kucheza kilicho upande wa juu kulia. Hilo litafungua terminali na kukimbia script yako ya Python hapo.

Chaguo jingine ni kufungua terminali na kukimbia programu kwa mkono. Hapa kuna mfano wa kukimbia programu inayoitwa `main.py`:

```bash
python main.py
```

Lazima ukimbie amri hii kutoka kwenye folda ambapo `main.py` imehifadhiwa. Kwa mfano, kama `main.py` iko ndani ya folda inayoitwa `python-projects`, kwanza tumia ifuatayo kwenye terminali:

```bash
cd python-projects
```

Kisha kimbia:

```bash
python main.py
```

## Kutumia Python Interactive Shell

- **Maelezo**: Interactive shell ni programu inayokuwezesha kuandika amri moja kwa moja na kuona matokeo. Fungua programu ya terminali na andika `python` kisha bonyeza `Enter`. Hii itaanzisha interactive shell ya Python. Alama ya `>>>` inaonyesha Python inasubiri uandike amri. Jaribu kuchapisha `"Hello, world!"` kwenye terminali:

```bash
print("Hello, world!")
```

Unapaswa kuona maandishi yakionekana kama haya:

```bash
>>> print("Hello, world!")
Hello, world!
```

Baada ya maandishi kuchapishwa, Python inarudi kwenye ifuatayo:

```bash
>>>
```

Hii ina maana unaweza kuandika amri nyingine.

- **Mzunguko wa Soma, Tathmini, Chapisha, Rudia (REPL)**: Kila unapoiandika amri, interpreter itasoma ingizo, itathmini, ichapishe matokeo na irudie kuonyesha `>>>` ili uendelee kuandika amri zaidi.

## Vigezo

- **Kutangaza Vigezo**: Ili kutangaza kigezo, anza na jina la kigezo likifuatiwa na opereta ya mgawo (`=`) kisha thamani. Hii inaweza kuwa nambari, mfuatano wa herufi, thamani ya kweli au siyo kweli, n.k. Hapa kuna mifano:

```py
name = 'John Doe'
age = 25
```

- **Mikataba ya Kuitwa kwa Vigezo**: Hapa ni mikataba unayopaswa kutumia kwa vigezo:

  - Majina ya vigezo yanaweza kuanza tu na herufi au mstari chini (_), si nambari.
  - Majina ya vigezo yanaweza kuwa na herufi na nambari (a-z, A-Z, 0-9) na mistari chini (_).
  - Majina ya vigezo yanazingatia herufi kubwa na ndogo — `age`, `Age`, na `AGE` yote yanachukuliwa kuwa tofauti.
  - Majina ya vigezo hayawezi kuwa maneno muhimu yaliyohifadhiwa ya Python kama `if`, `class`, au `def`.
  - Majina ya vigezo yenye maneno mengi hutenganishwa kwa mistari chini. Mfano, `snake_case`.

## Aina za Data za Kawaida katika Python

- **Utangulizi**: Python ni lugha inayobadilika aina za data kwa wakati wa utekelezaji kama JavaScript, maana huna haja ya kutangaza aina za data kwa vigezo. Lugha inajua aina ya kigezo kulingana na kile unachoweka.
- **Nambari Kamili**: Nambari isiyo na sehemu za desimali:

```py
my_integer_var = 10
print('Integer:', my_integer_var) # Integer: 10
```

- **Nambari ya Desimali (Float)**: Nambari yenye sehemu za desimali:

```py
my_float_var = 4.50
print('Float:', my_float_var) # Float: 4.5
```

- **Mfuatano wa Herufi (String)**: Mfuatano wa herufi uliowekwa ndani ya nukuu:

```py
my_string_var = 'hello'
print('String:', my_string_var) # String: hello
```

- **Thamani ya Kweli au Siyo Kweli (Boolean)**: Thamani inayowakilisha `True` au `False`:

```py
my_boolean_var = True
print('Boolean:', my_boolean_var) # Boolean: True
```

- **Seti**: Mkusanyiko usio na mpangilio wa vipengele vya kipekee:

```py
my_set_var = {7, 5, 8}
print('Set:', my_set_var) # Set: {7, 5, 8}
```

- **Kamusi (Dictionary)**: Mkusanyiko wa jozi za ufunguo-thamani, zikiwa ndani ya mabano ya mviringo:

```py
my_dictionary_var = {"name": "Alice", "age": 25}
print('Dictionary:', my_dictionary_var) # Dictionary: {'name': 'Alice', 'age': 25}
```

- **Tuple**: Mkusanyiko wa vitu uliopangwa na usiobadilika, uliowekwa ndani ya mabano ya kawaida:

```py
my_tuple_var = (7, 5, 8)
print('Tuple:', my_tuple_var) # Tuple: (7, 5, 8)
```

- **Range**: Mfululizo wa nambari, mara nyingi hutumika katika mizunguko:

```py
my_range_var = range(5)
print(my_range_var) # range(0, 5)
```

- **Orodha (List)**: Mkusanyiko uliopangwa wa vipengele unaounga mkono aina tofauti za data:

```py
my_list = [22, 'Hello world', 3.14, True]
print(my_list) # [22, 'Hello world', 3.14, True]
```

- **None**: Thamani maalum inayowakilisha kutokuwepo kwa thamani:

```py
my_none_var = None
print('None:', my_none_var) # None: None
```

## Aina zisizobadilika na zinazobadilika

- **Aina zisizobadilika**: Aina hizi haziwezi kubadilika baada ya kutangazwa, ingawa unaweza kuweka vigezo vyake kuashiria kitu kipya, kinachoitwa mgawo upya. Zinajumuisha nambari kamili, nambari ya desimali, nambari tata, thamani ya kweli au siyo kweli, mfuatano wa herufi, tuple, range, na `None`.
- **Aina zinazobadilika**: Aina hizi zinaweza kubadilika baada ya kutangazwa. Unaweza kuongeza, kuondoa, au kusahihisha vitu vyake. Zinajumuisha aina za mkusanyiko kama orodha, seti, na kamusi.
- **Kitendakazi cha `type()`**: Ili kuona aina ya kigezo, unaweza kutumia kitendakazi cha `type()` kama ifuatavyo:

```py
greeting = 'Hello there!'
age = 21

print(type(greeting)) # <class 'str'>
print(type(age)) # <class 'int'>
```

- **Kitendakazi cha `isinstance()`**: Hiki hutumika kuangalia kama kigezo kinaendana na aina fulani ya data:

```py
greeting = 'Hello world'
name = 'John Doe'

print(isinstance(greeting, str)) # True
print(isinstance(name, int)) # False
```

## Kufanya kazi na mfuatano wa herufi

- **Maelezo**: Kama unavyokumbuka kutoka JavaScript, mfuatano wa herufi ni usiobadilika, maana huwezi kubadilisha baada ya kuundwa. Katika Python, unaweza kutumia nukuu moja au maradufu. Inashauriwa kuchagua kanuni moja na kuifuata:

```py
developer = 'Jessica'
city = "Los Angeles"
```

- **Kupata herufi kutoka kwenye mfuatano wa herufi**: Unaweza kupata herufi kutoka kwenye mfuatano kwa kutumia uandishi wa mabano kama ifuatavyo:

```py
my_str = 'Hello world'

print(my_str[0])  # H
print(my_str[6])  # w

print(my_str[-1])  # d
print(my_str[-2]) # l
```

- **Kuepuka herufi katika mfuatano**: Unaweza kutumia mkwaju wa nyuma (`\`) ikiwa mfuatano wako una nukuu kama ifuatavyo:

```py
msg = 'It\'s a sunny day'
quote = "She said, \"Hello!\""
```

- **Kuunganisha mfuatano wa herufi**: Ili kuunganisha mfuatano wa herufi, unaweza kutumia opereta ya `+` kama ifuatavyo:

```py
developer = 'Jessica'
print('My name is ' + developer + '.') # My name is Jessica.
```

Njia nyingine ya kuunganisha mfuatano ni kutumia opereta ya `+=`. Hii hutumika kufanya kuunganisha na mgawo katika hatua moja kama ifuatavyo:

```py
greeting = 'My name is '
developer = 'Jessica.'

greeting += developer
print(greeting) # My name is Jessica.
```

- **`f-strings`**: Hii ni kifupi cha formatted string literals. Inakuwezesha kufanya uingizaji wa thamani na pia kuunganisha kwa sintaksia fupi na inayosomeka:

```py
developer = 'Jessica'
greeting = f'My name is {developer}.'
print(greeting) # My name is Jessica.
```

- **Kuchukua sehemu ya mfuatano wa herufi (String slicing)**: Hii ni pale unapotaka kutoa sehemu ya mfuatano. Hapa ni sintaksia ya msingi:

```py
str[start:stop:step]
```

Msimamo wa kuanzia unaonyesha kielezo ambacho kuchukua sehemu kuanzia. Msimamo wa kusitisha ni mahali sehemu inapaswa kuisha, msimamo huu haujumuishi. Msimamo wa hatua unaonyesha kiwango cha kuongezeka kwa kuchukua sehemu. Hapa kuna mifano:

```py
message = 'Python is fun!'

print(message[0:6])  # Python
print(message[7:])  # is fun!
print(message[::2])  # Pto sfn
```

- **Kupata urefu wa mfuatano wa herufi**: Kitendakazi cha `len()` hutumika kurudisha idadi ya herufi katika mfuatano:

```py
developer = 'Jessica'

print(len(developer)) # 7
```

## Kufanya kazi na opereta ya `in`

- **Opereta ya `in`**: Hii hurudisha thamani ya kweli au siyo kweli inayoonyesha kama herufi au herufi zipo ndani ya mfuatano au la:

```py
my_str = 'Hello world'

print('Hello' in my_str)  # True
print('hey' in my_str)    # False
print('hi' in my_str)    # False
print('e' in my_str)  # True
print('f' in my_str)  # False
```

## Njia za kawaida za mfuatano wa herufi

- **`upper()`**: Njia hii ya mfuatano hurudisha mfuatano mpya wenye herufi zote kuwa kubwa:

```py
developer = 'Jessica'

print(developer.upper()) # JESSICA
```

- **`lower()`**: Njia hii ya mfuatano hurudisha mfuatano mpya wenye herufi zote kuwa ndogo:

```py
developer = 'Jessica'

print(developer.lower()) # jessica
```

- **`strip()`**: Njia hii hurudisha nakala ya mfuatano na kuondoa herufi za mwanzo na mwisho zilizobainishwa (kama hakuna hoja, huondoa nafasi za mwanzo na mwisho).

```py
greeting = '  hello world  '

trimmed_my_str = greeting.strip()
print(trimmed_my_str)  # 'hello world'
```

- **`replace()`**: Hii hurudisha mfuatano mpya ambapo matukio yote ya mfuatano wa zamani yamebadilishwa na mfuatano mpya.

```py
greeting = 'hello world'

replaced_my_str = greeting.replace('hello', 'hi')
print(replaced_my_str)  # 'hi world'
```

- **`split()`**: Hii hutumika kugawanya mfuatano kuwa orodha kwa kutumia kitenzi cha kugawanya. Kitenzi cha kugawanya ni mfuatano unaoelezea mahali pa kugawanya.

```py
dashed_name = 'example-dashed-name'

split_words = dashed_name.split('-')
print(split_words)  # ['example', 'dashed', 'name']
```

- **`join()`**: Hii hutumika kuunganisha vipengele vya iterable kuwa mfuatano wa herufi kwa separator. Iterable ni mkusanyiko wa vipengele vinavyoweza kurudiwa kama orodha, mfuatano wa herufi au tuple.

```py
example_list = ['example', 'dashed', 'name']

joined_str = ' '.join(example_list)
print(joined_str)  # example dashed name
```

- **`startswith(prefix)`**: Njia hii ya mfuatano hurudisha thamani ya kweli au siyo kweli ikiwa mfuatano unaanza na kiambishi awali kilichobainishwa:

```py
developer = 'Naomi'

result = developer.startswith('N')
print(result)  # True
```

- **`endswith(suffix)`**: Njia hii ya mfuatano hurudisha thamani ya kweli au siyo kweli ikiwa mfuatano unaisha na kiambishi cha mwisho kilichobainishwa:

```py
developer = 'Naomi'

result = developer.endswith('N')
print(result)  # False
```

- **`find()`**: Njia hii ya mfuatano hurudisha kielezo cha tukio la kwanza la sehemu ndogo ya mfuatano. Ikiwa haipatikani, hurudisha `-1`:

```py
developer = 'Naomi'

result = developer.find('N')
print(result)  # 0

city = 'Los Angeles'
print(city.find('New')) # -1
```

- **`count(substring)`**: Njia hii ya mfuatano huhesabu mara ngapi sehemu ndogo ya mfuatano inaonekana:

```py
city = 'Los Angeles'
print(city.count('e')) # 2
```

- **`capitalize()`**: Njia hii ya mfuatano hurudisha mfuatano mpya wenye herufi ya kwanza kuwa kubwa na nyingine ndogo:

```py
dessert = 'chocolate cake'
print(dessert.capitalize()) # Chocolate cake
```

- **`isupper()`**: Njia hii ya mfuatano hurudisha `True` ikiwa herufi zote ni kubwa na `False` vinginevyo:

```py
dessert = 'chocolate cake'
print(dessert.isupper()) # False
```

- **`islower()`**: Njia hii ya mfuatano hurudisha `True` ikiwa herufi zote ni ndogo na `False` vinginevyo:

```py
dessert = 'chocolate cake'
print(dessert.islower()) # True
```

- **`title()`**: Njia hii ya mfuatano hurudisha mfuatano mpya wenye herufi ya kwanza ya kila neno kuwa kubwa:

```py
city = 'los angeles'
print(city.title()) # Los Angeles
```

- **`maketrans()`**: Njia hii ya mfuatano hutumika kuunda jedwali la ramani ya herufi 1 kwa 1 kwa ajili ya tafsiri. Mara nyingi hutumika pamoja na njia ya `translate()` ambayo hutumia jedwali hilo kwenye mfuatano na kurudisha matokeo yaliyotafsiriwa.

```py
trans_table = str.maketrans('abc', '123')
print(trans_table) # {97: 49, 98: 50, 99: 51}

result = 'abcabc'.translate(trans_table)
print(result)  # 123123
```

## Operesheni za kawaida zinazotumika na nambari kamili na nambari za desimali

- **Operesheni za Msingi za Hisabati**: Katika Python, unaweza kufanya operesheni za msingi za hisabati na nambari kamili na nambari za desimali ikiwa ni pamoja na kuongeza, kutoa, kuzidisha na kugawanya:

```py
int_1 = 56
int_2 = 12
float_1 = 5.4
float_2 = 12.0

# Addition

print('Integer Addition:', int_1 + int_2) # Integer Addition: 68
print('Float Addition:', float_1 + float_2) # Float Addition: 17.4

# Subtraction

print('Int Subtraction:', int_1 - int_2) # Int Subtraction: 44
print('Float Subtraction:',  float_2 - float_1) # Float Subtraction: 6.6

# Multiplication

print('Int Multiplication:', int_1 * int_2) # Int Multiplication: 672
print('Float Multiplication:', float_2 * float_1) # Float Multiplication: 64.80000000000001

# Division

print('Division:', int_1 / int_2) # Division: 4.666666666666667
print('Float Division:', float_2 / float_1) # Float Division: 2.222222222222222
```

Unapoongeza nambari ya desimali na nambari kamili, matokeo yatageuzwa kuwa nambari ya desimali kama ifuatavyo:

```py
int_1 = 56
float_1 = 5.4

print(int_1 + float_1) # 61.4
```

- **Opereta wa Modulo (`%`)**: Huu hurudisha mabaki wakati nambari ikigawanywa na nambari nyingine:

```py
int_1 = 56
int_2 = 12

print(int_1 % int_2) # 8
```

- **Mgawanyo wa Floor (`//`)**: Opereta hii hutumika kugawanya nambari mbili na kuzungusha matokeo hadi nambari kamili iliyo karibu zaidi:

```py
int_1 = 56
int_2 = 12

print(int_1 // int_2) # 4
```

- **Opereta wa Nguvu (`**`)**: Opereta hii hutumika kuinua nambari hadi nguvu ya nyingine:

```py
int_1 = 4
int_2 = 2

print(int_1 ** int_2) # 16
```

- **Kitendakazi cha `float()`**: Unaweza kutumia kitendakazi hiki kubadilisha nambari kamili kuwa nambari ya desimali.

```py
num = 4

print(float(num)) # 4.0
```

- **Kitendakazi cha `int()`**: Unaweza kutumia kitendakazi hiki kubadilisha nambari ya desimali kuwa nambari kamili.

```py
num = 4.0

print(int(num)) # 4
```

- **Kitendakazi cha `round()`**: Hiki hutumika kuzungusha nambari hadi nambari kamili iliyo karibu zaidi:

```py
num_1 = 3.4
num_2 = 7.7

print(round(num_1)) # 3
print(round(num_2)) # 8
```

- **Kitendakazi cha `abs()`**: Hiki hutumika kurudisha thamani kamili ya nambari:

```py
num = -13

print(abs(num)) # 13
```

- **Kitendakazi cha `pow()`**: Hiki hutumika kuinua nambari hadi nguvu ya nyingine:

```py
result = pow(2, 3) 
print(result)  # 8
```

## Mgawo ulioboreshwa

- **Maelezo**: Mgawo ulioboreshwa huunganisha operesheni ya binary na mgawo katika hatua moja. Huchukua kigezo, hutumia operesheni na thamani nyingine, na kuhifadhi matokeo tena kwenye kigezo hicho.

```py
# Addition assignment 
my_var = 10
my_var += 5

print(my_var) # 15

# Subtraction assignment
count = 14
count -= 3

print(count) # 11

# Multiplication assignment 
product = 65
product *= 7

print(product) # 455

# Division assignment 
price = 100
price /= 4

print(price) # 25.0

# Floor Division assignment 
total_pages = 23
total_pages //= 5

print(total_pages) # 4

# Modulo assignment 
bits = 35
bits %= 2

print(bits) # 1

# Exponentiation assignment 
power = 2
power **= 3

print(power) # 8
```

Kuna opereta nyingine za mgawo ulioboreshwa pia, kama zile za opereta za bitwise. Zinajumuisha `&=`, `^=`, `>>=`, na `<<=`.

## Kufanya kazi na kitendakazi

- **Maelezo**: Kitendakazi ni vipande vya msimbo vinavyoweza kutumika tena vinavyopokea ingizo (hoja) na kurudisha matokeo. Ili kuita kitendakazi, unahitaji kurejea jina la kitendakazi likifuatiwa na mabano ya kawaida:

```py
# Defining a function

def get_sum(num_1, num_2):
    return num_1 + num_2

result = get_sum(3, 4) # function call
print(result) # 7
```

Kama kitendakazi hakirudishi thamani waziwazi, thamani ya default ya return ni `None`:

```py
def greet():
    print('hello') 

result = greet() # hello
print(result) # None
```

Unaweza pia kutoa thamani za default kwa vigezo kama ifuatavyo:

```py
def get_sum(num_1, num_2=2):
    return num_1 + num_2

result = get_sum(3) 
print(result) # 5
```

Kama utaita kitendakazi bila idadi sahihi ya hoja, utapokea `TypeError`:

```py
def calculate_sum(a, b):
    print(a + b)

calculate_sum()

# TypeError: calculate_sum() missing 2 required positional arguments: 'a' and 'b'
```

## Wigo katika Python

- **Wigo wa ndani**: Hii ni pale kigezo kinapotangazwa ndani ya kitendakazi au darasa na kinaweza kupatikana tu ndani ya kitendakazi au darasa hilo.

```py
def my_func():
    num = 10
    print(num)
```

- **Wigo wa kuzunguka**: Hii ni pale kitendakazi kilichopangwa ndani ya kitendakazi kingine kinapoweza kupata vigezo vya kitendakazi cha juu.

```py
def outer_func():
    msg = 'Hello there!'
    
    def inner_func():
        print(msg)
    inner_func()

print(outer_func()) # Hello there!
```

- **Wigo wa ulimwengu**: Hii inahusu vigezo vilivyotangazwa nje ya vitendakazi au madarasa ambavyo vinaweza kupatikana kutoka sehemu yoyote ya programu.

```py
tax = 0.70 

def get_total(subtotal):
    total = subtotal + (subtotal * tax)
    return total

print(get_total(100))  # 170.0
```

- **Wigo wa ndani uliyojengwa**: Majina yaliyohifadhiwa katika Python kwa ajili ya vitendakazi vilivyotangazwa awali, moduli, maneno muhimu, na vitu.

```py
print(str(45)) # '45'
print(type(3.14)) # <class 'float'>
print(isinstance(3, str)) # False
```

## Opereta za kulinganisha

- **Sawa (`==`)**: Huu huangalia kama thamani mbili ni sawa:

```py
print(3 == 4) # False
```

- **Sio sawa (`!=`)**: Huu huangalia kama thamani mbili si sawa:

```py
print(3 != 4) # True
```

- **Kubwa zaidi kabisa (`>`)**: Huu huangalia kama thamani moja ni kubwa zaidi kuliko nyingine:

```py
print(3 > 4) # False
```

- **Ndogo zaidi kabisa (`<`)**: Huu huangalia kama thamani moja ni ndogo zaidi kuliko nyingine:

```py
print(3 < 4) # True
```

- **Kubwa au sawa (`>=`)**: Huu huangalia kama thamani moja ni kubwa au sawa na nyingine:

```py
print(3 >= 4) # False
```

- **Ndogo au sawa (`<=`)**: Huu huangalia kama thamani moja ni ndogo au sawa na nyingine:

```py
print(3 <= 4) # True
```

## Kufanya kazi na `if`, `elif` na `else` Statements

- **`if` Statements**: Hizi ni masharti yanayotumika kuamua kama kitu ni kweli au si kweli. Ikiwa sharti linathibitishwa kuwa `True`, basi sehemu hiyo ya msimbo itaendeshwa.

```py
age = 18

if age >= 18:
    print('You are an adult') # You are an adult
```

- **`elif` Clause**: Hizi ni masharti yanayofuata taarifa ya `if`. Ikiwa sharti la `elif` linathibitishwa kuwa `True`, basi sehemu hiyo ya msimbo itaendeshwa.

```py
age = 16

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')  # You are a teenager
```

- **`else` Clause**: Hii itaendeshwa ikiwa masharti mengine hayathibitishi kuwa `True`.

```py
age = 12

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')
else:
    print('You are a child')  # You are a child
```

Unaweza pia kutumia taarifa za `if` zilizo pangwa ndani kama ifuatavyo:

```py
is_citizen = True
age = 25

if is_citizen:
    if age >= 18:
        print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

## Thamani za kweli na uongo

- **Maelezo**: Katika Python, kila thamani ina thamani ya kweli au siyo kweli ya asili, au hisia ya ndani ya kama inapaswa kutendewa kama `True` au `False` katika muktadha wa mantiki. Thamani nyingi zinachukuliwa kuwa za kweli, yaani, zinathibitishwa kuwa `True` katika muktadha wa mantiki. Nyingine ni za uongo, zinazoathibitishwa kuwa `False`. Hapa kuna mifano ya thamani za uongo:

```md
None
False
Integer 0
Float 0.0
Empty strings ''
```

Thamani nyingine kama nambari zisizo za sufuri, na mfuatano wa herufi usio tupu ni za kweli.

## Kufanya kazi na kitendakazi cha `bool()`

- **Maelezo**: Ikiwa unataka kuangalia kama thamani ni ya kweli au uongo, unaweza kutumia kitendakazi cha ndani cha `bool()`. Hiki hubadilisha thamani kuwa sawa na thamani ya kweli au siyo kweli na hurudisha `True` kwa thamani za kweli na `False` kwa thamani za uongo. Hapa kuna mifano michache:

```py
print(bool(False)) # False
print(bool(0))  # False
print(bool('')) # False

print(bool(True)) # True
print(bool(1)) # True
print(bool('Hello')) # True
```

## Opereta za thamani ya kweli na uongo na kupunguza mizunguko

- **Maelezo**: Hizi ni opereta maalum zinazokuwezesha kuunganisha usemi kadhaa kuunda mantiki tata zaidi ya maamuzi katika msimbo wako. Kuna opereta tatu za thamani ya kweli na uongo katika Python: `and`, `or`, na `not`.
- **Opereta ya `and`**: Opereta hii huchukua operandi mbili na hurudisha operandi ya kwanza ikiwa ni ya uongo, vinginevyo hurudisha operandi ya pili. Operandi zote mbili lazima ziwe za kweli ili usemi uzalishwe thamani ya kweli.

```py
is_citizen = True
age = 25

print(is_citizen and age) # 25
```

Unaweza pia kutumia opereta ya `and` katika masharti kama ifuatavyo:

```py
is_citizen = True
age = 25

if is_citizen and age >= 18:
    print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

- **Opereta ya `or`**: Opereta hii hurudisha operandi ya kwanza ikiwa ni ya kweli, vinginevyo hurudisha operandi ya pili. Usemi wa `or` hutokea kuwa na thamani ya kweli ikiwa angalau operandi moja ni ya kweli. Hapa kuna mfano:

```py
age = 19
is_employed = False

print(age or is_employed) # 19
```

Kama ilivyo kwa opereta ya `and`, unaweza kutumia opereta ya `or` katika masharti kama ifuatavyo:

```py
age = 19
is_student = True

if age < 18 or is_student:
    print('You are eligible for a student discount') # You are eligible for a student discount
else:
    print('You are not eligible for a student discount')
```

- **Kupunguza mizunguko**: Opereta za `and` na `or` zinajulikana kama opereta za kupunguza mizunguko. Kupunguza mizunguko maana yake ni Python huchunguza thamani kutoka kushoto kwenda kulia na kusitisha mara tu inapotambua matokeo ya mwisho.
- **Opereta ya `not`**: Opereta hii huchukua operandi moja na kubadilisha thamani yake ya kweli au siyo kweli. Hubadilisha thamani za kweli kuwa `False` na thamani za uongo kuwa `True`. Tofauti na opereta zilizopita, `not` daima hurudisha `True` au `False`. Hapa kuna mifano:

```py
print(not '') # True, because empty string is falsy
print(not 'Hello') # False, because non-empty string is truthy
print(not 0) # True, because 0 is falsy
print(not 1) # False, because 1 is truthy
print(not False) # True, because False is falsy
print(not True) # False, because True is truthy
```

Hapa kuna mfano wa opereta ya `not` katika sharti:

```py
is_admin = False

if not is_admin:
    print('Access denied for non-administrators.') # Access denied for non-administrators.
else:
    print('Welcome, Administrator!')
```

- **Kuunda orodha kwa kutumia kijenzi cha `list()`**: Orodha pia zinaweza kuundwa kwa kutumia kijenzi cha `list()`. Kijenzi cha `list()` hutumika kubadilisha iterable kuwa orodha:

- **Ubadilishaji wa orodha**: Orodha ni zinazobadilika, maana unaweza kusahihisha kipengele chochote ndani ya orodha mradi tu uweke nambari halali ya kielezo. Ili kusahihisha orodha kwa kielezo fulani, unaweza kuweka thamani mpya kwa kielezo hicho:

```python
programming_languages = ['Python', 'Java', 'C++', 'Rust']
```

- **Kukagua kama kipengele kiko kwenye orodha**: Neno kuu la `in` linaweza kutumika kuangalia kama kipengele kiko kwenye orodha:

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
```


- **Kutoa thamani kutoka orodha (Unpacking)**: Kutoa thamani kutoka orodha ni mbinu inayotumika kuweka thamani kutoka orodha kwenye vigezo vipya. Hapa kuna mfano wa kutoa orodha ya `developer` kwenye vigezo vipya vinavyoitwa `name`, `age` na `job` kama ifuatavyo:

- **Muda wa hatua**: Pia inawezekana kubainisha muda wa hatua unaoamua kwa kiasi gani kuongezeka kati ya vigezo. Hapa kuna mfano ikiwa unataka kutoa orodha ya nambari za hata kwa kutumia kuchukua sehemu:

- **Kauli za `break` na `continue`**: Zinatumika katika mizunguko kubadilisha utekelezaji wa mzunguko.

- Kauli ya `break` hutumika kutoka mara moja kwenye mzunguko wakati hali fulani inatimia. Hapa kuna mfano wa kutumia kauli ya `break` kwa orodha ya `developer_names`:

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        break
    print(developer)
```

- Kauli ya `continue` hutumika kuruka kurudia kwa sasa na kuendelea na kurudia inayofuata ya mzunguko. Hapa kuna mfano wa kutumia kauli ya `continue` badala ya kauli ya `break`:

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        continue
    print(developer)
```

- Mizunguko ya `for` na `while` inaweza kuunganishwa na kifungu cha `else`, ambacho hutekelezwa tu wakati mzunguko haukuishwa kwa kauli ya `break`:

```python
words = ['sky', 'apple', 'rhythm', 'fly', 'orange']

for word in words:
    for letter in word:

    else:
```

```py
for num in range(3):
```

```py
for num in range(2, 11, 2):
```

## Kitendakazi cha iterable

- **`filter()`**: Inatumika kuchuja vipengele kutoka iterable kulingana na hali fulani. Inarudisha iterator yenye vipengele tu vinavyokidhi hali. Hapa kuna mfano wa kuunda orodha mpya ya maneno marefu zaidi ya herufi nne:

```py
words = ['tree', 'sky', 'mountain', 'river', 'cloud', 'sun']

def is_long_word(word):
    return len(word) > 4

long_words = list(filter(is_long_word, words))
print(long_words) # ['mountain', 'river', 'cloud']
```

- **`map()`**: Inatumika kutumia kitendakazi kwa kila kipengele katika iterable na kurudisha iterable mpya yenye matokeo. Hapa kuna mfano wa kutumia kitendakazi cha `map()` kubadilisha orodha ya joto kutoka celsius hadi fahrenheit:

```py
celsius = [0, 10, 20, 30, 40]

def to_fahrenheit(temp):
    return (temp * 9/5) + 32

fahrenheit = list(map(to_fahrenheit, celsius))
print(fahrenheit) # [32.0, 50.0, 68.0, 86.0, 104.0]
```

- **`sum()`**: Inatumika kupata jumla kutoka iterable kama orodha au tuple. Hapa kuna mfano wa kutumia kitendakazi cha `sum()`:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers)
print(total) # Result: 50
```

- Unaweza pia kupitisha hoja ya hiari ya `start` ambayo inaweka thamani ya mwanzo ya jumla. Hapa kuna mfano uliosasishwa ukitumia hoja ya `start` kama hoja ya nafasi:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, 10) # positional argument
print(total) # 60
```

- Unaweza pia kutumia hoja ya `start` kama hoja ya neno kama hii badala yake:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, start=10) # keyword argument
print(total) # 60
```

## Kamusi

- **Kamusi**: Kamusi ni muundo wa data uliyojengwa ndani unaohifadhi makundi ya jozi za ufunguo-thamani. Funguo zinapaswa kuwa aina za data zisizobadilika. Huu ni sintaksia ya jumla ya kamusi ya Python:

```python
dictionary = {
```

```python
pizza = dict([('name', 'Margherita Pizza'), ('price', 8.9), ('calories_per_slice', 250), ('toppings', ['mozzarella', 'basil'])])
```

```python
dictionary.get(key, default)
```

- **`keys()` na `values()`**: Njia za `keys()` na `values()` hurudisha kitu cha mtazamo chenye funguo zote na thamani zote katika kamusi, mtawalia. Kitu cha mtazamo ni njia ya kuona yaliyomo katika kamusi bila kuunda nakala tofauti ya data.

```python
pizza = {
```

```python
pizza.items()
# dict_items([('name', 'Margherita Pizza'), ('price', 8.9), ('calories_per_slice', 250)])
```

- **`clear()`**: Njia ya `clear()` huondoa jozi zote za ufunguo-thamani kutoka kamusi.

```python
pizza.clear()
```

- **`pop()`**: Njia ya `pop()` huondoa jozi ya ufunguo-thamani yenye ufunguo uliotajwa kama hoja ya kwanza na kurudisha thamani yake. pizza.pop('price', 10)

- **`update()`**: Njia ya `update()` husahihisha jozi za ufunguo-thamani kwa jozi za ufunguo-thamani za kamusi nyingine. Ikiwa zinafunguo sawa, thamani zao hubadilishwa. Funguo mpya zitaongezwa kama jozi mpya za ufunguo-thamani.

```python
pizza.update({ 'price': 15, 'total_time': 25 })
```

```python
products = {
    'Laptop': 990,
```

- **Kurudia juu ya funguo**: Ikiwa unahitaji kurudia juu ya funguo katika kamusi ya `products` hapo juu, unaweza kuandika `products.keys()` au `products` moja kwa moja.

```python
for product in products.keys():
```

Kuweka kielezo na kipengele katika vigezo tofauti vya mzunguko ni njia ya kawaida ya kutumia `enumerate()`. Kwa mfano, kwa `products.items()`, unaweza kupata jozi nzima ya thamani ya ufunguo pamoja na faharasa:

```python
for index, product in enumerate(products.items(), 1):
```

## Sets

- **Sets**: Sets ni muundo wa data uliyojengwa ndani katika Python ambao haukuruhusu thamani rudufu. Sets ni zinazobadilika na hazina mpangilio, maana vipengele vyake havihifadhiwi kwa mpangilio maalum, hivyo huwezi kutumia viashiria au funguo kuzipata. Pia, sets zinaweza kuwa na thamani za aina zisizobadilika tu, kama nambari, mfuatano wa herufi, na tuple.

- **Kufafanua set**: Ili kufafanua set, unahitaji kuandika vipengele vyake ndani ya mabano ya mviringo na kuvitenganisha kwa koma.

```python
my_set = {1, 2, 3, 4, 5}
```

- **Kufafanua set tupu**: Ikiwa unahitaji kufafanua set tupu, lazima utumie kitendakazi cha `set()`. Kuandika mabano ya mviringo tupu tu kutaunda kamusi moja kwa moja.

```python
set() # Set
{}    # Dictionary
```

## Njia za kawaida za set

- **`add()`**: Unaweza kuongeza kipengele kwenye set kwa njia ya `add()`, ukipitisha kipengele kipya kama hoja.

```python
my_set.add(6)
```

- **`remove()` na `discard()`**: Kuondoa kipengele kutoka set, unaweza kutumia njia ya `remove()` au `discard()`, ukipitisha kipengele unachotaka kuondoa kama hoja. Njia ya `remove()` itatupa hitilafu ya `KeyError` ikiwa kipengele hakipatikani, wakati `discard()` haitatupa hitilafu.

```python
my_set.remove(4)
my_set.discard(4)
```

- **`clear()`**: Njia ya `clear()` huondoa vipengele vyote kutoka set.

```python
my_set.clear()
```

## Operesheni za kihesabu za set

- **`issubset()` na `issuperset()`**: Njia za `issubset()` na `issuperset()` zinakagua kama set ni seti ndogo au seti kubwa ya set nyingine, mtawalia.

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 5}

print(your_set.issubset(my_set)) # True
print(my_set.issuperset(your_set)) # True
```

- **`isdisjoint()`**: Njia ya `isdisjoint()` inakagua kama sets mbili hazina vipengele vinavyofanana.

```python
my_set = {1, 2, 3}
your_set = {4, 5, 6}

print(my_set.isdisjoint(your_set)) # True
```

- **Operator ya muungano (`|`)**: Operator ya muungano `|` inarudisha set mpya yenye vipengele vyote kutoka sets zote mbili.

```python
my_set = {1, 2, 3}
your_set = {4, 5, 6}

my_set | your_set # {1, 2, 3, 4, 5, 6}
```

- **Operator ya mkusanyiko (intersection) (`&`)**: Operator ya mkusanyiko `&` inarudisha set mpya yenye vipengele vinavyopatikana katika sets zote mbili.

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

my_set & your_set # {2, 3, 4}
```

- **Operator ya tofauti (difference) (`-`)**: Operator ya tofauti `-` inarudisha set mpya yenye vipengele za seti ya kwanza ambazo hazipo katika seti ya pili.

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

my_set - your_set # {1, 5}
```

- **Operator ya tofauti ya usawa (symmetric difference) (`^`)**: Operator ya tofauti ya usawa `^` inarudisha set mpya yenye vipengele ambazo ziko katika seti ya kwanza au ya pili, lakini si zote mbili.

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

my_set ^ your_set # {1, 5, 6}
```

- **Operator ya kuangalia uwepo (`in`)**: Unaweza kuangalia kama kipengele kiko katika set au la kwa operator ya `in`.

```python
print(5 in my_set)
```

## Maktaba ya kawaida ya Python

- **Maktaba ya kawaida ya Python**: Maktaba inakupa msimbo uliandikwa tayari na unaoweza kutumika tena, kama vitendakazi, madarasa, na miundo ya data, ambayo unaweza kutumia katika miradi yako. Python ina maktaba kubwa ya kawaida yenye moduli zilizojengwa ndani zinazotekeleza suluhisho za viwango kwa matatizo na kazi nyingi. Baadhi ya mifano ya moduli maarufu zilizojengwa ndani ni `math`, `random`, `re` (mfupi wa "regular expressions"), na `datetime`.

## Kauli ya import

- **Kauli ya import**: Ili kupata vipengele vilivyotajwa katika moduli zilizojengwa ndani, unatumia kauli ya import. Kauli za import kwa kawaida huandikwa juu ya faili. Kauli za import hufanya kazi sawa kwa vitendakazi, madarasa, constants, vigezo, na vipengele vingine vyote vilivyotajwa katika moduli.

- **Kauli ya msingi ya import**: Unaweza kutumia neno kuu `import` ikifuatiwa na jina la moduli:

```python
import module_name
```

Kisha, ikiwa unahitaji kuita kitendakazi kutoka moduli hiyo, utatumia uandishi wa nukta, na jina la moduli ikifuatiwa na jina la kitendakazi.

```python
module_name.function_name()
```

Kwa mfano, utaandika ifuatayo katika msimbo wako kuleta moduli ya `math` na kupata mzizi wa mraba wa 36:

```python
import math

math.sqrt(36)
```

- **Kuleta moduli kwa jina tofauti**: Ikiwa unahitaji kuleta moduli kwa jina tofauti (inayojulikana kama "alias"), unaweza kutumia neno kuu `as` kama alias mwishoni mwa kauli ya import. Hii hutumika mara nyingi kwa majina marefu ya moduli au kuepuka migongano ya majina.

```python
import module_name as module_alias
```

Kwa mfano, kurejelea moduli ya `math` kama `m` katika msimbo wako, unaweza kuweka alias kama hii:

```python
import math as m
```

Kisha, unaweza kupata vipengele vya moduli ukitumia alias:

```python
m.sqrt(36)
```

- **Kuleta vipengele maalum**: Ikiwa huna haja ya kila kitu kutoka moduli, unaweza kuleta vipengele maalum kwa kutumia `from`. Katika kesi hii, kauli ya import huanza na `from`, ikifuatiwa na jina la moduli, kisha neno kuu `import`, na hatimaye majina ya vipengele unavyotaka kuleta.

```python
from module_name import name1, name2
```

Kisha, unaweza kutumia majina haya bila kiambishi cha moduli katika script yako ya Python. Kwa mfano:

```python
from math import radians, sin, cos

angle_degrees = 40
angle_radians = radians(angle_degrees)

sine_value = sin(angle_radians)
cos_value = cos(angle_radians)

print(sine_value) # 0.6427876096865393
print(cos_value)  # 0.766044443118978
```

Hii ni msaada, lakini inaweza kusababisha migongano ya majina ikiwa tayari una vitendakazi au vigezo vyenye majina sawa. Tafadhali zingatia hili unapoamua aina gani ya kauli ya import utumie.

Ikiwa unahitaji kuweka majina haya kuwa na alias, unaweza pia kufanya hivyo kwa kutumia neno kuu `as` ikifuatiwa na alias.

```python
from module_name import name1 as alias1, name2 as alias2
```

- **Kauli ya import yenye nyota (from module import `*`)**: Nyota inamwambia Python unataka kuleta kila kitu katika moduli hiyo, lakini unataka kuleta ili usihitaji kutumia jina la moduli kama kiambishi.

```python
from module_name import *
```

Kwa mfano, ikiwa utatumia hii kuleta moduli ya `math`, utaweza kuita kitendakazi chochote kilichotajwa katika moduli hiyo bila kuonyesha jina la moduli kama kiambishi.

```python
from math import *
print(sqrt(36))  # 6.0
```

Hata hivyo, hii kwa kawaida haipendekezwi kwa sababu inaweza kusababisha migongano ya majina na kufanya iwe vigumu kujua majina yanatoka wapi.

## `if __name__ == '__main__'`

- **Kigezo cha `__name__`**: `__name__` ni kigezo maalum kilichojengwa ndani katika Python. Wakati faili ya Python inatekelezwa moja kwa moja, Python huweka thamani ya kigezo hiki kuwa mfuatano wa herufi `"__main__"`. Lakini ikiwa faili ya Python inaletiwa kama moduli katika script nyingine ya Python, thamani ya kigezo `__name__` huwekwa kuwa jina la moduli hiyo.

Hii ndiyo sababu mara nyingi utapata masharti haya katika script za Python. Yana msimbo unaotaka kuendeshwa **tu** ikiwa script ya Python inaendeshwa kama programu kuu.

```python
if __name__ == '__main__': 
    # Code
```

## Makosa ya kawaida katika Python

- **SyntaxError**: Kosa ambalo Python hutupa wakati msimbo wako haufuati sheria za sintaksia. Kwa mfano, msimbo `print("Hello there"` utasababisha kosa la sintaksia lenye ujumbe `SyntaxError: '(' was never closed`, kwa sababu msimbo unakosa mabano ya kufunga.
- **NameError**: Python hutupa kosa la `NameError` wakati unajaribu kupata kigezo au kitendakazi ambacho hujafafanua. Kwa mfano, ikiwa una mstari `print(username)` katika msimbo wako bila kuwa na kigezo `username` kilichofafanuliwa awali, utapata kosa la jina lenye ujumbe `NameError: name 'username' is not defined`.
- **TypeError**: Huu ni kosa ambalo Python hutupa wakati unafanya operesheni kwenye aina za data zisizolingana. Kwa mfano, ikiwa unajaribu kuongeza mfuatano wa herufi kwenye nambari, utapata kosa `TypeError: can only concatenate str (not "int") to str`.
- **IndexError**: Utapata kosa la `IndexError` ikiwa utapata kielezo ambacho hakipo katika orodha au mfululizo mwingine kama tuple na mfuatano wa herufi. Kwa mfano, katika mfuatano wa herufi `Hello world`, kielezo cha herufi ya mwisho ni `10`. Ikiwa utaendelea na kupata herufi kwa njia hii, `greet = "hello world"; print(greet[11])`, utapata kosa lenye ujumbe `IndexError: string index out of range`.
- **AttributeError**: Python hutupa kosa hili wakati unajaribu kutumia njia au sifa ambayo haipo katika kitu cha aina hiyo. Kwa mfano, kuitisha `.append()` kwenye mfuatano wa herufi kama `"hello".append("!")` kutaongoza kwa kosa lenye ujumbe `AttributeError: 'str' object has no attribute 'append'`.

- **Kutumia kitendakazi cha `print`**: Kuingiza kauli za `print()` sehemu mbalimbali katika msimbo wako wakati wa kurekebisha makosa kunakusaidia kuona thamani za vigezo na jinsi msimbo unavyotiririka.
- **Kutumia debugger iliyojengwa ndani ya Python (`pdb`)**: Python hutoa moduli ya `pdb` kwa ajili ya kurekebisha makosa. Ni sehemu ya maktaba ya kawaida ya Python, hivyo daima inapatikana kutumia. Ukiwa na `pdb`, unaweza kuweka alama ya kufuatilia kwa kitendakazi cha `set_trace()` ili kuanza hatua kwa hatua kupitia msimbo na kuchunguza vigezo kwa njia ya mwingiliano.
- **Kutumia zana za kurekebisha makosa za IDE**: Mazingira mengi ya maendeleo (IDEs) na wahariri wa msimbo kama Pycharm na VS Code hutoa zana za kurekebisha makosa zenye alama za kusimama, utekelezaji wa hatua kwa hatua, uchunguzi wa vigezo, na vipengele vingine vya kurekebisha makosa.

## Kushughulikia makosa

- **`try...except`**: Hii hutumika kutekeleza sehemu ya msimbo ambayo inaweza kusababisha hitilafu. Kifungu cha `try` ndicho unachotarajia hitilafu itokee, wakati kifungu cha `except` kinachukua hitilafu maalum na kutekeleza ikiwa hitilafu hiyo itatokea. Hapa kuna mfano:

  ```py
  try:
    print(22 / 0)
  except ZeroDivisionError:
    print('You can\'t divide by zero!')
    # You can't divide by zero!
  ```

  Unaweza pia kuunganisha vifungu vingi vya `except` ili kushughulikia aina zaidi za makosa:

  ```py
  try:
    number = int(input('Enter a number: '))
    print(22 / number)
  except ZeroDivisionError:
    print('You cannot divide by zero!')
    # You cannot divide by zero! prints when you enter 0
  except ValueError:
    print('Please enter a valid number!')
    # Please enter a valid number! prints when you don't enter an integer
  ```

- **`else` na `finally`**: Vifungu hivi vinaongeza `try...except`. Ikiwa hakuna hitilafu itatokea, kifungu cha `else` hutekelezwa. Kifungu cha `finally` hutekelezwa kila wakati bila kujali kama hitilafu ilitokea au la.

  ```py
  try:
    result = 100 / 4
  except ZeroDivisionError:
    print('You cannot divide by zero!') # This will not run
  else:
    print(f'Result is {result}') # Result is 25.0
  finally:
    print('Execution complete!') # Execution complete!
  ```

- **Kipengele cha hitilafu**: Hiki kinakuruhusu kupata hitilafu yenyewe kwa ajili ya kurekebisha makosa bora na kuchapisha ujumbe wa kosa moja kwa moja. Ili kupata kipengele cha hitilafu, unahitaji kutumia neno kuu `as`. Hapa kuna mfano:

  ```py
  try:
      value = int('This will raise an error')
  except ValueError as e:
      print(f'Caught an error: {e}')
      # Caught an error: invalid literal for int() with base 10: 'This will raise an error'
  ```

- **Kauli ya `raise`**: Hii inakuwezesha kutupa hitilafu kwa mikono. Unaweza kuitumia kutupa hitilafu wakati hali fulani inatimia. Hapa kuna mfano:

  ```py
  def divide(a, b):
      if b == 0:
          raise ZeroDivisionError('You cannot divide by zero')
      return a / b
  ```
  
## Kuashiria hitilafu

- Kauli ya `raise` pia ni muhimu unapounda makosa yako maalum, kwani unaweza kuitumia kutupa hitilafu yenye ujumbe maalum. Hapa kuna mfano wa hilo:

  ```py
  class InvalidCredentialsError(Exception):
      def __init__(self, message="Invalid username or password"):
          self.message = message
          super().__init__(self.message)

  def login(username, password):
      stored_username = "admin"
      stored_password = "password123"
      
      if username != stored_username or password != stored_password:
          raise InvalidCredentialsError()
      
      return f"Welcome, {username}!"
  ```  

  Hapa ni jinsi unavyoweza kutumia kitendakazi cha `login` na hitilafu ya `InvalidCredentialsError`:

  ```py
  # failed login attempt
  try:
      message = login("user", "wrongpassword")
  except InvalidCredentialsError as e:
      print(f"Login failed: {e}")
  else:
      print(message)

  # successful login attempt
  try:
      message = login("admin", "password123")
  except InvalidCredentialsError as e:
      # This block is not executed because the login was successful
      print(f"Login failed: {e}")
  else:
      # The else block runs if the 'try' block completes without an exception
      print(message)
  ```

  Kauli ya `raise` inaweza pia kutumika pamoja na neno kuu `from` kuunganisha makosa, kuonyesha uhusiano kati ya makosa tofauti:

  ```py
  def parse_config(filename):
    try:
        with open(filename, 'r') as file:
            data = file.read()
            return int(data)
    except FileNotFoundError:
        raise ValueError('Configuration file is missing') from None
    except ValueError as e:
        raise ValueError('Invalid configuration format') from e

  config = parse_config('config.txt')
  ```

## Njia za dunder (za kichawi)

- **Ufafanuzi**: Njia maalum zinazotangulia na kumalizika na alama mbili za mstari chini (mfano, `__init__`, `__len__`, `__str__`, `__eq__`). Python hutumia hizi ndani kwa ajili ya operesheni zilizojengwa ndani.

```python
class Book:
    def __init__(self, title, pages):
        self.title = title
        self.pages = pages

    def __len__(self):
        return self.pages

    def __str__(self):
        return f"'{self.title}' has {self.pages} pages"

    def __eq__(self, other):
        return self.pages == other.pages

book1 = Book('Built Wealth Like a Boss', 420)
print(len(book1))        # 420
print(str(book1))        # 'Built Wealth Like a Boss' has 420 pages
```

- **Kuita njia za dunder kwa njia isiyo ya moja kwa moja**: Huhitaji kuita njia za dunder moja kwa moja. Badala yake, Python huziita moja kwa moja wakati matendo fulani yanapotokea. Operesheni hizi ni pamoja na:

  - **operesheni za kihesabu kama kuongeza, kutoa, kuzidisha, kugawanya, na nyinginezo**. `__add__()` huitwa kwa kuongeza, `__sub__()` kwa kutoa, `__mul__()` kwa kuzidisha, na `__truediv__()` kwa kugawanya.
        
  - **operesheni za mfuatano wa herufi kama kuunganisha, kurudia, kuunda muundo, na kubadilisha kuwa maandishi**. `__add__()` huitwa kwa kuunganisha, `__mul__()` kwa kurudia, `__format__()` kwa kuunda muundo, `__str__()` na `__repr__()` kwa kubadilisha kuwa maandishi, na kadhalika.
        
  - **operesheni za kulinganisha kama usawa, chini kuliko, juu kuliko, na nyinginezo**. `__eq__()` huitwa kwa ukaguzi wa usawa, `__lt__()` kwa chini kuliko, `__gt__()` kwa juu kuliko, na kadhalika.
        
  - **operesheni za kurudia kama kufanya kitu kiwe kinachoweza kurudiwa na kusonga mbele kupitia vitu**. `__iter__()` huitwa kurudisha kielezi cha kurudia na `__next__()` kupata kipengee kinachofuata.

## Mfano halisi wa dunia: Gari la ununuzi

- **Darasa la gari lenye njia za dunder**: Huruhusu kuongeza, kufuta, kurudia, na kuangalia yaliyomo kwa tabia zilizojengwa ndani.

```python
class Cart:
    def __init__(self):
        self.items = []

    def add(self, item):
        self.items.append(item)

    def remove(self, item):
        if item in self.items:
            self.items.remove(item)
        else:
            print(f'{item} is not in cart')

    def list_items(self):
        return self.items

    def __len__(self):
        return len(self.items)

    def __getitem__(self, index):
        return self.items[index]

    def __contains__(self, item):
        return item in self.items

    def __iter__(self):
        return iter(self.items)

cart = Cart()
cart.add('Laptop')
print(len(cart))        # 1
print('Laptop' in cart) # True
```

## Nini ni programu inayotegemea vitu?

- **Programu inayotegemea vitu**: Mtindo wa programu ambapo wasanidi programu hutumia kila kitu katika msimbo wao kama kitu halisi cha dunia. Inajulikana sana kama OOP. Kanuni kuu nne zinazokusaidia kupanga na kusimamia msimbo kwa ufanisi ni **ufungaji**, **urithi**, **polimofizimu**, na **ufupishaji**.
- **Madarasa**: Ramani ya kuunda vitu. Kila kitu kinachoundwa kutoka darasa kina sifa zinazofafanua data na njia zinazobainisha tabia za kitu hicho.

## Nini ni ufungaji?

- **Ufungaji**: Kuunganisha sifa na njia za kitu katika kitengo kimoja. Hukuruhusu kuficha hali ya ndani ya kitu nyuma ya seti rahisi ya njia na sifa za umma zinazofanya kazi kama milango. Nyuma ya milango hiyo kuna sifa na njia za faragha zinazodhibiti jinsi data inavyobadilika na nani anaweza kuiona.
- **Mfano wa ufungaji**: Ikiwa unataka kufuatilia salio la pochi, utaruhusu kuweka na kutoa fedha, lakini hutaki mtu yeyote kuingilia salio la pochi moja kwa moja:

```py
class Wallet:
   def __init__(self, balance):
       self.__balance = balance # Private attribute

   def deposit(self, amount):
       if amount > 0:
           self.__balance += amount # Add to the balance safely

   def withdraw(self, amount):
       if 0 < amount <= self.__balance:
           self.__balance -= amount # Remove from the balance safely

account = Wallet(500)
print(account.__balance) # AttributeError: 'Wallet' object has no attribute '__balance'
```

- **Tofauti kati ya kuweka kiambishi awali cha mstari mmoja na wa mistari miwili kwenye sifa**: Kuweka kiambishi awali cha mstari mmoja kwenye sifa na njia ina maana ni kwa matumizi ya ndani. Hii ni mkataba, na haizuizi upatikanaji wa sifa kutoka nje. Kuweka kiambishi awali cha mistari miwili kwenye sifa na njia huzuia kabisa upatikanaji kutoka nje ya darasa lao.

## Nini ni wapokeaji na wapezaji?

- **Wapokeaji na wapezaji**: Njia zinazokuwezesha kudhibiti jinsi sifa za darasa zinavyopatikana na kubadilishwa. Unapata thamani kwa wapokeaji na unaweka thamani kwa wapezaji.
- **Sifa**: Zinawaunganisha wapokeaji na wapezaji, na kuruhusu upatikanaji wa data. Hufanya mantiki ya ziada nyuma ya pazia wakati unapata, unaweka, au kufuta thamani.
- **Kwa nini sifa badala ya njia**: Sifa hutumika badala ya njia kwa ajili ya usomaji bora na msimbo safi. Zinakuwezesha kupata thamani kwa uandishi wa nukta, kama sifa za kawaida, bila mabano.
- **Kuunda mpokeaji**: Kuunda mpokeaji, tumia dekoreta wa `@property`. Hapa kuna mpokeaji anayepata mduara wa mduara:

```py
class Circle:
    def __init__(self, radius):
        self.radius = radius # Calling the setter

    @property
    def radius(self): # A getter to get the radius
        return self._radius
  
    @property
    def area(self):  # A getter to calculate area
        return 3.14 * (self._radius ** 2)

my_circle = Circle(3)

print(my_circle.radius) # 3
print(my_circle.area) # 28.26
```

Kutumia self.radius ndani ya `__init__()` huhakikisha wapezaji anafanya kazi wakati wa uundaji wa kitu, hivyo thamani zisizo halali za mduara zinakamatwa mara moja.

- **Kuunda mpezaji**: Kuunda mpezaji atakayoweka mduara, lazima ueleze njia nyingine yenye jina sawa na utumie `@<property_name>.setter` juu yake:

```py
class Circle:
    def __init__(self, radius):
        self.radius = radius # Calling the setter

    @property
    def radius(self):  # A getter to get the radius
        return self._radius

    @radius.setter
    def radius(self, value):  # A setter to set the radius
        if value <= 0:
            raise ValueError('Radius must be positive')
        self._radius = value

my_circle = Circle(3)
print('Initial radius:', my_circle.radius) # Initial radius: 3

my_circle.radius = 8
print('After modifying the radius:', my_circle.radius) # After modifying the radius: 8
```

- **Jinsi Python inavyoshughulikia wapokeaji na wapezaji**: Mara unapofafanua wapokeaji na wapezaji, Python huwaita moja kwa moja kila wakati unapotumia sintaksia ya kawaida ya sifa hivi:

```py
my_circle.radius # This will call the getter
my_circle.radius = 4 # This will call the setter
```

Ndani ya mpezaji, haufai kuweka jina la sifa yenyewe kwa sababu hiyo itasababisha `RecursionError`.

- **Mfuta**: Baada ya kuweka na kupata thamani kwa mpezaji na mpokeaji, unaweza kudhibiti jinsi inavyofutwa kwa `deleter`. Mfuta hufanya mantiki maalum wakati unapotumia tamko la `del` kwenye sifa. Kuunda mfuta, tumia dekoreta wa `@<property_name>.deleter`.

```py
  # Deleter
    @radius.deleter
    def radius(self):
        print("Deleting radius...")
        del self._radius
```

## Nini ni urithi na unavyosaidia kutumia msimbo tena?

- **Urithi**: Mchakato ambapo darasa la mtoto hutumia sifa na njia za darasa la mzazi. Urithi husaidia kutumia msimbo tena, hutoa hierarkia wazi, na kubadilisha tabia bila kuandika msimbo upya. Kutekeleza urithi, darasa la mtoto huchukua jina la darasa la mzazi:

```py
class Parent:
    # Parent attributes and methods

class Child(Parent):
    # Child inherits, extends, and/or overrides where necessary
```

- **Urithi wa moja kwa moja na wa mara nyingi**: Wakati darasa la mtoto linarithi sifa na njia kutoka kwa mzazi mmoja, kama unavyoona hapo juu, mchakato huo huitwa **urithi wa moja kwa moja**. Wakati darasa la mtoto linarithi sifa na njia kutoka kwa wazazi zaidi ya mmoja, hiyo ni **urithi wa mara nyingi**. Hapa ni sintaksia yake:

```py
class Parent:
    # Attributes and methods for Parent

class Child:
    # Attributes and methods for Child

class GrandChild(Parent, Child):
    # GrandChild inherits from both Parent and Child
    # GrandChild can combine or override behavior from each
```

- **Kitendakazi cha `super()`**: Kitendakazi kinachokuwezesha **kuita** njia kutoka kwa darasa la mzazi, wakati darasa lina utekelezaji tofauti wa njia hiyo, au linaongeza njia hiyo, bila kuandika msimbo mara mbili.

## Nini ni polimofizimu na inavyosaidia kutumia msimbo tena?

- **Polimofizimu**: Kanuni ya OOP inayoruhusu madarasa tofauti kutumia jina moja la njia, lakini kila darasa linautekeleza tofauti wakati linapoitwa. Hapa ni sintaksia yake:

```py
class A:
   def action(self): ...

class B:
   def action(self): ...

class C:
   def action(self): ...

Class().method()  # Works for A, B, or C
```

- **Polimofizimu inayotegemea urithi**: Mzazi huandaa njia, na kila darasa la mtoto huibadilisha kwa matumizi yao.

## Nini ni mabadiliko ya jina na inavyofanya kazi?

- **Mabadiliko ya jina**: Mchakato ambapo Python ndani yake hubadilisha jina la sifa lililo na kiambishi awali cha mistari miwili kwa kuongeza mstari chini na jina la darasa kama kiambishi awali, kubadilisha `__attribute` kuwa `_ClassName__attribute`.
- **Madhumuni ya mabadiliko ya jina**: Madhumuni kuu ni kuzuia upitishaji wa sifa na njia kwa bahati mbaya wakati unatumia urithi. Hapa kuna msimbo unaofanya hili kuwa rahisi kuelewa:

```py
class Parent:
    def __init__(self):
        self.__data = 'Parent data'

class Child(Parent):
    def __init__(self):
        super().__init__()
        self.__data = 'Child data'

c = Child()
print(c.__dict__) # {'_Parent__data': 'Parent data', '_Child__data': 'Child data'}
```

## Nini ni ufupishaji na unavyosaidia kuweka mifumo ngumu imepangwa?

- **Ufupishaji**: Dhana ya programu ambapo maelezo magumu ya utekelezaji wa kitu au mfumo hufichwa na sifa muhimu tu zinaonyeshwa. Katika Python na lugha nyingine za programu, ufupishaji hurahisisha mifumo ngumu kwa kuongeza matumizi tena.
- **Mfano wa ufupishaji**: Mfano mzuri wa ufupishaji katika maisha ya kila siku ni gari linalokuruhusu kutumia tu gurudumu, pedali, na mabadiliko bila kujua jinsi injini au breki zinavyofanya kazi.
- **Jinsi Python inavyotekeleza ufupishaji**: Python hufanya ufupishaji kupitia moduli ya `abc`. Moduli hutoa darasa la `ABC` (darasa la msingi la abstract) na dekoreta wa `@abstractmethod`. Darasa la msingi la abstract (ABC) linafafanua njia na sifa za kawaida ambazo madarasa yanayotegemea lazima yatekeleze. Haiwezi kuundwa moja kwa moja.
- **Jinsi njia ya abstract inavyofafanuliwa**: Njia ya abstract inafafanuliwa kwa `@abstractmethod` na lazima ibadilishwe katika madarasa yanayotegemea, hata kama ina utekelezaji wa msingi. Sintaksia ya msingi ya ufupishaji ni kama ifuatavyo:

```py
from abc import ABC, abstractmethod

# Define an abstract base class
class AbstractClass(ABC):
    @abstractmethod
    def abstract_method(self):
        pass

# Concrete subclass that implements the abstract method
class ConcreteClassOne(AbstractClass):
    def abstract_method(self):
        print('Implementation in ConcreteClassOne')

# Another concrete subclass
class ConcreteClassTwo(AbstractClass):
    def abstract_method(self):
        print('Implementation in ConcreteClassTwo')
```

## Algoriti na noti ya Big O

- **Algoriti**: Seti ya maagizo yasiyo na utata kwa kutatua tatizo au kutekeleza zoezi. Algoriti lazima zikamilike kwa idadi ya hatua isiyo na kikomo na kila hatua lazima iwe sahihi na isiyo na utata.

- **Noti ya Big O**: Inaelezea utendaji mbaya zaidi, au kiwango cha ukuaji, cha algoriti wakati ukubwa wa ingizo unavyoongezeka. Inazingatia jinsi matumizi ya rasilimali yanavyokua kwa ukubwa wa ingizo, ikiacha mambo ya kudumu na vipengele vya chini zaidi.

### Ugumu wa kawaida wa wakati

- **O(1) - Wakati wa kudumu**: Algoriti huchukua muda sawa bila kujali ukubwa wa ingizo.

```python
def check_even_or_odd(number):
    if number % 2 == 0:
        return 'Even'
    else:
        return 'Odd'
```

- **O(log n) - Wakati wa logaritmiki**: Muda huongezeka polepole kadri ingizo linavyokua. Mara nyingi hutumika katika algoriti zinazopunguza mara kwa mara ukubwa wa tatizo kwa sehemu (kama Utafutaji wa Kibainari).

- **O(n) - Wakati wa mstari**: Muda wa utekelezaji huongezeka kwa uwiano na ukubwa wa ingizo.

```python
for grade in grades:
    print(grade)
```

- **O(n log n) - Wakati wa log-mstari**: Ugumu wa kawaida wa algoriti za kupanga zenye ufanisi kama Merge Sort na Quick Sort.

- **O(n²) - Wakati wa mraba**: Muda wa utekelezaji huongezeka mraba. Mara nyingi huonekana katika mizunguko iliyopangwa ndani.

```python
for i in range(n):
    for j in range(n):
        print("Hello, World!")
```

### Ugumu wa nafasi

- **O(1) - Nafasi ya kudumu**: Algoriti hutumia kiasi sawa cha kumbukumbu bila kujali ukubwa wa ingizo.
- **O(n) - Nafasi ya mstari**: Matumizi ya kumbukumbu huongezeka kwa uwiano na ukubwa wa ingizo.
- **O(n²) - Nafasi ya mraba**: Matumizi ya kumbukumbu huongezeka mraba kwa ukubwa wa ingizo.

## Mbinu za kutatua matatizo

- **Kuelewa tatizo**: Soma taarifa ya tatizo mara kadhaa. Tambua ingizo, matokeo yanayotarajiwa, na jinsi ya kubadilisha ingizo kuwa matokeo.

- **Msimbo wa bandia**: Maelezo ya kiwango cha juu ya mantiki ya algoriti ambayo haitegemei lugha yoyote. Inatumia lugha ya kawaida iliyochanganywa na miundo ya programu kama `IF`, `ELSE`, `FOR`, `WHILE`.

```md

  ADD character TO THE BEGINNING OF reversed_string

```

- **Matukio ya kipekee**: Ingizo maalum, halali linalotokea katika mipaka ya kile algoriti inapaswa kushughulikia. Daima zingatia na jaribu matukio ya kipekee.

### Orodha zilizounganishwa za peke yake

- **Kupitia**: Inaweza kusogea mbele tu kutoka kichwa hadi mkia.
- **Nodi ya kichwa**: Nodi ya kwanza kwenye orodha, kawaida ndiyo nodi pekee inayopatikana moja kwa moja.

- **Kumbukumbu**: Inahitaji kumbukumbu zaidi kuliko orodha zilizounganishwa za peke yake kwa sababu ya rejea ya ziada.

## Ramani za hash na seti

### Ramani na ramani za hash

- **Ramani (Aina ya data ya abstract)**: Inasimamia makusanyo ya jozi za ufunguo-thamani. Kila ufunguo lazima uwe wa kipekee, lakini thamani zinaweza kurudiwa.

- **Ramani ya hash**: Utekelezaji halisi wa ramani ya ADT kwa kutumia mbinu ya hashing. Inatumia kitendakazi cha hash kuunda thamani za hash kwa funguo, ambazo huamua mahali pa kuhifadhi kwenye safu ya msingi.

### Ugumu wa wakati kwa ramani za hash

- **Kesi ya wastani**: O(1) kwa kuingiza, kupata, kufuta
- **Kesi mbaya zaidi**: O(n) wakati kuna migongano mingi ya hash

### Seti

- **Seti**: Makusanyo yasiyo na mpangilio ya vipengee vya kipekee. Hakuna rudufu, hakuna mpangilio maalum unaohifadhiwa.

- **Vipengee visivyobadilika tu**: Seti zinaweza kuwa na aina za data zisizobadilika (nambari, mfuatano wa herufi, tuple) kwa sababu thamani za hash lazima zibaki thabiti.

```python
# Creating sets

numbers.add(5)
numbers.remove(4)      # Raises KeyError if not found

set_a = {1, 2, 3, 4}

# Membership testing
5 in numbers
```

### Ugumu wa wakati kwa seti

- **Kesi ya wastani**: O(1) kwa kuongeza, kuondoa, kujaribu uanachama
- **Kesi mbaya zaidi**: O(n) kutokana na migongano ya hash

- **Mgongano wa hash**: Hutokea wakati funguo mbili tofauti zinazalisha thamani sawa ya hash.

- **Mbinu za kutatua migongano**:
  - **Uunganishaji wa mfululizo**: Kila kielezo cha safu kinaonyesha orodha iliyounganishwa inayohifadhi vipengee vyote vyenye thamani sawa ya hash
  - **Anwani wazi**: Tafuta kielezo kinachopatikana kinachofuata kwa kutumia mfululizo uliowekwa awali

## Wakati wa kutumia kila muundo wa data

- **Orodha**: Unapohitaji upatikanaji wa mpangilio, uliowekwa kielezo na haujui ukubwa mapema
- **Staki**: Kwa operesheni za LIFO (kazi ya kufuta, tathmini ya usemi, kurudi nyuma)
- **Foleni**: Kwa operesheni za FIFO (ratiba ya kazi, utafutaji wa upana kwanza)
- **Orodha zilizounganishwa**: Wakati wa kuingiza/kufuta mara kwa mara mwanzoni, ukubwa usiojulikana, hakuna upatikanaji wa nasibu unahitajika

- **Seti**: Kwa ukaguzi wa kipekee, operesheni za seti za kihesabu, kuondoa rudufu

## Algoriti za utafutaji

Algoriti za utafutaji hukuruhusu kutafuta lengo ndani ya orodha fulani ya vitu.

Katika sayansi ya kompyuta, kuna algoriti mbili za utafutaji unazopaswa kujua. Ni **utafutaji wa mstari** na **utafutaji wa kibainari**. Ni muhimu kuelewa tofauti kati ya algoriti hizi mbili na lini kutumia kila moja.

### Utafutaji wa mstari

- Utafutaji wa mstari hurudia kupitia orodha ya vitu, ukikagua kila kipengee kutoka mwanzo hadi kipengee lengwa kipatikane.
- Ikiwa kipengee lengwa kinapatikana, kielezo ambacho kiko kwenye orodha kinarejeshwa.
- Ikiwa lengwa halipatikani, hurudisha `-1`, ambayo ina maana ya **kielezo kisicho halali** katika lugha nyingi za programu.
- Kwa sababu utafutaji wa mstari hukagua kila kipengee hadi kupatikana lengwa, si mzuri kwa orodha kubwa ya vitu.
- Ugumu wa wakati wa utafutaji wa mstari ni `O(n)` kwa sababu muda unaochukua kutafuta kupitia orodha huongezeka kwa mstari na ukubwa wa orodha.
- Ugumu wa nafasi wa utafutaji wa mstari ni `O(1)` kwa sababu hauhitaji nafasi ya ziada kutafuta kupitia orodha.

### Utafutaji wa kibainari

- Utafutaji wa kibainari hufanya kazi kwa kugawanya orodha ya vitu nusu, na kukagua kama thamani lengwa iko katikati ya orodha.
- Sharti la utafutaji wa kibainari ni kwamba vitu katika orodha lazima viwe katika mpangilio wa kuongezeka.
- Utafutaji wa kibainari ni algoriti yenye ufanisi zaidi kwa kutafuta kupitia orodha kubwa ya vitu kwa sababu hugawanya orodha nusu na kupuuza nusu ambayo lengwa halipo.
- Ikiwa kipengee lengwa kinapatikana katikati ya orodha, kielezo cha kipengee lengwa kinarejeshwa.
- Ikiwa kipengee hakipatikani, algoriti hukagua kama lengwa liko nusu ya kushoto au ya kulia ya orodha.
- Inaendelea kugawanya sehemu zilizobaki za orodha nusu hadi kipengee lengwa kipatikane.
- Ikiwa kipengee lengwa hatimaye hakipatikani katika orodha, hurudisha `-1`.
- Ugumu wa wakati wa utafutaji wa kibainari ni `O(log n)` kwa sababu muda unaochukua kutafuta kupitia orodha huongezeka kwa logaritmiki na ukubwa wa orodha.
- Ugumu wa nafasi wa utafutaji wa kibainari ni `O(1)` kwa sababu hauhitaji nafasi ya ziada kutafuta kupitia orodha.

### Jinsi utafutaji wa mstari unavyotofautiana na utafutaji wa kibainari

- Utafutaji wa kibainari unafaa zaidi kwa orodha kubwa ya vitu ikilinganishwa na utafutaji wa mstari.
- Ugumu wa wakati wa utafutaji wa mstari ni `O(n)` kwa sababu muda unaochukua kutafuta kupitia orodha huongezeka kwa mstari na ukubwa wa orodha.
- Ugumu wa wakati wa utafutaji wa kibainari ni `O(log n)` kwa sababu muda unaochukua kutafuta kupitia orodha huongezeka kwa logaritmiki na ukubwa wa orodha.

## Algoriti za kupanga na mbinu ya kugawanya-na-kutawala

Katika sayansi ya kompyuta, kugawanya-na-kutawala ni mbinu inayotumika kugawanya tatizo kuwa matatizo madogo madogo ili yawe rahisi kutatua. Rekursia ni mbinu inayotumika mara nyingi katika kugawanya-na-kutawala, na ni mkakati wenye nguvu unaotumika kutekeleza algoriti nyingi za kupanga zenye ufanisi kama merge sort.

### Merge sort

- Merge sort ni algoriti ya kupanga inayofuata mbinu ya kugawanya-na-kutawala.
- Hufanya kazi kwa kugawanya orodha kwa rekursia kuwa orodha ndogo ndogo hadi kila orodha ndogo iwe na kipengee kimoja tu.
- Kisha huunganisha orodha ndogo ndogo hizo tena kwa mpangilio uliopangwa.
- Ugumu wa wakati wa merge sort ni `O(n log n)` kwa sababu orodha hugawanywa mara kwa mara nusu `(log n)` na kisha kuunganishwa pamoja `(O(n))`.
- Ugumu wa nafasi wa merge sort ni `O(n)` kwa sababu si algoriti ya kupanga inayoendeshwa mahali pake.

## Muhtasari wa grafu

Grafu ni seti ya nodi (vichwa) vilivyounganishwa na makundi (miunganisho). Kila nodi inaweza kuunganishwa na nodi nyingi, kuunda mtandao. Aina tofauti za grafu ni:

- Zenye mwelekeo: makundi yana mwelekeo (kutoka nodi moja hadi nyingine), mara nyingi huonyeshwa kwa mistari ya moja kwa moja na mishale.
- Zisizo na mwelekeo: makundi hayana mwelekeo, huonyeshwa kwa mistari rahisi.
- Vichwa: kila nodi ina lebo au kitambulisho.
- Zenye mizunguko: zina mizunguko (njia inayotangulia na kumalizika kwenye nodi ile ile).
- Zisizo na mizunguko (DAG): hazina mizunguko.
- Makundi yenye lebo: kila kundi lina lebo kawaida huandikwa karibu na kundi husika.
- Zenye uzito: makundi yana uzito (thamani) unaohusishwa nayo, unaoweza kutumika kufanya operesheni za kihesabu.
- Zisizounganishwa: zina nodi mbili au zaidi ambazo hazijaunganishwa na makundi yoyote.

Grafu hutumika katika programu mbalimbali kama ramani, mitandao, mifumo ya mapendekezo, utatuzi wa utegemezi.

## Kupitia grafu

Hii inahusisha kutembelea nodi zote katika grafu. Algoriti kuu mbili ni:

- **Utafutaji wa upana kwanza (BFS)**
  - Hutumia foleni.
  - Huchunguza ngazi kwa ngazi.
  - Hupata njia fupi katika grafu zisizo na uzito.

- **Utafutaji wa kina kwanza (DFS)**
  - Hutumia staki (au rekursia).
  - Huchunguza tawi lote kabla ya kurudi nyuma.
  - Inafaa kwa kugundua mizunguko na kutafuta njia.

## Uwawakilishi wa grafu

Grafu zinaweza kuwakilishwa kwa njia kuu mbili:

- **Orodha ya ujirani**
  - Kila nodi ina orodha ya majirani zake.
  - Inatumia nafasi kidogo kwa grafu zenye vipengee vichache.
  - Rahisi kurudia kupitia majirani.

- **Matriki ya ujirani**
  - Safu ya 2D ambapo safu na nguzo zinawakilisha nodi.
  - Inatumia nafasi nyingi kwa grafu kubwa.
  - Haraka kuangalia kama kuna kundi kati ya nodi mbili.

## Miti

Mti ni aina maalum ya grafu isiyo na mizunguko na imeunganishwa. Sifa kuu ni:

- Haina mizunguko au mizunguko (njia ambapo nodi ya mwanzo na mwisho ni ile ile).
- Lazima iwe imeunganishwa (kila nodi inaweza kufikiwa kutoka kwa nodi nyingine yoyote).

### Aina za kawaida za miti

Aina za kawaida za miti ni:

- Miti ya kibainari
  - Kila nodi ina watoto wawili angalau, mtoto wa kushoto na mtoto wa kulia.

- Miti ya utafutaji wa kibainari (BST)
  - Mti wa kibainari ambapo kila mtoto wa kushoto ni mdogo kuliko mzazi wake, na kila mtoto wa kulia ni mkubwa kuliko mzazi wake.


## Miti ya maneno (tries)

Pia huitwa miti ya kiambishi awali, hutumika kuhifadhi seti za mfuatano wa herufi, ambapo kila nodi inawakilisha herufi.

Kiambishi awali kinashirikiwa mara moja tu, na hufanya kazi kwa ufanisi kwa kazi kama kukamilisha maandishi na ukaguzi wa tahajia.

Operesheni za utafutaji na kuingiza zina ugumu wa wakati wa O(L), ambapo L ni urefu wa mfuatano wa herufi.

## Foleni za kipaumbele

Foleni ya kipaumbele ni aina ya data ya abstract ambapo kila kipengee kina kipaumbele.

Foleni na staki huchukulia tu mpangilio wa kuingizwa, wakati foleni za kipaumbele huchukulia kipaumbele cha vipengee.

Foleni za kawaida hufuata FIFO (Kwanza Kuingia Kwanza Kuondoka) na staki hufuata LIFO (Mwisho Kuingia Kwanza Kuondoka). Hata hivyo, katika foleni ya kipaumbele, vipengee vyenye kipaumbele kikubwa hudhibitiwa kwanza bila kujali mpangilio wa kuingizwa.

## Rundo

Ni muundo wa data wa mti maalum wenye sifa maalum inayoitwa sifa ya rundo.

Sifa ya rundo inaamua uhusiano kati ya nodi ya mzazi na nodi ya mtoto. Kuna aina mbili za rundo:

- Rundo kubwa (Max-Heap)
  - Thamani ya kila nodi ya mzazi ni kubwa au sawa na thamani za watoto wake.
  - Kipengee kikubwa zaidi kiko kwenye mzizi.

- Rundo ndogo (Min-Heap)
  - Thamani ya kila nodi ya mzazi ni ndogo au sawa na thamani za watoto wake.
  - Kipengee kidogo zaidi kiko kwenye mzizi.

### Mfano wa moduli ya `heapq` ya Python

```py
import heapq

# Create empty heap
my_heap = []

# Insert elements
heapq.heappush(my_heap, 9)
heapq.heappush(my_heap, 3)
heapq.heappush(my_heap, 5)
print(my_heap) # [3, 9, 5]

# Remove smallest element
print(heapq.heappop(my_heap))  # 3
print(my_heap) # [5, 9]

# Push + Pop in one step
print(heapq.heappushpop(my_heap, 7)) # 5
print(my_heap) # [7, 9]

# Transform list into heap
nums = [5, 7, 3, 1]
heapq.heapify(nums)
```

### Kutumia vipaumbele

```py
my_heap = []
heapq.heappush(my_heap, (3, "A"))
heapq.heappush(my_heap, (2, "B"))
heapq.heappush(my_heap, (1, "C"))

# Removes lowest number = highest priority
print(heapq.heappop(my_heap))  # (1, "C")
```

## Utangulizi wa programu ya mabadiliko

- **Ufafanuzi**: Programu ya mabadiliko ni mbinu ya algoriti inayotatua matatizo magumu kwa kuyagawa kuwa matatizo madogo madogo na kuhifadhi matokeo ili kuepuka mahesabu ya kurudiwa.
- **Matatizo yanayojirudia**: Matatizo madogo madogo yanayojirudia mara nyingi wakati wa kutatua tatizo kubwa. Badala ya kuhesabu tena mara kwa mara, tunahifadhi suluhisho lao.
- **Muundo bora wa matatizo**: Suluhisho bora la tatizo lina suluhisho bora za matatizo madogo madogo. Hii ina maana tunaweza kujenga suluhisho bora kwa kuunganisha suluhisho bora za sehemu ndogo.

## Suluhisho za programu ya mabadiliko

- **Memoization (Mbinu ya juu-chini)**: Memoization huhifadhi matokeo ya wito wa kitendakazi ghali na hurudisha matokeo yaliyohifadhiwa wakati ingizo sawa linapotokea tena.

```py
def climb_stairs_memo(n, memo={}):
    """Dynamic programming with memoization"""
    # Check if we've already calculated this value
    if n in memo:
        return memo[n]  # Return cached result - O(1) lookup!
    
    # Base cases
    if n <= 2:
        return n
    
    # Calculate once and store in memo for future use
    memo[n] = climb_stairs_memo(n-1, memo) + climb_stairs_memo(n-2, memo)
    return memo[n]
```

- **Tabulation (Mbinu ya chini-juu)**: Tabulation hujenga suluhisho kutoka chini, ikijaza jedwali na suluhisho za matatizo madogo.

```py
def climb_stairs_tabulation(n):
    """Dynamic programming with tabulation"""
    if n <= 2:
        return n
    
    # Create array to store results for all steps from 0 to n
    dp = [0] * (n + 1)
    dp[1] = 1  # 1 way to reach step 1
    dp[2] = 2  # 2 ways to reach step 2
    
    # Build up the solution iteratively
    for i in range(3, n + 1):
        # Ways to reach step i = ways to reach (i-1) + ways to reach (i-2)
        dp[i] = dp[i-1] + dp[i-2]
    
    return dp[n]
```
