# --description--

Mateo tiene 25 años, y Luna tiene 28 años.

Usa la combinación correcta de `veinti` y el número en palabras para llenar los espacios.

Para referencia:

| Number | Spanish Word |
| :---: | :---: |
| 1 | `uno` |
| 2 | `dos` |
| 3 | `tres` |
| 4 | `cuatro` |
| 5 | `cinco` |
| 6 | `seis` |
| 7 | `siete` |
| 8 | `ocho` |
| 9 | `nueve` | 

# --fillInTheBlank--

## --sentence--

`Luna tiene BLANK años.`

`Mateo tiene BLANK años.`

## --blanks--

`veintiocho`

### --feedback--

Piensa en cómo combinar `veinte` (20) y `ocho` (8) para formar una palabra.

---

`veinticinco`

### --feedback--

Recuerda que al combinar `veinte` (20) con `cinco` (5), se elimina la “e” final en `veinte`.

# --explanation--

Las respuestas correctas son `veintiocho` y `veinticinco`.

En español, los números del 21 al 29 se escriben como una sola palabra. Por ejemplo:

- `Luna tiene veintiocho años` - Luna tiene 28 años.

- `Mateo tiene veinticinco años` - Mateo tiene 25 años.
