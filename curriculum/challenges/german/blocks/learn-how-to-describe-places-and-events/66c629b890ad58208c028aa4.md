# --explanation--

`Walkthrough` besteht aus zwei Teilen:

- `Walk` bezieht sich auf das schrittweise Vorangehen, normalerweise zu Fuß.

- `Through` bedeutet, sich von einer Seite zur anderen oder von Anfang bis Ende zu bewegen.

Wenn diese Wörter zu `walkthrough` kombiniert werden, bedeutet das eine Schritt-für-Schritt-Erklärung oder Anleitung, die jemandem hilft zu verstehen, wie man etwas macht. Zum Beispiel:

- `The software includes a walkthrough to help new users get started.` – Es gibt eine Anleitung, die jeden Schritt zur Nutzung der Software erklärt.

- `Let me give you a walkthrough of the process.` – Sie erklären jeden Schritt des Prozesses im Detail.
